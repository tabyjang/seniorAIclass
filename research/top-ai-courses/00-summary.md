## 🎯 시니어 AI 클래스에 적용할 핵심 원칙

### 1. 코딩 없이 개념 중심
```
Elements of AI 방식:
이론 → 예시 → 퀴즈 → 다음 주제
(코드 한 줄 없음!)
```

### 2. 짧은 모듈 구성
```
Google AI 방식:
1개 모듈 = 1-2시간 (최대!)
→ 집중력 유지, 성취감
```

### 3. 실생활 예시 중심
```
모든 top 클래스 공통:
- 집값 예측 (회귀)
- 스팸 필터 (분류)
- 영화 추천 (추천 시스템)
→ "이게 내 생활에 어떻게?"
```

### 4. 결과 먼저 (fast.ai 방식)
```
전통: 이론 → 코드 → 결과
fast.ai: 결과 → 코드 → 이론

→ 시니어: 결과 → 개념 → 활용법
```

### 5. 단계별 난이도
```
Coursera ML 방식:
Week 1: 쉬움 (자신감)
Week 2: 조금 어려움
Week 3: 중간
Week 4: 도전
```
