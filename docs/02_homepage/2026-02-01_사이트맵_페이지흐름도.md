## 4. 시니어 친화 UX 원칙

### 네비게이션
- ✅ **최대 2단계 깊이**: 메인 → 하위 페이지 → 완료
- ✅ **뒤로 가기 항상 가능**: 브라우저 뒤로가기 + UI 버튼
- ✅ **현재 위치 표시**: 헤더 메뉴 활성화 + 브레드크럼

### 페이지 구조
- ✅ **한 페이지, 한 목적**: 레벨테스트는 테스트만, 상담은 상담만
- ✅ **스크롤 최소화**: 핵심 정보는 위쪽, CTA는 항상 보임
- ✅ **클릭 최소화**: 3클릭 내 목표 달성

### CTA 배치
- ✅ **모든 페이지 하단**: [상담 신청] 버튼
- ✅ **플로팅 버튼**: 우측 하단 [카톡 상담] (모바일)
- ✅ **히어로 CTA**: 가장 강렬한 버튼 (레벨테스트)
