## 질문 철학

### 기존 문제점
- "가능하신가요?" → 능력 평가 같음, 부담감
- "사용해보셨나요?" → 모르면 부끄러움
- "만들어본 적 있나요?" → 성취 압박

### 개선 방향
- **"편안하게 생각해보세요"** (압박 ↓)
- **"어려워도 괜찮아요"** (안심)
- **"여러분의 이야기를 듣고 싶어요"** (겸손)

## 5개 질문 (정중한 버전)

### 질문 1: 디지털 기초
```
스마트폰이나 컴퓨터로
인터넷 검색을 해보신 적이 있으신가요?

혹시 아직 안 해보셨어도, 
저희가 함께 도와드릴 거예요.

[네, 해봤어요]  [아직은 어려워요]
```

**변경점**:
- "가능하신가요" → "해보신 적이 있으신가요" (시도에 초점)
- "아직은 어려워요" 옵션 추가 (부끄러움 ↓)
- "함께 도와드릴 거예요" (지원 약속)

---

### 질문 2: AI 경험
```
요즘 많이들 쓰는 ChatGPT나 Claude 같은
AI를 혹시 만나보신 적이 있으신가요?

처음이라면 저희가 차근차근 
친구처럼 소개해드릴게요.

[네, 알고 있어요]  [아직 못 만나봤어요]
```

**변경점**:
- "사용해보셨나요" → "만나보신 적이 있으신가요" (사람처럼)
- "처음이라면"으로 안심
- "친구처럼 소개" (친근감)

---

### 질문 3: 창작 활동
```
AI 도우미와 함께
편지를 쓰거나, 그림을 그리거나,
여행 계획을 세워보신 적이 있으신가요?

아직이시면, 저희가 함께 
첫 작품을 만들어볼 수 있어요.

[네, 해봤어요]  [아직이에요]
```

**변경점**:
- "AI로 만들어본" → "AI 도우미와 함께" (협업 느낌)
- "아직이시면" (정중한 표현)
- "첫 작품을 만들어볼 수 있어요" (기대감)

---

### 질문 4: 공유/가르침
```
가족이나 주변 분들께
AI를 알려주거나 함께 사용해보신 적이 있으신가요?

혹시 앞으로 주변 분들께
AI를 소개해드리고 싶으신가요?

[네, 해봤어요/하고 싶어요]  [아직은요]
```

**변경점**:
- "알려준 적 있나요" → "함께 사용해보신 적이" (부담 ↓)
- 미래 지향적 질문 추가 ("하고 싶으신가요")
- "소개해드리고" (겸손)

---

### 질문 5: 미래 계획
```
AI를 활용해서
블로그나 유튜브를 운영하시거나,
강사 활동을 해보시고 싶으신 생각이 있으신가요?

아니면 지금처럼 가족들과
즐겁게 AI를 사용하시는 것만으로도 충분하신가요?

[네, 도전해보고 싶어요]  [지금처럼 즐겁게 쓰고 싶어요]
```

**변경점**:
- "수익 창출" → "블로그나 유튜브 운영" (부드러운 표현)
- "충분하신가요" (지금 상태 긍정)
- 두 가지 모두 긍정적 옵션 (양자택일 X)
