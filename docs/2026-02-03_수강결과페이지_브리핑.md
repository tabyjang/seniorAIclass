## 2. 구현 계획

### 2.1 App.tsx 수정

### 2.2 results.html 새 페이지 생성

```
페이지 구조:
┌─────────────────────────────────────┐
│           헤더 (네비게이션)          │
├─────────────────────────────────────┤
│                                     │
│   🎓 수강 결과물                     │
│   시니어AI클래스에서 만든 프로덕트들  │
│                                     │
├─────────────────────────────────────┤
│                                     │
│   ┌─────────────────────────────┐  │
│   │ 프로덕트 목록 (표 형태)       │  │
│   │                             │  │
│   │ 이름 | 설명 | 상태 | 바로가기 │  │
│   │ ──── ──── ──── ────         │  │
│   │ QR코드생성기 | ... | ✅ | 열기│  │
│   │ 타자연습 | ... | ✅ | 열기    │  │
│   │ AI자서전 | ... | 🔜 | 준비중  │  │
│   │ ...                          │  │
│   └─────────────────────────────┘  │
│                                     │
├─────────────────────────────────────┤
│           푸터                       │
└─────────────────────────────────────┘
```

### 2.3 디자인 스펙
- 기존 사이트 스타일 유지 (Tailwind CSS)
- 색상: primary-500 (네이비), accent-400 (골드)
- 표: 깔끔한 카드 스타일, 호버 효과
- 상태 표시:
  - ✅ 실행 가능 → 초록 뱃지 + "열기" 버튼
  - 🔜 연결예정 → 회색 뱃지 + "준비중" 텍스트
