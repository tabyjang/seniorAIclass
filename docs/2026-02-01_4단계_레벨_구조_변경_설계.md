# 5단계 → 4단계 레벨 구조 변경 설계

> **변경일**: 2026-02-01  
> **설계자**: 그러쥬 (Gruju)  
> **목적**: 5단계 레벨을 4단계로 통합하여 시니어 학습자의 진입 장벽 낮추기

---

## 1. 변경 개요

### 현재 구조 (5단계)
```
Lv.1 AI 기초 마스터 (10강/2주)
Lv.2 프롬프트 전문가 (10강/2week)
Lv.3 콘텐츠 크리에이터 (10강/3주)
Lv.4 수익화 전문가 (10강/3주)
Lv.5 AI 비즈니스 리더 (10강/4주)
```
**총**: 50강 / 14주

### 변경 구조 (4단계)
```
Lv.1 초보자 (Beginner) - 15강/3주
Lv.2 중급자 (Intermediate) - 12강/3주
Lv.3 고급자 (Advanced) - 12강/3주
Lv.4 전문가 (Expert) - 11강/3주
```
**총**: 50강 / 12주 (2주 단축)

---

## 2. 변경 이유

### 왜 4단계인가?

| 문제점 (5단계) | 해결책 (4단계) |
|----------------|----------------|
| 레벨이 너무 많아 진입 장벽 느껴짐 | 4단계로 명확한 성장 경로 제시 |
| 시니어는 "12주"가 "14주"보다 심리적 부담 ↓ | 총 12주로 심리적 진입 장벽 ↓ |
| Lv.2/3/4 구분이 모호함 | 초보→중급→고급→전문가 명확한 단계 |
| 10강×5 구조는 반복적 느낌 | 15-12-12-11 강의 분포로 변화감 |

### 시니어 타겟에 최적화
- **"3개월 완성"**이 마케팅하기 쉬움
- **Lv.4 전문가**가 최종 목표로 명확
- 각 레벨별 **10개 프로덕트**로 성취감 유지
