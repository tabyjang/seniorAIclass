## 1️⃣ 가격 비교의 기술

### 가격 비교 도구

**Step 1: 가격 추적 사이트**
```
다나와 (danawa.com):
- 전자제품 가격 비교
- 가격 변동 그래프
- 알림 설정 가능

네이버 쇼핑 (shopping.naver.com):
- 다양한 카테고리
- 리뷰/평점 확인
- 최저가 검색

Keepa (keepa.com):
- 아마존 가격 추적
- 가격 변동 알림
- 히스토리 확인
```

**Step 2: 가격 알림 설정**
```
설정 방법:

1. 관심 상품 등록
   → 희망 가격 입력
   → 알림 방법 선택 (앱/메일)

2. 최저가 알림 받기
   → 가격이 설정값 이하로 떨어지면 알림
   → 블랙프라이데이/세일 대비

3. 가격 추적
   → 3개월 가격 추이 확인
   → 적정 구매 시점 파악
```

### 구매 타이밍

**계절별 세일:**
```
1월: 신년 세일, 겨울 옷 할인
2월: 설날 세일, 발렌타인
3월: 새학기/새출발 세일
5월: 어버이날, 가정의 달
6월: 여름 세일 시작
7-8월: 여름 대 clearance
9월: 추석 세일
10월: 블랙프라이데이 준비
11월: 블랙프라이데이/사이버먼데이
12월: 연말 세일, 크리스마스
```

## 2️⃣ 쿠폰과 적립금 활용

### 쿠폰 수집 전략

**Step 1: 기본 쿠폰 체크리스트**
```
항상 확인하는 곳:

□ 네이버 쇼핑
  - 출석 체크 쿠폰
  - 멤버십 쿠폰
  - 카드사 쿠폰

□ 쿠팡
  - 와우 멤버십 쿠폰
  - 오늘의 쿠폰
  - 카테고리 쿠폰

□ 카드사 앱
  - 삼성/현대/국민 등
  - 10% 청구할인
  - 무이자 할부
```

**Step 2: 중복 할인 노하우**
```
최대 할인 조합:

상품가: ₩100,000
- 쇼핑몰 쿠폰: 10% (-₩10,000)
- 카드사 청구할인: 10% (-₩9,000)
- 적립금 사용: (-₩5,000)
- 최종가: ₩76,000 (24% 할인)

주의:
- 쿠폰 중복 사용 가능 여부 확인
- 카드사 할인 한도 확인
- 최소 결제 금액 확인
```

### 적립금/포인트 관리

**적립금 모으기:**
```
현금처럼 쓰는 적립금:

1. 쿠팡 캐시
   - 리뷰 작성 적립
   - 출석 체크
   - 친구 초대

2. 네이버 페이 포인트
   - 네이버 쇼핑 이용
   - 네이버페이 결제
   - 멤버십 등급별 추가 적립

3. 카드 포인트
   - 삼성/현대/신한 등
   - 포인트몰에서 상품권 교환
   - 현금처럼 사용

관리법:
□ 한 달에 한 번 적립금 현황 체크
□ 소멸 임박 적립금 우선 사용
□ 적립률 높은 카드/사이트 활용
```

## 3️⃣ 리뷰 읽는 법

### 리뷰 분석법

**Step 1: 리뷰 필터링**
```
신뢰할 수 있는 리뷰:

✅ 사진/동영상 첨부 리뷰
→ 실제 사용 모습 확인

✅ 상세한 장단점
→ "좋아요"가 아닌 "이유" 설명

✅ 장기 사용 후기
→ 1개월/3개월/6개월 후기

✅ 구매 인증 리뷰
→ 실제 구매자 확인

❌ 의심 리뷰:
- 내용이 비슷한 리뷰 여러 개
- 극단적인 칭찬만
- 사진이 홍보성
```

**Step 2: 키워드 검색**
```
리뷰 내 검색:

내가 궁금한 점:
→ "배송" → 배송 속도 확인
→ "품질" → 내구성 확인
→ "사이즈" → 사이즈 정보
→ "색상" → 실제 색상 확인
→ "AS" → 고객 서비스 확인
```

### 평점 해석하기

```
평점 분석:

4.8-5.0: 매우 우수
- 대부분 만족
- 단점이 거의 없음

4.5-4.7: 우수
- 대체로 만족
- 사소한 단점 존재

4.0-4.4: 보통
- 만족도 편차 있음
- 상세 리뷰 필수 확인

3.9 이하: 주의
- 불만족 리뷰 다수
- 구매 신중 고려

⚠️ 리뷰 수 고려:
- 4.5점 (리뷰 1,000개) > 5.0점 (리뷰 10개)
```

## 4️⃣ 배송과 AS

### 배송 옵션 비교

```
배송 유형:

🚚 일반 배송 (묶음)
- 2-3일 소요
- 묶음
- 추적 가능

⚡ 로켓/프레시 배송
- 당일/익일 배송
- 쿠팡 와우 등 멤버십 필요
- 빠르지만 비용 발생

🎁 퀵/새벽 배송
- 수도권 한정
- 추가 비용 ₩3,000-5,000
- 긴급할 때만 이용

💡 배송비 절약:
- ₩30,000 이상 묶음 배송
- 멤버십 가입 (자주 이용 시)
- 주변인과 공동구매
```

### 교환/환불 정책

**사전 확인사항:**
```
반품/교환:
□ 반품 가능 기간 (보통 7일)
□ 반품 배송비 (묶음/유료)
□ 반품 가능 조건 (미개봉/사용 등)
□ 환불 소요 기간 (3-7일)

AS 정책:
□ AS 기간 (1년/2년/무상)
□ AS 접수 방법 (전화/온라인)
□ AS 수리 기간
□ 유상/무상 구분
```

## 5️⃣ 소비 기록 관리

### 가계부 작성

**간단한 가계부:**
```
엑셀/스프레드시트 템플릿:

| 날짜 | 카테고리 | 상품명 | 가격 | 결제수단 | 필요도 |
|------|----------|--------|------|----------|--------|
| 3/1 | 식비 | 장보기 | 85,000 | 카드 | 필수 |
| 3/2 | 쇼핑 | 옷 | 120,000 | 카드 | 욕구 |
| 3/3 | 교통 | 주유 | 60,000 | 카드 | 필수 |

월간 분석:
- 총 지출: ₩XXX
- 카테고리별 비율
- 충동구매 비율
- 다음 달 예산
```

**앱 활용:**
```
가계부 앱 추천:

💰 머니트리
- 자동 분류
- 카드 연동
- 리포트 제공

💰 네이버 가계부
- 네이버페이 연동
- 간편한 입력
- 묶음

💰 편한 가계부
- 심플한 UI
- 위젯 지원
- 묶음
```

### 충동구매 방지법

**24시간 법칙:**
```
구매 전 24시간 대기:

1. 장바구니에 담기
2. 24시간 대기
3. 다음 날 다시 확인
   → 여전히 필요한가?
   → 더 좋은 대안은?
   → 예산에 맞는가?

효과:
- 충동구매 70% 감소
- 합리적 소비 습관
- 후회 줄이기
```

**구매 결정 매트릭스:**
```
평가 항목 (1-5점):

필요성: _____ / 5
  → 정말 필요한가?
  
긴급성: _____ / 5
  → 지금 당장 필요한가?
  
예산 적합성: _____ / 5
  → 예산에 맞는가?
  
대안 존재: _____ / 5
  → 더 나은 대안은?
  
총점: _____ / 20

16-20점: 즉시 구매
11-15점: 고민 후 결정
10점 이하: 보류
```

## 🎯 현명한 쇼핑 루틴

### 주간 쇼핑 계획

```
월요일: 필요 목록 작성
□ 이번 주 필요한 물건 리스트업
□ 우선순위 정하기
□ 예산 설정

수요일: 가격 조사
□ 목록별 가격 비교
□ 쿠폰/할인 정보 확인
→ 장바구니에 담기 (24시간 법칙)

금요일: 구매 실행
□ 24시간 지난 물건 다시 확인
□ 진짜 필요한 것만 구매
□ 결제 수단별 혜택 확인
```

### 월간 쇼핑 리뷰

```
매월 말:

□ 지출 분석
  - 카테고리별 비율
  - 예산 대비 실제 지출
  
□ 구매 평가
  - 잘한 구매는?
  - 후회되는 구매는?
  
□ 다음 달 계획
  - 예산 재조정
  - 필요 물건 예측
```
