## 1. ChatGPT 시작하기 (5분)

### 1.1 접속 방법

**컴퓨터:**
1. 인터넷 켜기
2. 주소창에 chat.openai.com 입력
3. "Sign up" 클릭
4. 구글 계정으로 로그인

**스마트폰:**
1. 앱스토어에서 "ChatGPT" 검색
2. 설치
3. 구글 계정으로 로그인

> **무료**입니다. 카드 등록 필요 없어요!

### 1.2 화면 설명

```
[입력창] ← 여기에 글 쓰면 됩니다
          카톡 치는 것처럼!
```

정말 이게 전부입니다.

## 2. 첫 대화 예시

### 2.1 인사부터 시작

**당신:**
> 안녕, 나는 AI 처음이야

**ChatGPT:**
> 안녕하세요! AI와 대화가 처음이시군요. 천천히 알려드릴게요. 무엇이든 편하게 물어보세요!

### 2.2 간단한 질문

**질문 1: 날씨**
> 오늘 서울 날씨 어때?

**질문 2: 요리**
> 냉장고에 계란이랑 김치 있는데 뭐 해먹을까?

**질문 3: 상식**
> 비타민C 많은 과일 알려줘

### 2.3 일상 도움

**쇼핑 목록:**
> 일주일 식료품 장보기 목록 만들어줘

**일정 관리:**
> 내일 할 일 정리 좀 도와줘

**축하 메시지:**
> 친구 생일인데 축하 메시지 써줘

## 3. 자주 쓰는 표현

### 3.1 기본 표현

| 상황 | 이렇게 말하세요 |
|-----|----------------|
| 모를 때 | "~가 뭐야?" |
| 추천받고 싶을 때 | "~추천해줘" |
| 정리하고 싶을 때 | "~목록으로 만들어줘" |
| 쉽게 설명 원할 때 | "초등학생도 알게 설명해줘" |
| 더 알고 싶을 때 | "더 자세히 알려줘" |

### 3.2 마법의 문장들

**1. "쉽게 설명해줘"**
> 복리가 뭐야? 쉽게 설명해줘

**2. "예시 들어줘"**
> 절약하는 법 예시 들어줘

**3. "표로 정리해줘"**
> 비타민 종류별 효능 표로 정리해줘

**4. "짧게 요약해줘"**
> 이 기사 짧게 요약해줘

## 4. 실생활 활용 예시

### 4.1 요리

**질문:**
> 돼지고기 300g으로 저녁 메뉴 추천해줘. 만드는 법도 알려줘.

**결과:**
- 메뉴 3가지 추천
- 재료 목록
- 단계별 조리법

### 4.2 건강

**질문:**
> 허리가 아픈데 집에서 할 수 있는 스트레칭 알려줘

**결과:**
- 5가지 스트레칭 동작
- 각 동작 설명
- 주의사항

### 4.3 쇼핑

**질문:**
> 50대 엄마 생신 선물 10만원 이내로 추천해줘

**결과:**
- 선물 아이디어 5가지
- 가격대별 정리
- 구매 팁

### 4.4 여행

**질문:**
> 부산 2박 3일 여행 코스 짜줘. 맛집도 포함해서.

**결과:**
- 날짜별 일정
- 추천 맛집
- 이동 방법

## 5. 자주 하는 실수와 해결

### 5.1 "답변이 이상해요"

**원인:** 질문이 너무 짧거나 모호함

**해결:**
```
❌ "여행"
✅ "제주도 3박4일 여행 계획 세워줘. 카페 좋아하고 예산은 50만원이야"
```

### 5.2 "영어로 답해요"

**해결:**

### 5.3 "틀린 정보 같아요"

**해결:**

AI도 틀릴 수 있어요. 중요한 건 **직접 확인**하세요.
