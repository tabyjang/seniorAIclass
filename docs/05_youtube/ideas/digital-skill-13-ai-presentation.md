## 1. AI 프레젠테이션의 장점

### 1-1. 기존 방식 vs AI 방식

**기존 파워포인트:**
```
시간: 4-8시간
과정:
1. 템플릿 찾기 (30분)
2. 슬라이드 구조 설계 (30분)
3. 내용 작성 (2시간)
4. 디자인/배치 (3시간)
5. 애니메이션 (1시간)
```

**AI 프레젠테이션:**
```
시간: 30분-1시간
과정:
1. 주제 입력 (5분)
2. AI가 구조/디자인 자동 생성 (1분)
3. 내용 검토/수정 (20분)
4. 디테일 조정 (30분)
```

**결과:**
```
시간 절약: 80%
품질: 전문 디자이너 수준
일관성: 브랜드 가이드 준수
```

### 1-2. 주요 AI 프레젠테이션 도구

| 도구 | 특징 | 가격 | 한국어 |
|------|------|------|--------|
| **Gamma** | 웹사이트/문서/발표 | 묶/Pro | ✓ |
| **Beautiful.ai** | 스마트 디자인 | $12/월 | ✓ |
| **Tome** | 스토리텔링 중심 | 묶/Pro | ✓ |
| **SlidesAI** | Google Slides용 | $10/월 | ✓ |
| **Canva Magic** | Canva 통합 | 묶/Pro | ✓ |

## 2. Gamma 마스터하기

### 2-1. Gamma란?

**특징:**
```
✅ 텍스트 → 슬라이드/웹사이트/문서
✅ AI로 구조 자동 생성
✅ 실시간 협업
✅ 웹에서 바로 공유
✅ 모바일 최적화
```

**가격:**
```
🆓 묶: 400 AI 크레딧 (약 10개)
💼 Plus: $8/월 (무제한)
🏢 Pro: $15/월 (팀 기능)
```

### 2-2. 첫 프레젠테이션 만들기

**AI로 생성:**
```
1. gamma.app 가입
2. "Create new AI" 클릭
3. 주제 입력:
   "2024년 마케팅 전략: 
   AI를 활용한 디지털 마케팅"

4. 옵션 선택:
   - Type: Presentation
   - Pages: 8-10
   - Language: 한국어

5. Generate 클릭
6. AI가 자동으로:
   - 구조 설계
   - 내용 생성
   - 디자인 적용
   - 이미지 추천

7. 완성된 슬라이드 확인
8. 수정/보완
```

**텍스트로 변환:**
```
1. 기존 문서/아웃라인 준비
2. "Paste in text" 선택
3. 텍스트 붙여넣기
4. AI가 구조화
5. 슬라이드 자동 생성
```

### 2-3. 카드 기반 편집

**Gamma의 특징:**
```
전통 슬라이드가 아닌
카드(카드뉴스 형태)로 구성
```

**편집 방법:**
```
1. 카드 클릭
2. 텍스트 직접 수정
3. "/" 명령어로 요소 추가:
   - /h1: 제목
   - /img: 이미지
   - /video: 동영상
   - /web: 웹사이트 임베드
   - /embed: 외부 콘텐츠
```

**AI 편집:**
```
- 카드 선택
- "Edit with AI" 클릭
- "더 구체적으로" / "간결하게" 등 명령
- AI가 자동 수정
```

### 2-4. 테마와 디자인

**테마 선택:**
```
- Professional: 비즈니스
- Creative: 창의적
- Minimal: 미니멀
- Bold: 강렬한
- Academic: 학술
```

**브랜드 설정:**
```
Settings → Theme:
- Brand colors 설정
- Logo 업로드
- Font 선택
→ 모든 슬라이드에 일괄 적용
```

## 3. Beautiful.ai로 스마트 디자인

### 3-1. Beautiful.ai 특징

**강점:**
```
✅ Smart Slide: 자동 레이아웃
✅ 디자인 규칙 자동 적용
✅ PowerPoint export
✅ 팀 라이브러리
```

**가격:**
```
💼 Pro: $12/월
🏢 Team: $40/월
```

### 3.2 Smart Slide 사용법

**자동 레이아웃:**
```
1. "New Slide" 클릭
2. 내용 유형 선택:
   - Title
   - Bullet Points
   - Two Columns
   - Chart
   - SmartArt

3. 내용 입력
4. AI가 자동으로:
   - 최적 레이아웃 적용
   - 폰트 크기 조절
   - 여백 조정
   - 정렬 맞춤

5. 요소 추가/삭제 시
   → 자동으로 재조정
```

**예시:**
```
Before:
- 텍스트 5줄 입력
- 수동으로 폰트 조절
- 위치 맞추기 힘듦

After (Beautiful.ai):
- 텍스트 5줄 입력
- 자동으로 폰트 크기 조절
- 완벽한 여백과 정렬
```

### 3-3. 차트와 데이터

**스마트 차트:**
```
1. "Chart" 슬라이드 선택
2. 데이터 입력 (엑셀처럼)
3. 차트 타입 선택:
   - Bar
   - Line
   - Pie
   - Funnel
   - etc.
4. AI가 최적 형태로 변환
5. 값 변경 시 차트 자동 업데이트
```

## 4. Tome으로 스토리텔링

### 4-1. Tome의 특징

**강점:**
```
✅ 스토리 중심 구조
✅ AI 이미지 생성 통합 (DALL-E)
✅ 인터랙티브 요소
✅ 네러티브 최적화
```

**사용법:**
```
1. "Create" → "AI Presentation"
2. 프롬프트 입력:
   "The future of remote work:
   Trends and predictions for 2024"
3. AI가:
   - 스토리 구조 생성
   - 슬라이드별 내용
   - 관련 이미지 생성
   - 전환 효과 추가
4. 검토 및 수정
```

### 4-2. 인터랙티브 요소

**기능:**
```
- Live Embed: 유튜브, 피그마 등
- 3D Models: 삼차원 모델 삽입
- Forms: 구글폼, 에어테이블
- Annotations: 댓글/메모
```

```
   - 회사 정보 업데이트
```

## 6. 고급 팁과 워크플로우

### 6.1 AI + 전통 툴 조합

**최적의 조합:**
```
1. AI로 초안 (Gamma/Beautiful.ai)
   → 빠른 구조/디자인

2. PowerPoint로 세부 편집
   → 애니메이션
   → 복잡한 그래픽
   → 최종 마무리

3. Canva로 시각 자료
   → 인포그래픽
   → 아이콘
   → 일러스트
```

### 6.2 협업하기

**팀 작업:**
```
Gamma:
- 실시간 동시 편집
- 댓글/제안
- 버전 히스토리

Beautiful.ai:
- 팀 템플릿 라이브러리
- 슬라이드 잠금
- 승인 워크플로우
```

### 6.3 프레젠테이션 성공 팁

**내용:**
```
✅ 한 슬라이드에 하나의 메시지
✅ 6-6-6 규칙:
   - 최대 6 bullet
   - 각 6단어 이내
   - 폰트 24pt 이상
✅ 시각적 요소 70%, 텍스트 30%
```

**발표:**
```
✅ 스토리텔링 구조
✅ 청중 참여 유도
```
