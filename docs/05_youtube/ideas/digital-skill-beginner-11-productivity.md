## 1️⃣ Google 캘린더로 일정 관리 시작하기

### 왜 캘린더가 중요한가요?

**Google 캘린더**는 단순한 일정 기록을 넘어 **시간을 시각화**하고 **패턴을 파악**하는 강력한 도구입니다.

### 처음 시작하기

**Step 1**: 앱 설치 및 계정 연동
- Play Store/App Store에서 "Google Calendar" 설치
- Google 계정으로 로그인
- 기존 일정 자동 동기화

**Step 2**: 첫 일정 등록
```
입력 예시:

제목: 팀 회의
날짜: 2024년 3월 15일
시간: 오후 2:00 - 3:00
장소: 회의실 A
알림: 15분 전
참석자: 팀원 이메일 추가
```

**Step 3**: 반복 일정 설정
```
매주 같은 시간에 하는 일:
→ 반복 설정: 매주 월요일
→ 종료일: 없음 (또는 특정 날짜)
→ 알림: 10분 전
```

### 실용 팁

**색상 코드로 구분:**
```
🔴 빨강: 중요한 미팅/약속
🟡 노랑: 개인 일정
🟢 초록: 업무 일정
🔵 파랑: 학습/자기계발
```

**타임 블로킹:**
```
하루 일정 예시:

09:00-10:00 🟢 이메일 확인 및 답변
10:00-12:00 🟢 집중 업무 시간
12:00-13:00 🟡 점심 시간
13:00-15:00 🟢 회의
15:00-17:00 🟢 업무
17:00-18:00 🔵 학습/정리
```

## 2️⃣ Todoist로 할 일 관리하기

### Todoist란?

**Todoist**는 가장 직관적인 할 일 관리 앱입니다. 복잡한 기능 없이 **오늘 무엇을 해야 하는지** 명확히 보여줍니다.

### 기본 사용법

**Step 1**: 할 일 추가
```
기본 입력:
"오늘 오후 3시에 팀장님께 보고서 제출"

Todoist가 자동으로 인식:
- 작업: 보고서 제출
- 날짜: 오늘
- 시간: 오후 3시
```

**Step 2**: 프로젝트 만들기
```
프로젝트 예시:

🏢 업무
├── 보고서 작성
├── 이메일 확인
└── 회의 준비

🏠 개인
├── 장보기
├── 운전면허 갱신
└── 병원 예약

📚 학습
├── 영어 단어 암기
├── 온라인 강의 수강
└── 독서
```

**Step 3**: 우선순위 설정
```
우선순위 단계:

🔴 P1: 지금 당장 해야 함
🟡 P2: 오늘 중으로 해야 함
🔵 P3: 이번 주 중으로 해야 함
⚪ P4: 나중에 합도 됨
```

### 고급 기능

**자연어 입력:**
```
"내일 오전 10시에 #업무로 프레젠테이션 준비 p1"
→ 내일 오전 10시
→ 프로젝트: 업무
→ 우선순위: P1
```

**반복 작업:**
```
"매일 아침 8시 물 마시기"
"매주 월요일 주간 계획 세우기"
"매월 1일 전기요금 납부"
```

## 3️⃣ Forest로 집중력 키우기

### Forest의 마법

**Forest**는 게임처럼 재미있는 **집중력 타이머**입니다. 스마트폰을 사용하지 않으면 나무가 자라고, 사용하면 나무가 죽습니다!

### 사용 방법

**Step 1**: 타이머 설정
```
집중 시간 선택:
- 짧게: 10분 (이메일 확인)
- 보통: 25분 (포모도로)
- 길게: 60분 (집중 업무)
```

**Step 2**: 집중 시작
```
타이머가 작동하는 동안:
✅ 다른 앱 사용 불가 (설정 시)
✅ 나무가 자람
✅ 배경음악 재생 가능

❌ 스마트폰 사용 시:
→ 나무가 죽음
→ 연속 기록 리셋
```

**Step 3**: 숲 만들기
```
성과:
- 하루: 4그루 심음
- 한 주: 20그루 심음
- 한 달: 100그루 심음

친구와 함께:
→ 같이 집중하면 더 큰 나무 성장
→ 경쟁/협력 모드
```

### 포모도로 기법

```
25분 집중 + 5분 휴식 (1세트)

09:00-09:25 작업 A
09:25-09:30 휴식
09:30-09:55 작업 A
09:55-10:00 휴식
10:00-10:25 작업 A
10:25-10:30 휴식
10:30-10:55 작업 A
10:55-11:25 긴 휴식 (30분)

효과:
- 피로 감소
- 집중력 향상
- 지속 가능한 업무 리듬
```

## 4️⃣ Notion으로 메모 정리하기

### Notion 입문

**Notion**은 노트 + 데이터베이스 + 위키의 결합체입니다. 초보자는 **간단한 메모**부터 시작하세요.

### 기본 구조

**Step 1**: 페이지 만들기
```
페이지 예시:

📔 일기
├── 2024년 3월
│   ├── 3월 1일
│   ├── 3월 2일
│   └── ...

📝 회의록
├── 3월 1주차 회의
├── 3월 2주차 회의
└── ...

💡 아이디어
├── 블로그 주제
├── 사업 아이디어
└── ...
```

**Step 2**: 템플릿 사용
```
Notion 제공 템플릿:
- 일일 플래너
- 주간 회고
- 독서 노트
- 여행 계획
- 프로젝트 관리
```

**Step 3**: 블록 사용법
```
/ 입력 시 나타나는 블록:

/h1 → 큰 제목
/bullet → 글머리 기호
/todo → 체크박스
/image → 이미지
/table → 표
/calendar → 캘린더
```

### 초보자용 템플릿

**일일 플래너:**
```
📅 2024년 3월 15일 (금요일)

🎯 오늘의 목표
- [ ] 주요 목표 1
- [ ] 주요 목표 2
- [ ] 주요 목표 3

⏰ 시간대별 일정
- 09:00 이메일 확인
- 10:00 프로젝트 A 작업
- 13:00 점심
- 14:00 회의
- 16:00 보고서 작성

💭 오늘의 기록
- 잘한 점:
- 개선할 점:
- 내일 할 일:
```

## 5️⃣ Freedom으로 디지털 디톡스하기

### 왜 디지털 디톡스가 필요할까요?

평균적으로 사람들은 하루에 **스마트폰을 96번** 확인합니다. **Freedom**은 유해한 앱을 차단하고 집중하는 시간을 만듭니다.

### 사용법

**Step 1**: 차단 목록 만들기
```
차단할 앱/웹사이트:

SNS:
- Instagram
- Facebook
- Twitter
- TikTok

엔터테인먼트:
- YouTube
- Netflix
- 게임 앱

쇼핑:
- 쿠팡
- 11번가
- 네이버쇼핑
```

**Step 2**: 세션 시작
```
집중 세션 예시:

이름: "오전 집중 업무"
시간: 09:00 - 12:00 (3시간)
차단: SNS, 뉴스, 쇼핑
허용: 업무용 앱, 전화

긴급 상황:
→ "나가기" 버튼 (의지 테스트)
→ 5분 대기 시간
→ 해제 시 기록됨
```

**Step 3**: 스케줄 설정
```
매일 자동 실행:

아침 집중 시간:
- 매일 09:00-12:00
- SNS 차단

점심 시간:
- 매일 12:00-13:00
- 완전 해제

저녁 휴식:
- 매일 22:00-07:00
- 모든 앱 차단 (수면)
```

### 디지털 디톡스 단계별 가이드

**Level 1: 앱 정리 (1주일)**
```
□ 사용하지 않는 앱 삭제
□ 알림 끄기 (긴급 연락만 예외)
□ 홈 화면 정리 (생산성 앱만 남기기)
```

**Level 2: 시간 제한 (2주일)**
```
□ SNS 30분/일 제한
□ 유튜브 1시간/일 제한
□ 스마트폰 사용 총 3시간/일 목표
```

**Level 3: 완전 디톡스 (주말)**
```
□ 주말 아침 3시간 무스마트폰
□ 오후에는 SNS만 확인
□ 저녁 2시간 무스마트폰
```
