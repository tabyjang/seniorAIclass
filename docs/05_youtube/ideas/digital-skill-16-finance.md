## 1️⃣ Expensify - 영수증 자동화의 선구자

### Expensify란?

**Expensify**는 영수증 사진만 찍으면 **자동으로 항목 분류하고 회계 처리**하는 AI 기반 경비 관리 도구입니다. 출장이 잦은 직장인과 사업자에게 최적화되어 있습니다.

### 핵심 기능

**1. 스마트 스캔 (SmartScan)**
- 영수증 사진 → 자동 데이터 추출
- 날짜, 금액, 상호명, 품목 인식
- 99% 정확도의 OCR 기술
- 한글 영수증 완벽 지원

**2. 자동 분류**
- AI가 지출 항목 자동 분류
- 회사 정책과 자동 매칭
- 중복 청크 자동 감지

**3. 자동화된 보고서**
- 일일/주간/월간 보고서 자동 생성
- 회계 소프트웨어와 동기화
- 승인 워크플로우 자동화

**4. 결제 통합**
- Expensify Card 실시간 추적
- 법인카드 연동
- 현금/카드 구분 자동 처리

### 실습: 출장 경비 정리 자동화

**시나리오**: 한 달간의 출장 영수증 자동 처리

**Step 1**: Expensify 앱 설치
- iOS/Android 앱 설치
- 계정 생성 및 회사/개인 선택

**Step 2**: 첫 영수증 스캔
```
SmartScan 방법:
1. 앱 실행 → '+' 버튼 클릭
2. 영수증을 평평한 곳에 놓기
3. 자동 인식되거나 수동 촬영
4. 5-10초 후 데이터 확인

인식된 정보:
☑ 날짜: 2024-02-15
☑ 상호명: 스타벅스 강남점
☑ 금액: ₩18,500
☑ 카테고리: 식대 (자동 분류)
```

**Step 3**: 카테고리 및 태그 설정
```
Expense Details:
- Category: 식대 (Meals)
- Tag: 출장 (Business Trip)
- Project: Q1 마케팅 캠페인
- Location: 서울 강남
- Note: 클라이언트 미팅
```

**Step 4**: 자동화 규칙 설정
```
Automation Rules:
1. ₩50,000 이상 식대 → 추가 승인 필요
2. 택시비 → 자동으로 교통비 분류
3. 주유 영수증 → 차량번호 자동 기록
4. 숙박비 → 1박당 ₩150,000 한도 체크
```

**Step 5**: 보고서 생성
```
Monthly Report 자동 생성:
━━━━━━━━━━━━━━━━━━━━━
총 지출: ₩2,450,000
- 식대: ₩850,000 (35%)
- 교통비: ₩320,000 (13%)
- 숙박비: ₩980,000 (40%)
- 기타: ₩300,000 (12%)
━━━━━━━━━━━━━━━━━━━━━
제출일: 매월 25일 자동
승인자: 팀장님 자동 알림
```

## 2️⃣ Dext (Receipt Bank) - AI 회계 어시스턴트

### Dext란?

**Dext** (구 Receipt Bank)는 영수증과 인보이스를 **자동으로 회계 데이터로 변환**하는 AI 도구입니다. 세무사와 연동하여 실시간 장부 기장이 가능합니다.

### 핵심 기능

**1. AI 데이터 추출**
- 영수증/인보이스 자동 판독
- 공급가액/부가세 분리
- 사업자등록번호 검증
- 카드/현금 구분

**2. 자동 장부 기장**
- 회계 소프트웨어와 실시간 연동
- 전표 자동 생성
- 마감 자동화

**3. 승인 워크플로우**
- 지출 결의 승인 프로세스
- 알림 및 리마인더
- 모바일 승인

**4. 세무 리포트**
- 부가가치세 신고 자료
- 소득세 추정 계산
- 현금흐름 분석

### 실습: 사업자 회계 자동화

**시나리오**: 프리랜서 디자이너의 월간 회계 처리

**Step 1**: Dext 설정
- dext.com 가입
- 회사 정보 등록

**Step 2**: 영수증 업로드 방법 (4가지)
```
업로드 옵션:

1. 모바일 앱 촬영
   → 실시간 AI 인식

2. 이메일 전송
   → receipts@yourcompany.dext.com
   → 첨부 파일 자동 처리

3. 드래그앤드롭
   → 웹사이트에 파일 업로드

4. 자동 수집
   → 구글 드라이브/이메일 연동
   → 영수증 자동 가져오기
```

**Step 3**: 회계 계정과목 매칭
```
AI 자동 분류 학습:

Dext 인식 → 회계 계정과목
───────────────────────
카페 영수증 → 접대비
택시 영수증 → 교통비
문구점 → 소모품비
소프트웨어 구매 → 프로그램 사용료
코워킹 공간 → 임차료

(정확도가 낮으면 수동 수정 → AI 학습)
```

**Step 4**: 세무사 연동
```
Integration:
- 더존/아이큐브 ERP 연동
- 세무사 대행 자료 자동 제공
- 실시간 장부 확인
```

**Step 5**: 보고서 확인
```
Dashboard에서 확인:

이번 달 현황:
- 총 매출: ₩5,200,000
- 총 지출: ₩1,850,000
- 순이익: ₩3,350,000
- 예상 세금: ₩502,500

현금흐름:
- 현재 잔액: ₩8,200,000
- 다음 달 예상 지출: ₩2,100,000
- 안전 마진: 3.9개월
```

## 3️⃣ QuickBooks - 통합 회계 관리

### QuickBooks란?

**QuickBooks**는 미국에서 가장 많이 사용하는 **통합 회계 소프트웨어**입니다. 한국어 버전도 출시되어 국내에서도 사용 가능합니다.

### 핵심 기능

**1. AI 기반 자동 분류**
- 은행 거래내역 자동 동기화
- AI가 거래 유형 자동 분류
- 규칙 기반 자동 처리

**2. 인보이스 자동화**
- 반복 인보이스 자동 발행
- 자동 리마인더 발송
- 온라인 결제 통합

**3. 실시간 리포트**
- 손익계산서 실시간 확인
- 현금흐름 예측
- 사업 성과 분석

**4. 세무 준비**
- 세금 추정 계산
- 공제 항목 자동 식별
- 회계사 공유 기능

### 실습: 소상공인 회계 시스템 구축

**Step 1**: QuickBooks 설정
- quickbooks.intuit.com 가입
- 한국어 설정 (Settings → Language)

**Step 2**: 은행 계좌 연결
```
Bank Connections:
- 주거래 은행 선택
- 인증서/공동인증서로 연결
- 실시간 거래내역 동기화

지원 은행:
☑ KB국민은행
☑ 신한은행
☑ 우리은행
☑ 기업은행
☑ 카카오뱅크
```

**Step 3**: 카테고리 설정
```
Chart of Accounts 설정:

수익:
├── 상품 매출
├── 서비스 매출
└── 기타 수익

비용:
├── 원재료비
├── 인건비
├── 임차료
├── 광고선전비
├── 접대비
└── 기타 비용
```

**Step 4**: 자동화 규칙 생성
```
Automation Rules:

1. "OO마트" 거래
   → Category: 잡화비
   → Tag: 생활용품

2. "세무사" 입금
   → Category: 전문서비스비
   → 자동 인보이스 연결

3. 매월 1일
   → 고정비 자동 기록
   → 월세, 공과금, 구독료
```

**Step 5**: 인보이스 자동화
```
Recurring Invoice 설정:

월간 구독 고객:
- 발행일: 매월 1일
- 기한: 발행일 + 7일
- 자동 리마인더: 3일, 1일 전
- 온라인 결제 링크 포함

예상 효과:
- 청구 업무 90% 감소
- 미수금 회수율 40% 향상
```

```
Level 1: 개인 재무 (Month 1)

├── 세무사 연동
├── 자동 장부 기장
└── 실시간 재무 현황

Level 4: 고급 자동화 (Month 7+)
├── Zapier로 전사적 연동
├── 예산 알림 설정
├── 현금흐름 예측
└── 세무 최적화
```

## ⚠️ 주의사항 및 한계

### AI 회계 도구의 한계

```
1. AI 인식 오류
   → 흐릿한 영수증은 인식 실패
   → 수동 검토 필수

2. 복잡한 거래 처리 한계
   → 할부, 환불, 분할 결제
   → 수동 조정 필요

3. 세법 변화 반영 지연
   → 최신 세법 자동 반영 어려움
   → 회계사 확인 필요

4. 보안 우려
   → 금융 정보 클라우드 저장
   → 보안 인증 확인 필수
```
