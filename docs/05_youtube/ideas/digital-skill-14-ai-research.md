## 1️⃣ Perplexity AI - 스마트한 AI 검색

### Perplexity란?

Perplexity AI는 AI가 질문에 답변하면서 **정확한 출처를 함께 제공**하는 검색 엔진입니다. 단순 검색을 넘어 **질문-답변-근거**를 한 번에 제공합니다.

### 핵심 기능

**1. 출처가 명확한 답변**
- 모든 답변에 참고 출처 링크 표시
- 학술적 신뢰성 확보
- 사실 여부 직접 확인 가능

**2. 집중 모드 (Focus Mode)**
- **Academic**: 학술 논문 중심 검색
- **Writing**: 창작 글쓰기 지원
- **Wolfram**: 수학/과학 계산
- **Reddit**: 커뮤니티 의견
- **YouTube**: 동영상 콘텐츠

**3. Copilot 기능 (Pro)**
- 질문을 더 구체적으로 만들어줌
- 맥락을 이해한 심층 답변
- 후속 질문 자동 제안

### 실습: 연구 주제 검색하기

**시나리오**: "원격 근무가 직원 생산성에 미치는 영향"에 대해 알아보기

**Step 1**: Perplexity 접속 (perplexity.ai)

**Step 2**: 질문 입력하기
```
원격 근무가 직원 생산성에 미치는 영향에 대한 최근 3년간의 연구 결과를 알려주세요.
```

**Step 3**: Academic 모드 선택
- 상단 드롭다운에서 "Academic" 선택
- 논문 중심의 답변 받기

**Step 4**: 출처 확인
- 답변 하단의 참고 문헌 클릭
- 원본 논문 직접 확인

### Pro 팁 💡

```
고급 프롬프트 예시:

"디지털 마케팅에서 AI 활용의 효과에 대한 
메타분석 연구를 찾아주세요. 
- 2022년 이후 출판된 논문
- 샘플 수 1000명 이상
- 피어 리뷰 완료된 논문만"
```

## 2️⃣ Elicit - 논문 리서치 자동화

### Elicit이란?

Elicit은 **논문 검색부터 요약, 데이터 추출까지** 자동화해주는 AI 리서치 어시스턴트입니다. 수십 시간의 리서치를 몇 분으로 단축합니다.

### 핵심 기능

**1. 자동 논문 검색**
- 자연어로 질문하면 관련 논문 찾기
- 200만+ 논문 데이터베이스
- 관련성 점수로 정렬

**2. 스마트 요약**
- 논문 초록 자동 요약
- 핵심 방법론 추출
- 주요 결과 정리

**3. 데이터 추출**
- 여러 논문의 데이터를 표로 정리
- 연구 방법, 샘플 크기, 결과 비교
- CSV 낮으기 지원

**4. 연구 맵 생성**
- 주제별 연구 현황 시각화
- 지식 격파 파악
- 연구 트렌드 분석

### 실습: 체계적 문헌 고찰 만들기

**시나리오**: "마인드풀니스가 직장인 스트레스 감소에 미치는 영향" 주제로 리서치

**Step 1**: Elicit 접속 (elicit.org)

**Step 2**: 질문 입력
```
Does mindfulness meditation reduce workplace stress?
```

**Step 3**: 결과 분석
- 검색된 논문 목록 확인
- 각 논문의 "Abstract summary" 확인
- "Extract data" 버튼 클릭

**Step 4**: 데이터 추출 설정
```
추출할 데이터 항목:
☑ 연구 방법 (RCT/准실험/상관관계)
☑ 샘플 크기
☑ 참여자 특성 (직군, 연령)
☑ 중재 기간
☑ 주요 결과 (스트레스 점수 변화)
☑ 통계적 유의성
```

**Step 5**: 결과 테이블 생성
- 추출된 데이터가 표 형태로 정리됨
- Google Sheets로 낮으기 가능

### 리서치 워크플로우 템플릿

```
1단계: 주제 정의
   → Elicit에 연구 질문 입력

2단계: 초기 검색
   → 상위 20개 논문 선별

3단계: 데이터 추출
   → 주요 변수 표로 정리

4단계: 갭 분석
   → 연구 맵으로 빈틈 파악

5단계: 참고문헌 정리
   → BibTeX/APA 형식으로 낮으기
```

## 3️⃣ Consensus - 연구 기반 질문 답변

### Consensus란?

Consensus는 **과학적 연구 결과를 바탕**으로 질문에 답변하는 AI 도구입니다. "의학적으로 검증된 답변"을 제공합니다.

### 핵심 기능

**1. 연구 기반 답변**
- 2억+ 논문 분석
- 과학적 합의(consensus) 제시
- 찬반 의견 균형 있게 표시

**2. 신뢰도 표시**
- 논문 수 및 인용 횟수
- 연구 디자인 품질
- 재무 이해관계 공개 여부

**3. 간단한 설명**
- 전문 용어 설명
- 복잡한 개념 쉽게 풀이
- 시각적 요약 제공

### 실습: 건강/과학 질문에 답변받기

**시나리오**: "아침 운동 vs 저녁 운동,哪个更健康？"

**Step 1**: Consensus 접속 (consensus.app)

**Step 2**: 질문 입력
```
Is it better to exercise in the morning or evening for weight loss?
```

**Step 3**: 결과 해석
- **Consensus Meter**: 연구들의 전반적 의견
- **Summary**: AI가 정리한 핵심 결론
- **Papers**: 근거 논문 목록

**Step 4**: 상세 분석
```
체크포인트:
□ 논문 수 (충분한 증거인가?)
□ 연구 디자인 (RCT vs 관찰 연구)
□ 샘플 특성 (나이, 성별, 건강 상태)
□ 재무 이해관계 (스폰서 여부)
```

### 주의사항 ⚠️

```
Consensus의 한계:

1. 의학적 조언 대체 불가
   → 참고용으로만 사용
   → 전문의 상담 필수

2. 최신 연구 반영 지연
   → 최근 6개월 이내 논문 누락 가능

3. 맥락 이해 한계
   → 복잡한 상호작용 간과 가능
```

## 4️⃣ Semantic Scholar - AI 논문 검색 엔진

### Semantic Scholar란?

AI2(Allen Institute for AI)에서 개발한 **묶은 AI 기반 학술 검색 엔진**입니다. 2억+ 논문을 보유하고 있습니다.

### 핵심 기능

**1. 의미 기반 검색**
- 키워드 매칭을 넘어 의미 이해
- 동의어 자동 확장
- 관련 개념 추천

**2. TL;DR 요약**
- 논문 핵심을 1문장으로 요약
- 초록 읽기 전 빠른 파악
- 한글 번역 지원

**3. 인용 그래프**
- 인용 관계 시각화
- 영향력 있는 논문 파악
- 연구 흐름 이해

**4. 연구 피드**
- 관심 주제 새 논문 알림
- 저자 팔로우 기능
- 맞춤형 추천

### 실습: 연구 동향 파악하기

**시나리오**: 생성형 AI의 교육 활용 연구 동향 확인

**Step 1**: Semantic Scholar 접속 (semanticscholar.org)

**Step 2**: 검색어 입력
```
generative AI in education
```

**Step 3**: 필터 적용
```
Filters:
☑ Since 2023 (최신 연구)
☑ Highly Cited (인용 많은 논문)
☑ PDF Available (전문 확인 가능)
```

**Step 4**: TL;DR 확인
- 각 논문의 1문장 요약 읽기
- 관심 있는 논문 클릭

**Step 5**: 인용 그래프 탐색
- "Citations" 탭 클릭
- 어떤 논문이 인용되었는지 확인

- 데이터 기반 의사결정

## ⚠️ 주의사항 및 한계

### AI 도구의 한계

```
1. 환각(Hallucination) 위험
   → AI가 없는 논문을 인용할 수 있음
   → 반드시 원문 확인 필수

2. 최신성 문제
   → 학습 데이터의 시차 존재
   → 최근 3-6개월 논문 누락 가능

3. 맥락 이해 한계
   → 복잡한 방법론적 문제 간과
   → 연구의 한계점 자동 파악 어려움

4. 언어 편향
   → 영어 논문 위주
   → 비영어권 연구 누락 가능
```

### 책임 있는 사용 원칙

```
✅ 해야 할 것:
- 원본 논문 직접 확인
- 여러 소스 교차 검증
- 연구 방법론 평가
- 최신 업데이트 확인

❌ 하지 말아야 할 것:
- AI 요약만 믿고 인용
- 의학적 조언 대체
- 저작권 위반 복사
- 사실 확인 없이 공유
```
