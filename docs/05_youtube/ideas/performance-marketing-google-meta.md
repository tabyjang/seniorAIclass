디지털 광고 시장은 연간 수백조 원 규모로 성장하고 있습니다. 하지만 많은 기업들이 광고비만 쓰고 성과를 못 내는 경우가 많습니다. 오늘은 구글과 메타(페이스북/인스타그램) 광고를 중심으로 퍼포먼스 마케팅의 기초부터 최적화까지 알아보겠습니다.

## 1. 퍼포먼스 마케팅이란?

### 1.1 정의

퍼포먼스 마케팅은 **측정 가능한 성과(클릭, 전환, 매출)**에 기반하여 광고비를 집행하는 마케팅 방식입니다.

기존 마케팅과의 차이:

| 구분 | 기존 마케팅 | 퍼포먼스 마케팅 |
|------|-----------|----------------|
| 목표 | 브랜드 인지도 | 전환, 매출 |
| 측정 | 어려움 | 실시간 추적 |
| 과금 | 노출 기준 | 성과 기준 (CPC, CPA) |
| 최적화 | 캠페인 종료 후 | 실시간 최적화 |

### 1.2 핵심 지표 (KPI)

- **CTR (클릭률)**: 노출 대비 클릭 비율
- **CPC (클릭당 비용)**: 1회 클릭에 드는 비용
- **CVR (전환율)**: 클릭 대비 전환 비율
- **CPA (전환당 비용)**: 1건 전환에 드는 비용
- **ROAS (광고 수익률)**: 광고비 대비 매출 비율

> 💡 **팁**: ROAS 300%는 광고비 100만원을 쓰고 매출 300만원을 올렸다는 의미입니다.

## 2. 구글 광고 (Google Ads)

### 2.1 캠페인 유형

| 유형 | 특징 | 적합한 상황 |
|------|------|-----------|
| 검색 광고 | 키워드 검색 시 노출 | 구매 의도가 명확한 고객 |
| 디스플레이 | 웹사이트 배너 노출 | 브랜드 인지도, 리타겟팅 |
| 쇼핑 광고 | 상품 이미지+가격 노출 | 이커머스 |
| 유튜브 | 영상 광고 | 브랜드 스토리텔링 |
| 퍼포먼스 맥스 | AI 기반 자동 최적화 | 종합적인 성과 극대화 |

### 2.2 검색 광고 설정 가이드

**키워드 매치 타입**:

```
[정확히 일치]: 해당 검색어만
"구문 일치": 구문 포함 검색어
+변형 일치: 관련 변형 포함
광범위 일치: 관련 모든 검색어 (주의 필요)
```

**키워드 연구 방법**:

1. 구글 키워드 플래너로 검색량 확인
2. 경쟁 강도 및 예상 CPC 확인
3. 의도별 분류 (정보 검색 / 구매 의도)
4. 네거티브 키워드 설정

> ⚠️ **주의**: 광범위 일치만 사용하면 관련 없는 검색어에도 광고가 노출되어 예산이 낭비됩니다.

### 2.3 광고 문구 작성

**반응형 검색 광고 구조**:
- 제목 15개 (최대 30자)
- 설명 4개 (최대 90자)

**효과적인 문구 요소**:
- 키워드 포함
- 숫자/할인율 명시
- 긴급성 표현 (한정, 오늘만)
- 명확한 CTA (지금 신청, 무료 상담)

## 3. 메타 광고 (Meta Ads)

### 3.1 캠페인 목표 선택

목표에 따라 최적화 방향이 달라집니다:

| 목표 | 최적화 대상 | 적합한 상황 |
|------|-----------|-----------|
| 인지도 | 도달, 노출 | 신규 브랜드 런칭 |
| 트래픽 | 링크 클릭 | 웹사이트 방문 유도 |
| 참여 | 좋아요, 댓글 | 콘텐츠 확산 |
| 리드 | 양식 제출 | B2B, 상담 신청 |
| 전환 | 구매, 가입 | 이커머스, 서비스 |

### 3.2 타겟팅 전략

**1단계: 코어 오디언스**
- 연령, 성별, 지역
- 관심사, 행동
- 직업, 학력

**2단계: 커스텀 오디언스**
- 웹사이트 방문자 (픽셀 필수)
- 고객 리스트 업로드
- 앱 사용자
- 인스타그램/페이스북 참여자

**3단계: 유사 오디언스 (Lookalike)**
- 기존 고객과 유사한 신규 사용자
- 1%~10% 범위 선택 (1%가 가장 유사)

> 💡 **팁**: 처음에는 넓은 타겟으로 시작하고, 데이터가 쌓이면 커스텀/유사 오디언스로 좁혀가세요.

### 3.3 크리에이티브 제작

**이미지 광고 체크리스트**:
- 텍스트 20% 미만 (더 많은 노출)
- 밝고 눈에 띄는 색상
- 사람 얼굴 포함 시 효과적
- 제품 혜택 명확히 표현

**영상 광고 체크리스트**:
- 첫 3초에 핵심 메시지
- 세로형 (9:16) 또는 정사각형 (1:1)
- 자막 필수 (85%가 무음으로 시청)
- 15초 이내 권장

## 4. 전환 추적 설정

### 4.1 구글 태그 매니저 (GTM)

모든 추적 코드를 한 곳에서 관리:

1. GTM 계정 생성
2. 컨테이너 코드를 웹사이트에 설치
3. 태그, 트리거, 변수 설정
4. 전환 이벤트 추적

**필수 추적 이벤트**:
- 페이지 조회
- 버튼 클릭
- 폼 제출
- 구매 완료

### 4.2 메타 픽셀

메타 광고 최적화의 핵심:

```javascript
// 기본 페이지 뷰
fbq('track', 'PageView');

// 장바구니 담기
fbq('track', 'AddToCart', {value: 10000, currency: 'KRW'});

// 구매 완료
fbq('track', 'Purchase', {value: 50000, currency: 'KRW'});
```

### 4.3 전환 API (CAPI)

iOS 14+ 이후 쿠키 추적이 제한되면서 서버 사이드 추적이 중요해졌습니다.

- 픽셀 + CAPI 병행 사용 권장
- 이벤트 중복 제거 설정 필수
- 더 정확한 전환 데이터 확보

## 5. 최적화 전략

### 5.1 A/B 테스트

테스트 우선순위:

1. **오디언스**: 어떤 타겟이 반응하는가
2. **크리에이티브**: 어떤 이미지/영상이 효과적인가
3. **카피**: 어떤 메시지가 클릭을 유도하는가
4. **랜딩 페이지**: 어떤 페이지가 전환율이 높은가

### 5.2 예산 배분 전략

**70-20-10 법칙**:
- 70%: 검증된 캠페인 (성과가 나는 것)
- 20%: 최적화 테스트
- 10%: 새로운 실험

### 5.3 입찰 전략

| 전략 | 특징 | 권장 상황 |
|------|------|----------|
| 최저 비용 | 자동 입찰 | 초보자, 데이터 부족 |
| 비용 상한 | CPA 상한 설정 | 목표 CPA가 명확할 때 |
| 입찰 상한 | 수동 입찰 | 전문가, 세밀한 제어 |
| 타겟 ROAS | 목표 수익률 | 충분한 전환 데이터 |

## 6. 흔한 실수와 해결책

### 6.1 피해야 할 실수

| 실수 | 문제점 | 해결책 |
|------|--------|--------|
| 타겟 너무 좁음 | 노출 부족, 학습 안됨 | 최소 100만 이상 |
| 예산 너무 적음 | 최적화 불가 | 일 3만원 이상 |
| 너무 잦은 수정 | 학습 단계 초기화 | 3-7일 유지 |
| 전환 추적 누락 | 최적화 불가 | 설정 먼저 완료 |

### 6.2 성과가 안 날 때 체크리스트

1. 전환 추적이 제대로 되고 있는가?
2. 타겟 오디언스가 적절한가?
3. 크리에이티브가 매력적인가?
4. 랜딩 페이지 경험이 좋은가?
5. 제안/오퍼가 충분히 매력적인가?

## 7. 실전 캠페인 구조 예시

### 7.1 이커머스 캠페인 구조

```
캠페인 1: 신규 고객 획득
├── 광고 세트 A: 관심사 타겟
├── 광고 세트 B: 유사 타겟 1%
└── 광고 세트 C: 유사 타겟 3%

캠페인 2: 리타겟팅
├── 광고 세트 A: 장바구니 이탈 (1-3일)
├── 광고 세트 B: 조회만 한 사용자 (7일)
└── 광고 세트 C: 과거 구매자 (30일)

캠페인 3: 브랜드 인지도
└── 영상 조회 최적화
```

## 마무리
