## 1. 이미 AI를 쓰고 계십니다

### 1.1 모르고 쓰는 AI들

| 일상 활동 | 사용되는 AI |
|----------|-----------|
| 넷플릭스 추천 | 추천 알고리즘 AI |
| 네이버 검색 | 검색 AI |
| 카카오 번역 | 번역 AI |
| 쿠팡 상품 추천 | 추천 AI |
| 스마트폰 얼굴인식 | 이미지 인식 AI |
| 음성으로 문자 보내기 | 음성인식 AI |

> **오늘 하루에도 AI를 최소 10번 이상 사용하셨습니다.**

### 1.2 AI vs 스마트폰

스마트폰 처음 나왔을 때 기억하시나요?
- "터치로 전화를? 어렵겠다"
- "앱이 뭐야?"
- "카톡? 문자 있는데 왜?"

지금은요? **누구나 씁니다.**

AI도 마찬가지입니다. 지금은 어색하지만, 곧 **당연해집니다.**

## 2. AI가 무서운 진짜 이유

### 2.1 모르면 무섭다

| 두려움의 원인 | 실제 상황 |
|-------------|----------|
| "AI가 사람을 대체한다" | 도구일 뿐, 사람이 조종함 |
| "AI가 너무 똑똑하다" | 지시한 것만 함 |
| "나도 모르게 감시당한다" | 스마트폰이 더 많이 수집 |
| "복잡해서 못 쓴다" | 카톡보다 쉬움 |

### 2.2 진짜 무서운 건 AI를 안 배우는 것

> **"AI를 안 쓰면 손해입니다"**

- AI 쓰는 사람: 1시간 일
- AI 안 쓰는 사람: 5시간 일

같은 월급 받으면서 누가 더 힘들까요?

## 3. AI와 친해지는 첫걸음

### 3.1 쉬운 것부터 시작

**오늘 바로 해볼 수 있는 것:**

1. **네이버 파파고**로 영어 번역해보기
2. **카카오톡 AI 요약**으로 긴 글 요약해보기
3. **음성 메모**로 말한 거 글로 바꿔보기

### 3.2 ChatGPT 첫 대화

무료로 바로 시작 가능합니다.

**첫 대화 예시:**
```
"안녕, 나는 AI 처음이야. 뭘 할 수 있어?"
```

ChatGPT가 친절하게 설명해줍니다.

**실생활 질문 예시:**
- "오늘 저녁 메뉴 추천해줘"
- "여행 짐 싸기 체크리스트 만들어줘"
- "생일 축하 메시지 써줘"

## 4. AI 두려움 극복 5단계

### 4.1 단계별 접근

**1단계: 구경하기**
- AI 관련 유튜브 영상 보기
- "이런 것도 되네?" 감탄하기

**2단계: 따라하기**
- 간단한 튜토리얼 따라하기
- 실패해도 OK

**3단계: 써보기**
- 실생활에 1가지 적용
- 요리 레시피 물어보기

**4단계: 익숙해지기**
- 매일 1번씩 대화하기
- 다양한 질문 시도

**5단계: 활용하기**
- 업무에 적용
- 주변에 알려주기

### 4.2 실패해도 괜찮습니다

> **AI는 절대 화내지 않습니다.**

- 이상한 질문해도 OK
- 같은 거 여러 번 물어봐도 OK
- 실수해도 OK

마음껏 연습하세요.
