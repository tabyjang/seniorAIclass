## 1. 이메일 업무 자동화

### 1.1 이메일 초안 작성

**Before**: 30분 고민하며 작성
**After**: 2분 만에 완성

```
"거래처에 미팅 일정 조율 요청하는 이메일 써줘.
- 다음 주 화~목 중 오후 시간대 가능
- 1시간 정도 소요 예정
- 비대면/대면 모두 가능
톤은 정중하게."
```

### 1.2 이메일 답장

받은 이메일 복사 후:
```
"이 이메일에 대한 답장 써줘. 요청을 수락하되, 일정은 목요일로 변경 요청해줘."
```

### 1.3 이메일 요약

긴 이메일 타래 복사 후:
```
"이 이메일 스레드 요약해줘. 핵심 논의 사항과 결정된 것, 남은 것 정리해줘."
```

## 2. 보고서/문서 작성

### 2.1 보고서 구조 잡기

```
"분기별 마케팅 성과 보고서 작성하려고 해. 목차 잡아줘.
포함할 내용: SNS 성과, 광고 성과, 다음 분기 계획"
```

### 2.2 보고서 초안 작성

```
"이 데이터를 바탕으로 보고서 본문 작성해줘:
- 인스타 팔로워: 1만 → 1.5만 (50% 증가)
- 광고 클릭률: 2.1% → 3.4%
- 전환율: 1.2% → 1.8%"
```

### 2.3 보고서 다듬기

```
"이 보고서 문장을 더 매끄럽게 다듬어줘. 전문적인 톤 유지."
```

## 3. 회의 관련 업무

### 3.1 회의 안건 정리

```
"내일 팀 회의 안건 정리해줘:
1. 이번 주 프로젝트 진행 상황
2. 다음 주 일정 공유
3. 이슈 사항 논의
각 안건당 예상 소요 시간도 넣어줘."
```

### 3.2 회의록 작성

회의 녹음 → 텍스트 변환(클로바노트) 후:
```
"이 회의 내용을 회의록으로 정리해줘.
형식: 참석자, 일시, 안건, 논의 내용, 결정 사항, 액션 아이템"
```

### 3.3 회의 후 팔로업 이메일

```
"오늘 회의 내용 요약해서 팔로업 이메일 써줘.
결정된 사항, 각자 할 일, 다음 회의 일정 포함."
```

## 4. 데이터/분석 업무

### 4.1 엑셀 수식

```
"엑셀에서 A열 금액 중 B열이 '서울'인 것만 합계 내는 수식 알려줘"
```

### 4.2 데이터 해석

```
"이 매출 데이터 분석해줘:
1월: 1억, 2월: 1.2억, 3월: 0.9억, 4월: 1.5억
트렌드와 특이점 알려줘."
```

### 4.3 그래프 추천

```
"월별 매출 추이와 제품별 비중을 동시에 보여주려면 어떤 그래프가 좋을까?"
```

```

- 예산: 500만원
```
