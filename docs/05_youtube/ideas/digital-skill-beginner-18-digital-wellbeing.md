- 스마트폰 중독을 극복하고 싶으신 분
- 디지털 디톡스에 관심 있으신 분
- 기술과 삶의 균형을 찾고 싶으신 분

## ⚖️ 디지털 웰빙이란?

| 측면 | 설명 | 목표 |
|------|------|------|
| **신체적** | 눈, 자세, 수면 | 디지털 기기 사용으로 인한 신체 문제 최소화 |
| **정신적** | 스트레스, 집중력, 불안 | 기술로 인한 정신적 부담 감소 |

| **생산적** | 생산성, 창의력 | 기술을 도구로 활용, 지배당하지 않기 |

## 1️⃣ 스마트폰 중독 인식하기

### 중독 징후 체크

**자가 진단:**
```
□ 아침에 일어나자마자 스마트폰 확인
□ 스마트폰이 없으면 불안함
□ 식사 중에도 계속 확인
□ 대화 중에도 휴식 없이 확인
□ 잠들기 전까지 사용
□ 화장실에서도 사용
□ 1시간 이상 집중 못 함
□ 업무 중 자주 확인
□ 스마트폰 사용 시간이 늘어남
□ 줄이려 항상 실패

결과:
0-2개: 양호
3-5개: 주의
6-8개: 위험
9-10개: 중독 의심
```

### 사용 패턴 분석

**데이터 확인:**
```
iPhone:
설정 → 화면 사용 시간
→ 모든 활동 보기
→ 주간 리포트

Android:
설정 → 디지털 웰빙
→ 대시보드
→ 주간 리포트

확인 항목:
□ 하루 평균 사용 시간
□ 가장 많이 쓴 앱 TOP 3
□ 픽업 횟수
□ 알림 수
□ 밤 시간대 사용
```

## 2️⃣ 스크린 타임 관리

### 시간 제한 설정

**iOS 스크린 타임:**
```
설정 → 화면 사용 시간

1. 앱 한도 설정:
   → 소셜: 1시간/일
   → 게임: 30분/일
   → 엔터테인먼트: 1시간/일

2. 휴식 시간:
   → 22:00-07:00
   → 앱 사용 제한
   → 전화는 가능

3. 항상 허용:
   → 전화
   → 메시지
   → 지도
   → 친구/가족 연락처
```

**Android 디지털 웰빙:**
```
설정 → 디지털 웰빙

1. 앱 타이머:
   → 각 앱별 시간 설정
   → 초과 시 아이콘 흑백
   → 사용 제한

2. 휴식 모드:
   → 설정한 시간에 DND
   → 알림 중단
   → 화면 흑백

3. 자녀 보호:
   → Family Link 연동
   → 자녀 기기 관리
```

### 알림 관리

**알림 정리:**
```
끄기 권장:
❌ 쇼핑 앱 프로모션
❌ 게임 알림
❌ 뉴스 (긴급 아님)
❌ SNS 좋아요/댓글
❌ 광고/마케팅

남겨두기:
✅ 전화/문자
✅ 업무용 메시지
✅ 캘린더 일정
✅ 은행/카드 알림
✅ 가족/친한 친구

설정 방법:
iPhone: 설정 → 알림
Android: 설정 → 앱 → 알림
```

## 3️⃣ 디지털 디톡스 실천

### 일상 디톡스

**아침 루틴 (무스마트폰 1시간):**
```
06:00 기상
→ 스마트폰 금지

06:05 스트레칭/요가 (10분)
06:15 물 마시기 + 명상 (10분)
06:25 샤워
06:35 아침 식사
06:50 오늘의 3가지 목표 설정
07:00 스마트폰 첫 확인

효과:
- 하루 시작을 여유롭게
- 스트레스 호륨몬 감소
- 집중력 향상
```

**저녁 루틴 (무스마트폰 2시간):**
```
21:00 디지털 디톡스 시작
→ 스마트폰 침실 밖

21:05 따뜻한 샤워
21:30 스킨케어
21:45 독서/일기
22:30 명상/호흡
23:00 취침

효과:
- 수면의 질 향상
- 멜라토닌 분비 촉진
- 뇌 휴식
```

**업무 시간 디톡스:**
```
집중 시간:
09:00-11:00 집중 업무
→ SNS 앱 끄기
→ 알림 끄기
→ Forest 앱 사용

휴식 시간:
11:00-11:15 쉬는 시간
→ 스마트폰 OK
→ 물 마시기
→ 스트레칭

점심 시간:
12:00-13:00
→ 스마트폰 사용 OK
→ SNS 체크
→ 메시지 답장
```

### 주간 디톡스

**주말 무SNS:**
```
토요일:
□ 아침 3시간 무SNS
□ 점심 이후 1시간만 사용
□ 저녁 가족/친구와 시간

일요일:
□ SNS 30분만 사용
□ 일주일 정리
□ 다음 주 계획
□ 저녁 일찍 취침

활동 아이디어:
- 공원 산책
- 카페에서 책 읽기
- 친구와 만남
- 요리
- 운전/여행
- 취미 활동
```

**월간 디지털 디톡스:**
```
매월 마지막 주말:

토요일:
00:00-24:00 완전 무스마트폰
→ 아날로그 시계 사용
→ 지도 대신 종이 지도
→ 사진 대신 눈에 담기
→ 메모 대신 수첩

일요일:
□ 디지털 현상금 복귀
□ 느낀 점 기록
□ 다음 달 계획
```

## 4️⃣ 신체 건강 지키기

### 눈 건강

**20-20-20 법칙:**
```
20분마다
→ 20피트(6m) 이상 떨어진 곳
→ 20초 동안 바라보기

앱 도움:
- Eye Care 20 20 20
- Stretchly
- Flux (블루라이트 차단)

실천:
□ 스마트폰 알림 설정
□ 매시간 휴식
□ 눈 운동
```

**블루라이트 관리:**
```
야간 모드 설정:
iPhone: 설정 → 디스플레이 → Night Shift
Android: 설정 → 디스플레이 → 블루 라이트 필터

컴퓨터:
Windows: 설정 → 디스플레이 → 야간 모드
Mac: 시스템 설정 → 디스플레이 → Night Shift

Blue Light Glasses:
- 저녁 2시간 전부터 착용
- 수면의 질 향상
```

### 자세와 신체

**에르고노믹스:**
```
모니터:
- 눈높이와 같은 높이
- 50-70cm 거리
- 20도 아래로

의자:
- 무릎 90도
- 등받이 지지
- 팔걸이 사용

키보드/마우스:
- 팔꿈치 90도
- 손목 중립
- 팜레스트 사용

스마트폰:
- 눈높이로 들기
- 거북목 방지
- 20분마다 휴식
```

**스트레칭 루틴:**
```
매시간 5분:

1. 목 스트레칭
   - 좌우로 기울이기
   - 천천히 회전

2. 어깨 스트레칭
   - 어깨 돌리기
   - 팔 벌리기

3. 손목 스트레칭
   - 손목 돌리기
   - 손가띭 펼치기

4. 등 스트레칭
   - 고양이-소 자세
   - 뒤로 젖히기

5. 눈 운동
   - 상하좌우 움직이기
   - 눈 감고 휴식
```

## 5️⃣ 정신 건강과 마음 챙김

### 명상과 마음 챙김

**입문자용 명상:**
```
5분 명상법:

1. 편안한 자세
   - 의자나 바닥
   - 등을 곧게

2. 호흡에 집중
   - 숨 들이쉬기 (4초)
   - 멈추기 (4초)
   - 내쉬기 (4초)

3. 생각 관찰
   - 생각이 떠오르면 인정
   - 다시 호흡으로

4. 점진적 확장
   - 1주: 5분
   - 2주: 10분
   - 3주: 15분
   - 4주: 20분

앱 추천:
- Insight Timer (묶음)
- Calm (유료)
- Headspace (유료)
```

**디지털 명상:**
```
단계별 접근:

Level 1: 짧은 휴식
- 1분 눈 감고 호흡
- 스마트폰 알림으로 알림

Level 2: 명상 앱
- 가이드 명상 5분
- 매일 아침/저녁

Level 3: 디지털 없음
- 아날로그 명상
- 자연 소리만

Level 4: 심화 명상
- 20-30분 독립적 명상
- 일상에서 마음 챙김
```

### 디지털 불안 다루기

**FOMO 극복:**
```
FOMO (Fear Of Missing Out):
- 무엇인가 놓치고 있다는 불안
- SNS 지속 확인
- 타인과의 비교

해결책:
□ JOMO (Joy Of Missing Out)
  - 놓쳐도 괜찮다는 마음
  - 자신에게 집중

□ 그라티튜드 저널
  - 매일 감사한 일 3가지
  - 자신의 삶에 집중

□ 디지털 휴식
  - 의도적으로 끊기
  - 현재 순간 즐기기
```

**정보 과부하 관리:**
```
증상:
- 끊임없는 뉴스 확인
- SNS 중독
- 집중력 저하
- 불안감 증가

관리법:
□ 뉴스 다이어트
  - 하루 2회만 확인
  - 신뢰할 수 있는 매체만

□ SNS 클린업
  - 부정적인 계정 언팔로우
  - 팔로우 수 제한

□ 단일 태스킹
  - 멀티태스킹 금지
  - 한 가지에 집중

□ 자연 접촉
  - 야외 시간 늘리기
  - 디지털 기기 없는 시간
```
