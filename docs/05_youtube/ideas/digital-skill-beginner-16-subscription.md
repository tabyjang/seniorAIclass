## 2️⃣ 구독료 최적화 전략

### 중복 제거

**중복 서비스 찾기:**
```
영상 중복:
□ Netflix (₩17,000)
□ Disney+ (₩13,000)
□ Coupang Play (묶음 - 와우)
□ Tving (묶음 - 티빙)
□ 왓챠 (₩12,000)

→ 한 달에 하나씩만 구독
→ 번갈아 가며 사용
→ 연간 ₩200,000 절약

음악 중복:
□ Spotify (₩10,900)
□ Apple Music (₩10,900)
□ YouTube Music (묶음 - 프리미엄)
□ 멜론 (₩10,900)

→ 하나만 선택
→ 가족 공유 활용
→ 연간 ₩130,000 절약
```

### 공유/분할 결제

**가족 공유 활용:**
```
가족 공유 가능 서비스:

Netflix:
- 스탠다드: 2명 동시 시청
- 프리미엄: 4명 동시 시청
- ₩17,000 ÷ 4 = ₩4,250/인

Spotify Family:
- 최대 6명
- ₩16,900 ÷ 6 = ₩2,800/인

YouTube Premium Family:
- 최대 6명
- ₩14,900 ÷ 6 = ₩2,480/인

iCloud+:
- 200GB 공유 (최대 6명)
- ₩3,300 ÷ 6 = ₩550/인

⚠️ 주의:
- 같은 가구/가족
- 계정 공유는 위험
- 서비스별 약관 확인
```

### 연간 결제 할인

**연간 vs 월간 비교:**
```
Netflix:
- 월간: ₩17,000 × 12 = ₩204,000
- 연간: 할인 없음
→ 월간 유지

Spotify:
- 월간: ₩10,900 × 12 = ₩130,800
- 연간: ₩109,000 (2개월 묶음)
→ ₩21,800 절약

YouTube Premium:
- 월간: ₩10,400 × 12 = ₩124,800
- 연간: ₩104,000
→ ₩20,800 절약

Adobe Creative Cloud:
- 월간: ₩65,000 × 12 = ₩780,000
- 연간: ₩620,000
→ ₩160,000 절약

계산법:
연간 ÷ 12 = 월간 환산
월간 정가 × 12 = 1년 비용
```

## 3️⃣ 불필요한 구독 해지

### 해지 우선순위

**해지 기준:**
```
즉시 해지:
❌ 3개월 이상 사용 안 함
❌ 같은 기능의 묶음 대체 가능
❌ 가치보다 비용이 큼

재고:
△ 사용 빈도가 낮음
△ 구독료 인상됨
△ 대안이 있음

유지:
✅ 매일 사용
✅ 업무에 필수
✅ 가성비 좋음
```

**해지 방법:**
```
iPhone:
설정 → [이름] → 구독
→ 해지할 서비스 선택
→ 구독 취소

Android:
Play Store → 프로필 → 결제 및 구독
→ 구독 선택
→ 구독 취소

웹사이트:
각 서비스 설정 페이지
→ 구독/멤버십 관리
→ 해지/취소
```

### 해지 대신 일시 정지

**일시 정지 활용:**
```

유용한 시기:
```
