마케팅에서 가장 흔한 실수 중 하나는 신규 고객 획득에만 집중하는 것입니다. 하지만 통계에 따르면 기존 고객에게 판매할 확률은 60-70%인 반면, 신규 고객에게 판매할 확률은 5-20%에 불과합니다. 오늘은 이탈 고객을 다시 불러오고, 기존 고객의 가치를 극대화하는 리타겟팅과 CRM 마케팅 전략을 알아보겠습니다.

## 1. 리타겟팅의 이해

### 1.1 리타겟팅이란?

리타겟팅(Retargeting)은 이미 브랜드와 상호작용한 적 있는 사용자에게 다시 광고를 노출하는 전략입니다.

**일반 광고 vs 리타겟팅**:

| 구분 | 일반 광고 | 리타겟팅 |
|------|----------|---------|
| 타겟 | 새로운 잠재 고객 | 방문/관심 보인 고객 |
| 전환율 | 1-2% | 10-20% |
| 비용 | 높음 | 상대적으로 낮음 |
| 목적 | 인지/유입 | 전환/재방문 |

### 1.2 리타겟팅이 효과적인 이유

- **이미 관심 표현**: 사이트 방문 = 어느 정도 관심 있음
- **인지도 형성**: 여러 번 노출로 신뢰 구축
- **결정 지원**: 구매 고민 중인 고객에게 마지막 푸시
- **이탈 방지**: 떠난 고객을 다시 데려옴

> 💡 **팁**: 고객은 평균 7번 이상 브랜드와 접촉해야 구매를 결정합니다. 리타겟팅은 이 접촉 횟수를 효율적으로 늘립니다.

## 2. 리타겟팅 캠페인 설계

### 2.1 세그먼트별 전략

모든 방문자에게 같은 광고를 보여주지 마세요:

| 세그먼트 | 정의 | 메시지 전략 |
|----------|------|------------|
| 홈페이지만 방문 | 1페이지 이탈 | 브랜드 인지도, 베스트 상품 |
| 상품 조회 | 특정 상품 봄 | 해당 상품 + 유사 상품 |
| 장바구니 이탈 | 담고 안 삼 | 할인 쿠폰, 재고 알림 |
| 결제 이탈 | 결제 직전 이탈 | 간편 결제, 무료 배송 |
| 과거 구매자 | 한 번 구매 | 재구매 유도, 관련 상품 |

### 2.2 타이밍 전략

방문 후 시간에 따른 리타겟팅:

```
1-3일: 뜨거운 리드 → 강력한 CTA, 긴급성
4-7일: 따뜻한 리드 → 혜택 강조, 리마인더
8-14일: 미지근한 리드 → 새로운 제안, 다른 상품
15-30일: 식은 리드 → 큰 할인, 신상품 소식
30일+: 차가운 리드 → 리인게이지먼트 캠페인
```

### 2.3 빈도 관리

**리타겟팅의 적정 빈도**:
- 일일 노출: 3-5회 이하
- 주간 노출: 15-20회 이하
- 총 기간: 30-60일 (이후 제외)

> ⚠️ **주의**: 과도한 노출은 브랜드 이미지에 부정적입니다. "스토킹 당하는 느낌"을 주지 마세요.

## 3. 플랫폼별 리타겟팅 설정

### 3.1 메타 (페이스북/인스타그램)

**커스텀 오디언스 생성**:

1. 메타 광고 관리자 → 오디언스 → 커스텀 오디언스 생성
2. 소스 선택: 웹사이트 (픽셀 필요)
3. 이벤트 선택: 페이지 조회, 장바구니, 구매 등
4. 기간 설정: 1-180일

**주요 커스텀 오디언스**:
- 전체 웹사이트 방문자 (180일)
- 특정 페이지 방문자 (상품 페이지)
- 장바구니 담기 but 구매 안 함 (30일)
- 과거 구매자 (180일)

### 3.2 구글 광고

**리마케팅 리스트 생성**:

1. 구글 광고 → 도구 → 공유 라이브러리 → 오디언스 관리자
2. 세그먼트 생성 → 웹사이트 방문자
3. 규칙 설정 (URL 포함, 이벤트 등)

**구글 리마케팅 옵션**:
- 디스플레이 리마케팅: GDN 배너 광고
- 검색 리마케팅 (RLSA): 검색 광고에 리마케팅 적용
- 유튜브 리마케팅: 영상 광고
- 동적 리마케팅: 본 상품 자동 노출

### 3.3 동적 리타겟팅

고객이 본 상품을 자동으로 광고에 노출:

**필요 조건**:
- 상품 피드 (카탈로그) 연동
- 픽셀/태그 이벤트 설정
- 상품 ID 매칭

**효과**:
- 일반 리타겟팅 대비 전환율 2-3배
- 개인화된 상품 추천
- 자동화된 크리에이티브 생성

## 4. CRM 마케팅 전략

### 4.1 CRM이란?

CRM(Customer Relationship Management)은 고객 데이터를 기반으로 관계를 관리하고 마케팅하는 전략입니다.

**CRM 마케팅의 핵심 채널**:
- 이메일
- 문자(SMS)/카카오 알림톡
- 앱 푸시 알림
- 다이렉트 메일

### 4.2 고객 생애 주기 마케팅

고객 상태에 따른 맞춤 커뮤니케이션:

```
[잠재 고객] → 웰컴 시퀀스 → [신규 고객]
    ↓
온보딩 콘텐츠
    ↓
[활성 고객] → 로열티 프로그램 → [VIP 고객]
    ↓
재구매 유도
    ↓
[휴면 고객] → 윈백 캠페인 → [이탈 고객]
```

### 4.3 RFM 분석

고객을 세그먼트하는 가장 효과적인 방법:

- **R (Recency)**: 마지막 구매일 - 최근일수록 가치 높음
- **F (Frequency)**: 구매 빈도 - 자주 살수록 가치 높음
- **M (Monetary)**: 구매 금액 - 많이 쓸수록 가치 높음

**RFM 세그먼트 예시**:

| 세그먼트 | RFM 점수 | 전략 |
|----------|----------|------|
| 챔피언 | 높음-높음-높음 | VIP 대우, 독점 혜택 |
| 충성 고객 | 중간-높음-중간 | 업셀/크로스셀 |
| 유망 고객 | 높음-낮음-낮음 | 온보딩, 재구매 유도 |
| 위험 고객 | 낮음-높음-높음 | 윈백 캠페인, 설문 |
| 이탈 | 낮음-낮음-낮음 | 큰 할인 또는 제외 |

## 5. 윈백 캠페인 (이탈 고객 복귀)

### 5.1 윈백 이메일 시퀀스

**Day 1**: 우리가 보고 싶어요
```
[고객명]님, 안녕하세요.

마지막 방문 이후 [XX]일이 지났네요.
요즘 어떻게 지내세요?

저희가 보고 싶었습니다. 🙂
```

**Day 7**: 특별 혜택 제안
```
[고객명]님을 위한 특별 쿠폰을 준비했어요.

[20% 할인 쿠폰]
유효기간: 7일

돌아오시면 좋겠어요!
```

**Day 14**: 마지막 기회
```
[고객명]님, 쿠폰이 곧 만료됩니다.

[20% 할인 쿠폰] - 3일 남음

이 기회를 놓치지 마세요.
```

**Day 30**: 피드백 요청
```
[고객명]님, 솔직한 피드백을 듣고 싶어요.

저희 서비스에서 아쉬웠던 점이 있으셨나요?
1분 설문에 참여해주시면 [혜택]을 드립니다.

[설문 참여하기]
```

### 5.2 윈백 성공 사례

**사례: 이커머스 A사**

- **상황**: 6개월 이상 미구매 고객 5만 명
- **캠페인**: 3단계 윈백 이메일 + 리타겟팅 광고
- **혜택**: 30% 할인 + 무료 배송
- **결과**:
  - 12% 이메일 오픈율
  - 8% 복귀율 (4,000명 재활성화)
  - 복귀 고객 평균 구매 금액 기존보다 15% 높음

## 6. 고객 유지를 위한 로열티 프로그램

### 6.1 포인트 프로그램

기본적이지만 효과적인 방법:

- 구매 금액의 3-5% 적립
- 리뷰 작성 시 추가 포인트
- 생일/기념일 보너스 포인트
- 등급별 적립률 차등

### 6.2 VIP 프로그램

상위 고객을 위한 특별 대우:

| 등급 | 조건 | 혜택 |
|------|------|------|
| 실버 | 연 30만원 | 5% 상시 할인 |
| 골드 | 연 100만원 | 10% 할인 + 무료 배송 |
| 플래티넘 | 연 300만원 | 15% 할인 + 전담 CS + 신상 프리뷰 |

### 6.3 서프라이즈 & 딜라이트

예상치 못한 작은 감동:

- 구매 후 손편지
- 생일에 깜짝 선물
- 100번째 구매 기념 혜택
- 무작위 업그레이드

> 💡 **팁**: 작은 감동이 SNS 바이럴과 충성도를 만듭니다.

## 7. LTV (고객 생애 가치) 극대화

### 7.1 LTV 계산 공식

```
LTV = 평균 구매 금액 × 구매 빈도 × 고객 유지 기간
```

**예시**:
- 평균 구매 금액: 5만원
- 연간 구매 빈도: 4회
- 평균 유지 기간: 3년
- LTV = 50,000 × 4 × 3 = 600,000원

### 7.2 LTV를 높이는 전략

| 요소 | 전략 |
|------|------|
| 평균 구매 금액 | 업셀, 번들, 무료 배송 최소 금액 |
| 구매 빈도 | 구독 모델, 리마인더, 시즌 캠페인 |
| 유지 기간 | 로열티 프로그램, 훌륭한 CS |

### 7.3 CAC vs LTV

**건강한 비율**: LTV : CAC = 3:1 이상

- LTV가 CAC의 3배 미만: 고객 획득 비용이 너무 높음
- LTV가 CAC의 5배 이상: 마케팅 투자를 늘려도 됨
