## 1. 코딩 없이 AI 활용이 가능한 이유

### 1.1 AI의 민주화

과거: AI = 박사급 전문가만 가능
현재: AI = **누구나 대화로 사용**

| 과거 (2020년 이전) | 현재 (2024년 이후) |
|-------------------|-------------------|
| 코드 작성 필수 | 대화로 사용 |
| 전문 지식 필요 | 일반인도 가능 |
| 비싼 장비 필요 | 스마트폰이면 OK |
| 기업만 사용 | 개인도 사용 |

### 1.2 노코드 AI 시대

**대화형 AI**: ChatGPT, Claude, Bard
**이미지 AI**: Canva AI, 미리캔버스
**음성 AI**: 클로바노트, Whisper
**영상 AI**: RunwayML, Pika

> **클릭과 대화만으로 AI를 다룰 수 있습니다!**

## 2. AI 학습 로드맵 (비전공자용)

### 2.1 전체 로드맵

```
[1단계] 기초 (2주)
   AI 개념 이해 + 첫 사용
        ↓
[2단계] 활용 (4주)
   업무/일상에 적용
        ↓
[3단계] 심화 (4주)
   다양한 AI 도구 조합
        ↓
[4단계] 응용 (계속)
   나만의 활용법 개발
```

### 2.2 1단계: 기초 (2주)

**1주차: AI 개념 이해**
- AI가 뭔지 영상으로 이해 (유튜브 15분)
- ChatGPT가 어떻게 작동하는지 개념만 파악
- **코드 이해 필요 없음!**

**2주차: 첫 사용**
- ChatGPT 가입하고 대화하기
- 일상적인 질문 10개 해보기
- "오, 이게 되네?" 감각 익히기

### 2.3 2단계: 활용 (4주)

**3-4주차: 일상 활용**

| 상황 | AI 활용 |
|-----|---------|
| 요리 | 레시피 추천 |
| 여행 | 일정 계획 |
| 건강 | 운동/식단 조언 |
| 쇼핑 | 상품 비교 |

**5-6주차: 업무 활용**

| 업무 | AI 활용 |
|-----|---------|
| 이메일 | 초안 작성 |
| 보고서 | 구조 잡기 |
| 회의 | 요약/정리 |
| 기획 | 아이디어 |

### 2.4 3단계: 심화 (4주)

**7-8주차: 이미지 AI**
- Canva AI로 디자인
- DALL-E, Midjourney 체험
- 썸네일, 배너 직접 만들기

**9-10주차: 다양한 AI 도구**
- 음성 AI: 회의록 자동화
- 영상 AI: 간단한 영상 편집
- 검색 AI: Perplexity로 리서치

### 2.5 4단계: 응용 (계속)

- 나만의 프롬프트 만들기
- 업무 자동화 구축
- AI 활용 콘텐츠 제작
- 부업으로 수익화

## 5. 흔한 실수 피하기

### 5.1 피해야 할 학습 방식

**이론부터 공부하기**
→ 지루해서 포기

**코딩/수학 먼저 배우려 하기**
→ 필요 없음, 시간 낭비

**완벽하게 이해하려 하기**
→ 일단 써보면서 배우기

### 5.2 권장 학습 방식

**일단 써보기**
→ 손에 익으면 자연스럽게 이해됨

**매일 조금씩**
→ 하루 10분이라도 꾸준히

**실제 문제에 적용**
→ 내 업무/일상에 바로 활용
