## SNS, 왜 해야 하나요?

"SNS가 뭔가요?"  
"인스타그램 어떻게 쓰나요?"  
"저도 할 수 있을까요?"

SNS(Social Network Service)는 소셜 네트워크 서비스의 약자예요. 간단히 말해 **사람들과 연결되고, 소통하고, 정보를 공유하는 온라인 공간**이죠.

요즘은 SNS가 선택이 아니라 필수가 되었습니다. 가족과 소통하고, 좋아하는 브랜드의 소식을 받고, 새로운 정보를 얻는 데 모두 SNS가 쓰여요.

## 1. SNS 기초 개념

### 1-1. SNS가 뭐예요?

**정의:**  
인터넷을 통해 사람들과 연결되고, 콘텐츠를 공유하는 서비스

**쉬운 비유:**
> SNS는 디지털 시대의 **마을 버스 정류장**이에요.  
> 사람들이 모여서 소식을 나누고, 사진을 보여주고,  
> 이야기를 나누는 공간이죠. 단, 전 세계 사람들이 함께하는!

### 1-2. 주요 SNS 종류

| SNS | 특징 | 주요 사용자층 | 활용 목적 |
|-----|------|--------------|-----------|
| **인스타그램** | 사진/동영상 중심 | 10~40대 | 일상 공유, 브랜드 팔로우 |
| **페이스북** | 종합적 | 20~50대 | 가족/친구 연락, 그룹 활동 |
| **카카오스토리** | 친중 중심 | 10~40대 | 친한 친구들과 공유 |
| **트위터/X** | 짧은 글 | 10~30대 | 실시간 뉴스, 의견 표현 |
| **링크드인** | 비즈니스 | 직장인 | 커리어, 네트워킹 |
| **유튜브** | 동영상 | 전 연령 | 영상 콘텐츠 |
| **틱톡** | 짧은 동영상 | 10~20대 | 재미있는 영상 |

### 1-3. SNS의 장점

**소통**
- 먼 곳에 있는 가족/친구와 연락
- 공통 관심사를 가진 사람들과 교류
- 실시간으로 소식 공유

**정보**
- 뉴스와 트렌드 파악
- 관심 있는 분야의 정보 습득
- 브랜드/기업의 최신 소식

**기록**
- 일상을 사진으로 기록
- 추억 저장
- 나만의 디지털 앨범

**기회**
- 새로운 인맥 형성
- 취미 활동 (요리, 여행, 사진)
- 비즈니스/부업 기회

## 2. 인스타그램 시작하기

### 2-1. 인스타그램이 뭔가요?

**특징:**
- **사진과 동영상 중심**의 SNS
- **예쁜 사진**과 **짧은 글**로 소통
- **팔로우**하는 사람들의 게시물이 피드에 표시
- **좋아요**와 **댓글**로 반응
- **스토리**로 일상을 실시간 공유

**누가 쓰나요?**
- 연예인, 인플루언서
- 브랜드/기업
- 일반 사용자 (1인 가구, 부모님, 학생 등)
- 취미 활동가 (요리, 운동, 여행)

### 2-2. 가입하기

**앱 설치**
```
[Android]
Play Store → "인스타그램" 검색 → 설치

[iPhone]
App Store → "Instagram" 검색 → 설치
```

**회원가입**
```
방법 1: 이메일로 가입
1. 이메일 주소 입력
2. 성명 입력
3. 사용자 이름(아이디) 설정
4. 비밀번호 설정
5. 생일 입력
6. 가입 완료!

방법 2: 페이스북으로 가입
- 페이스북 계정이 있으면 한 번에 가입
- 연동되어 편리함
```

**프로필 설정**
```
1. 프로필 사진 등록
   - 본인 사진 또는 좋아하는 이미지
   
2. 사용자 이름(@아이디) 설정
   - @son_example
   - 고유해야 함
   - 나중에 변경 가능
   
3. 이름 입력
   - 본명 또는 닉네임
   
4. 소개(바이오) 작성
   - 나를 소개하는 짧은 글
   - 예: "🌱 식물 키우는 직장인"
          "맛집 탐방러 🍜"
          "30대 엄마의 육아일기 👶"
```

### 2-3. 기본 용어 알아보기

**핵심 용어:**
```
📱 피드 (Feed)
→ 팔로우하는 사람들의 게시물이 보이는 메인 화면

❤️ 좋아요 (Like)
→ 게시물을 마음에 들어할 때 누르는 하트 버튼

💬 댓글 (Comment)
→ 게시물에 남기는 의견이나 반응

📤 공유 (Share)
→ 다른 사람에게 게시물을 복사해서 별거나 스토리로 공유

🔖 저장 (Save)
→ 나중에 다시 보고 싶은 게시물을 보관함에 저장

👤 팔로우 (Follow)
→ 다른 사용자의 게시물을 내 피드에서 볼 수 있도록 구독

📊 팔로워 (Follower)
→ 나를 팔로우하는 사람들
```

### 2-4. 첫 게시물 올리기

**사진 올리기**
```
1. 하단 중앙 "+" 버튼 클릭
2. "게시물" 선택
3. 갤러리에서 사진 선택 (1~10장)
4. "다음" 클릭
5. 필터 적용 (선택사항)
   - 오리지널: 원본 그대로
   - 클레어: 밝고 깨끗하게
   - 파리: 감성적인 느낌
6. "다음" 클릭
7. 문구 작성
   - 사진에 대한 설명
   - 해시태그 (#여행 #맛집 등)
8. "공유" 클릭
```

**게시물 문구 팁:**
```
✅ 좋은 예:
"오랜만에 친구들과 제주도 여행 🍊
날씨도 좋고 바다도 예뻤어요!
.
.
.
#제주도 #여행스타그램 #바다 #힐링"

❌ 피할 것:
- 개인정보 (주소, 전화번호)
- 민감한 정보
- 타인 동의 없는 사진
```

### 2-5. 스토리 기능 활용

**스토리란?**
- 24시간 후 사라지는 임시 게시물
- 일상을 가볍게 공유
- 댓글보다는 메시지(디엠)로 반응

**올리는 방법:**
```
1. 메인 화면에서 오른쪽으로 스와이프
   또는 상단 왼쪽 "내 스토리" 클릭
2. 침자로로 사진/동영상 촬영
   또는 갤러리에서 선택
3. 스티커/텍스트/그림 추가 (선택)
4. "내 스토리" 클릭해서 공유
```

**스토리 꾸미기:**
```
- 스티커: 이모지, 투표, 질문, GIF
- 텍스트: 글씨 쓰기 (폰트/색상 변경)
- 하이라이트: 중요한 스토리는 프로필에 고정
- 필터: 다양한 색감 효과
```

### 2-6. 안전하게 사용하기

**프라이버시 설정**
```
설정 → 개인정보 및 보안
→ 계정 공개 범위:
   - 공개 계정: 누구나 내 게시물 확인
   - 비공개 계정: 팔로워 승인 필요

→ 댓글 관리:
   - 특정 단어 차단
   - 댓글 필터링
   
→ 활동 상태:
   - 내가 접속 중인지 표시 여부
```

**차단 및 신고**
```
불쾌한 사용자가 있다면:
1. 프로필 방문
2. 우측 상단 "..." 클릭
3. "차단" 또는 "신고" 선택

괴롭힘/스팸 신고:
- 게시물 우측 상단 "..."
- "신고" 클릭
- 사유 선택
```

## 3. 페이스북 시작하기

### 3-1. 페이스북이 뭔가요?

**특징:**
- **종합적인 SNS** (글, 사진, 동영상, 링크)
- **실명 기반** (주로 본명 사용)
- **그룹** 기능으로 관심사별 모임
- **페이지**로 브랜드/기업 팔로우
- **친구** 관계 (상호 승인 필요)

**누가 쓰나요?**
- 전 연령대 (특히 30~50대 많음)
- 가족/친구와 연락
- 동창/동호회 모임
- 브랜드 소식 확인
- 지역 커뮤니티

### 3-2. 가입하기

**앱 설치 또는 웹사이트**
```
앱:
- Play Store / App Store에서 "Facebook" 검색

웹:
- www.facebook.com 접속
```

**회원가입**
```
1. 이름 (성, 이름) 입력
2. 휴태폰 또는 이메일 입력
3. 비밀번호 설정
4. 생일 선택
5. 성별 선택
6. "가입" 클릭
7. 휴태폰/이메일 인증
```

**프로필 완성**
```
1. 프로필 사진 업로드
2. 커버 사진 설정 (배경 이미지)
3. 학력/직장 정보 (선택)
4. 거주지역 입력 (선택)
5. 소개글 작성
```

### 3-3. 친구 추가하기

**친구 찾기**
```
방법 1: 검색
- 상단 검색창에 이름 입력
- 결과에서 친구 선택
- "친구 추가" 클릭

방법 2: 연락처 동기화
- 설정 → 친구 추천
- 휴태폰 연락처와 매칭

방법 3: 아는 사람 찾기
- 메인 화면 좌측 "아는 사람 찾기"
- 추천 목록에서 추가
```

**친구 요청 관리**
```
내가 받은 요청:
- 상단 "알림" 아이콘
- 확인 후 "수" 또는 "거절"

보낸 요청 취소:
- 프로필 → "친구" 클릭
- "친구 요청 보냄" → "취소"
```

### 3-4. 게시물 올리기

**글 작성하기**
```
1. 메인 화면 상단 "무슨 생각을 하고 계신가요?" 클릭
2. 글 작성
3. 사진/동영상 첨부 (선택)
4. 위치 추가 (선택)
5. 공개 범위 설정 (선택)
   - 공개
   - 친구만
   - 친구 제외
   - 특정 친구만
6. "게시" 클릭
```

**게시물 종류**
```
📄 글만: 생각, 의견, 질문
📷 사진: 1장 또는 여러 장
🎥 동영상: 직접 촬영 또는 업로드
🔗 링크: 유튜브, 뉴스 기사 등 공유
😊 기분/활동: "기분좋음", "운동중" 등
📍 체크인: 현재 위치 공유
```

### 3-5. 그룹 활용하기

**그룹이란?**
- 공통 관심사를 가진 사람들의 모임
- 회원제 또는 공개
- 정보 공유, 질문, 소통

**그룹 찾기**
```
1. 검색창에 관심 키워드 입력
   예: "부산 맛집", "육아", "반려동물"
2. "그룹" 탭 클릭
3. 원하는 그룹 선택
4. "그룹 가입" 클릭
```

**인기 있는 그룹 유형**
```
- 지역 커뮤니티 (○○동 주민모임)
- 동창회 (○○고등학교)
- 취미/관심사 (요리, 운동, 여행)
- 육아/교육 (○○개월 아기 엄마)
- 정보 공유 (알뜰정보, 이벤트)
```

### 3-6. 페이지 팔로우

**페이지란?**
- 기업, 브랜드, 연예인, 공공기관의 공식 계정
- 친구 추가 없이 소식 받기 가능

**페이지 팔로우하기**
```
1. 좋아하는 브랜드 검색
   예: "스타벅스", "네이버", "BTS"
2. "좋아요" 버튼 클릭
3. 피드에서 해당 페이지 소식 확인
```

**추천 페이지**
```
- 뉴스: BBC, CNN, 연합뉴스
- 브랜드: Nike, Apple, 삼성전자
- 콘텐츠: TED, NatGeo
- 연예인: 좋아하는 배우/가수
```

## 4. SNS 예절과 안전

### 4-1. SNS 예절

**기본 매너:**
```
✅ 할 것:
- 타인 사진 올릴 때는 동의 받기
- 글은 명확하고 정중하게
- 댓글은 배려 있는 언어로
- 출처 표시 (인용할 때)
- 사실 확인 후 공유

❌ 피할 것:
- 과도한 개인정보 공개
- 타인 비난/욕설
- 허위 정보 유포
- 무단 도용/저작권 침해
- 스팸성 게시물
```

### 4-2. 개인정보 보호

**공개하지 말 것:**
```
❌ 주민등록번호
❌ 신용카드 정보
❌ 상세한 주소
❌ 현재 위치 실시간 (집)
❌ 자녀의 학교/학원 정보
❌ 휴가 중 집 비었다는 정보
```

**프라이버시 설정:**
```
인스타그램:
- 비공개 계정으로 설정
- 스토리 공개 범위 설정
- 댓글/메시지 필터링

페이스북:
- 게시물 공개 범위 "친구만" 설정
- 태그 승인 설정
- 위치 정보 끄기
```

### 4-3. 디지털 품앗이

**가족/친구에게:**
```
- SNS에 가족 사진 올릴 때 동의 구하기
- 부모님 SNS 가르쳐드리기
- 친구 게시물에 반응 남기기
- 의미 있는 소통하기
```

**사회적 책임:**
```
- 확인되지 않은 정보 공유 자제
- 타인의 사생활 존중
- 온라인 따돌림 반대
- 긍정적인 콘텐츠 공유
```

## 5. SNS 활용 팁

### 5-1. 콘텐츠 기획

**내 피드 컨셉 잡기**
```
예시 1: 일상 기록형
- 매일의 작은 행복
- 식사, 산책, 풍경
- "오늘의 한 줄"

예시 2: 취미 특화형
- 요리: 직접 만든 음식
- 운동: 운동 인증
- 여행: 다녀온 곳
- 독서: 읽은 책 리뷰

예시 3: 정보 공유형
- 맛집 리뷰
- 상품 리뷰
- 유용한 정보
- 알뜰 쇼핑 팁
```

### 5-2. 좋은 사진 찍기

**기본 팁:**
```
📸 조명:
- 자연광이 최고 (창가, 야외)
- 직사광선 피하기
```
