## 3. 식비 절약 (맛있게)

### 3.1 밀프렙 (Meal Prep)

주말에 한 번 요리해서 소분:
- 시간 절약
- 배달비 절약
- 건강도 챙김

**주간 밀프렙 예시:**
```
일요일 2시간 투자
→ 5일치 점심 + 3일치 저녁
→ 1인 기준 주 10만원 → 4만원
```

### 3.2 배달 앱 삭제

배달 앱의 함정:
- 배달비 3,000원
- 최소 주문 금액 맞추려 더 시킴
- 할인인 것 같지만 원래 가격보다 비쌈

> **배달 앱 삭제하고 직접 포장**하면 월 10만원 이상 절약!

### 3.3 편의점 대신 마트

편의점 삼각김밥 1,500원 × 20일 = 30,000원
마트 즉석밥 + 반찬 = 15,000원

**절약 효과: 월 10-15만원**
