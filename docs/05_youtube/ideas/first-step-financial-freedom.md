**발행일**: 2026-02-05
**카테고리**: 실무 팁
**태그**: 재테크입문, 사회초년생, 저축, 자산형성, 월급관리

---

"재테크 해야 하는 건 아는데, 어디서부터 시작해야 할지 모르겠어요."

사회초년생들에게 가장 많이 듣는 말입니다. 월급 200만원 받으면서 뭘 어떻게 해야 할까요?

오늘은 **현실적이고 구체적인** 자산 형성 로드맵을 알려드리겠습니다.

---

## 1. 재테크 시작 전 준비

### 1.1 현재 상태 파악하기

재테크 전에 반드시 알아야 할 3가지:

1. **월 수입**: 세후 실수령액
2. **월 지출**: 고정비 + 변동비
3. **현재 자산/부채**: 통장 잔고, 빚

> **팁**: 가계부 앱(뱅크샐러드, 토스)으로 3개월만 기록해보세요.

### 1.2 재테크 3단계 원칙

```
1단계: 지키기 (비상금)
    ↓
2단계: 모으기 (저축)
    ↓
3단계: 불리기 (투자)
```

> **주의**: 순서를 건너뛰면 안 됩니다!

---

## 2. 1단계: 비상금 만들기

### 2.1 왜 비상금이 먼저인가?

비상금 없이 투자하면:
- 급한 일에 투자금 인출 → 손실
- 마이너스 통장 → 이자 폭탄
- 불안해서 투자 판단 흐림

### 2.2 비상금 목표액

| 상황 | 목표 금액 |
|-----|----------|
| 최소 | 월 생활비 × 1개월 |
| **권장** | **월 생활비 × 3개월** |
| 이상적 | 월 생활비 × 6개월 |

월 지출 150만원이면 → 비상금 **450만원**

### 2.3 비상금 모으는 법

- **파킹통장**: 토스뱅크, 카카오뱅크 (연 2%대)
- **CMA**: 증권사 CMA (연 2-3%)
- **목표**: 6개월 안에 완성

---

## 3. 2단계: 저축 시스템 만들기

### 3.1 50-30-20 법칙

월급을 나누는 황금 비율:

| 항목 | 비율 | 월급 200만원 기준 |
|-----|------|------------------|
| **필수 지출** | 50% | 100만원 |
| **원하는 지출** | 30% | 60만원 |
| **저축/투자** | 20% | **40만원** |

### 3.2 자동이체의 마법

> **"월급날 다음 날, 자동이체로 저축"**

수동 저축 성공률: 30%
자동 저축 성공률: **90%**

설정 방법:
1. 월급 통장에서 저축 통장으로 자동이체
2. 월급일 다음날로 설정
3. 없는 돈이라 생각하고 생활

### 3.3 청년 우대 상품 활용

| 상품 | 금리 | 대상 |
|-----|------|------|
| 청년희망적금 | 최대 10% | 만 19-34세, 소득 조건 |
| 청년도약계좌 | 6% + 지원금 | 만 19-34세 |
| 청년내일저축계좌 | 정부 매칭 | 저소득 청년 |

> **이런 상품 먼저 채우고** 일반 저축하세요!

## 4. 3단계: 투자 시작하기

### 4.1 투자 전 체크리스트

- [ ] 비상금 3개월치 완성
- [ ] 매달 저축할 돈 확보
- [ ] 6개월 이상 안 쓸 돈
- [ ] 잃어도 생활에 지장 없는 돈

### 4.2 초보자용 투자 상품

| 상품 | 위험도 | 추천도 | 설명 |
|-----|:-----:|:-----:|------|
| **적금** | 낮음 | 중간 | 안전하지만 수익 낮음 |
| **ETF** | 중간 | 높음 | 분산투자, 추천! |
| **개별주식** | 높음 | 낮음 | 초보자 비추 |
| **코인** | 매우높음 | 매우낮음 | 고위험, 초보자 금지 |

### 4.3 추천 첫 투자: ETF

**ETF(상장지수펀드)**란?
- 여러 주식을 묶어놓은 상품
- 1개 사면 수십~수백 개 기업에 분산 투자
- 적은 금액으로 시작 가능

**초보자 추천 ETF**:
- **KODEX 200**: 한국 대표 기업 200개
- **TIGER 미국S&P500**: 미국 대표 기업 500개
- **KODEX 배당성장**: 배당 주는 기업들

### 4.4 적립식 투자

```
매달 일정 금액을 정해진 날짜에 투자

예: 매월 25일, 20만원씩 S&P500 ETF 매수
```

장점:
- 타이밍 고민 불필요
- 평균 매입 단가 효과
- 감정에 흔들리지 않음

## 5. 월급 200만원 현실적 로드맵

### 5.1 1년 차 (기초 다지기)

```
월급: 200만원
- 고정지출: 100만원 (월세, 식비 등)
- 변동지출: 60만원 (용돈, 문화생활)
- 저축: 40만원

1년 저축: 480만원
목표: 비상금 300만원 + 투자 시작
```

### 5.2 3년 차 (자산 형성)

```
월급: 230만원 (연봉 인상)
- 고정지출: 100만원
- 변동지출: 60만원
- 저축/투자: 70만원

3년 누적: 약 2,000만원
(저축 + 투자수익)
```

### 5.3 5년 차 (첫 5천만원)

```
월급: 280만원
- 저축/투자: 100만원

5년 누적: 약 5,000만원
(복리 효과 시작!)
```

### 5.4 10년 차 (1억 달성)

```
꾸준히 투자 + 복리 효과
→ 1억 돌파!

이후 복리 가속
→ 2억, 3억은 훨씬 빠름
```

## 6. 흔한 실수 피하기

### 6.1 절대 하면 안 되는 것

1. **리볼빙 사용** → 연 15%+ 이자 폭탄
2. **과소비 후 "다음 달에 더 저축"** → 안 됨
3. **대출로 투자** → 절대 금지
4. **한 종목 몰빵** → 분산 필수
5. **급하게 돈 불리려는 조급함** → 사기 당함

### 6.2 재테크 마인드

> **"느리지만 확실하게"**

- 대박 없음, 꾸준함만 있음
- 10년 뒤를 보고 투자
- 부자는 하루아침에 안 됨

## 마무리

재테크의 핵심은 간단합니다:

1. **비상금** 먼저
2. **자동 저축** 설정
3. **꾸준히 투자**
4. **시간**이 답

지금 월급이 200만원이든 500만원이든, **시작하는 것**이 가장 중요합니다.

> **"복리의 가장 큰 적은 시작하지 않는 것이다."**
