## 1️⃣ Gmail 마스터하기

### Gmail의 강력한 기능

Gmail은 단순한 이메일이 아닙니다. **스마트 분류**, **강력한 검색**, **자동화**가 가능한 생산성 도구입니다.

### 기본 설정

**Step 1**: 받은편지함 정리
```
기본 탭:
📥 받은편지함 (Primary) - 중요한 메일
🛒 프로모션 (Promotions) - 광고/쿠폰
👥 소셜 (Social) - SNS 알림
📰 업데이트 (Updates) - 뉴스/알림

설정 방법:
→ 톱니바퀴 아이콘
→ 받은편지함 탭 설정
→ 사용할 탭 선택
```

**Step 2**: 레이블 만들기
```
레이블 예시:

🏢 업무
├── 긴급
├── 보류
└── 완료

🏠 개인
├── 쇼핑
├── 여행
└── 금융

📚 학습
├── 강의
├── 자료
└── 인증
```

### 실용 기능

**필터 설정:**
```
자동 분류 규칙:

1. 특정 발신자 메일 자동 레이블링
   → from:boss@company.com → 레이블: "상사"

2. 특정 키워드 포함 메일 정리
   → subject:("청구서" OR "결제") → 레이블: "금융"

3. 뉴스레터 자동 분류
   → from:unsubscribe@* → 레이블: "뉴스레터"
```

**빠른 검색:**
```
검색어 예시:

from:boss@company.com → 특정 발신자
subject:회의 → 제목에 '회의' 포함
has:attachment → 첨부 파일 있는 메일
older_than:1y → 1년 이상 된 메일
label:업무 AND subject:긴급 → 조건 조합
```

## 2️⃣ Google Drive 완벽 정리

### 폴더 구조 설계

**Step 1**: 기본 폴더 만들기
```
내 드라이브/
├── 01_업무/
│   ├── 2024/
│   │   ├── 01_프로젝트A/
│   │   ├── 02_프로젝트B/
│   │   └── 03_회의록/
│   └── 완료/
├── 02_개인/
│   ├── 문서/
│   ├── 사진/
│   └── 자격증/
├── 03_학습/
│   ├── 온라인강의/
│   ├── 독서노트/
│   └── 자격증준비/
└── 99_임시/
    └── (정리 대기 파일)
```

**Step 2**: 파일 네이밍 규칙
```
좋은 예:
2024-03-15_프로젝트A_제안서_v1.docx
2024-03-15_회의록_주간회의.pdf
영수증_2024-03_카페_15000원.jpg

나쁜 예:
제안서.docx (날짜 없음)
회의록.pdf (내용 불명확)
KakaoTalk_20240315.jpg (의미 없음)
```

### 공유와 협업

**공유 설정:**
```
링크 공유 옵션:

🔴 제한 - 특정 사용자만
→ 이메일 주소 입력
→ 권한: 뷰어/댓글/편집자

🟡 회사/학교 내
→ 같은 조직 사용자
→ 외부 사용자는 접근 불가

🟢 누구나
→ 링크만 있으면 접근 가능
→ 보안에 주의
```

**팀 프로젝트 예시:**
```
팀 프로젝트 폴더/
├── 📋 프로젝트 개요 (모두 편집 가능)
├── 📁 기획/
│   └── (기획팀 편집, 나머지 댓글)
├── 📁 디자인/
│   └── (디자인팀 편집)
├── 📁 개발/
│   └── (개발팀 편집)
└── 📁 회의록/
    └── (모두 편집)
```

## 3️⃣ Google Docs로 문서 협업

### 기본 사용법

**Step 1**: 새 문서 만들기
```
만들기 옵션:
- 빈 문서 (Ctrl+Alt+Shift+N)
- 템플릿 갤러리
- 마크다운 붙여넣기
```

**Step 2**: 문서 서식
```
제목 계층:
제목 1 (Ctrl+Alt+1) - 문서 제목
제목 2 (Ctrl+Alt+2) - 장/섹션
제목 3 (Ctrl+Alt+3) - 절/소제목

본문:
일반 텍스트
인용구 (Ctrl+Shift+8)
코드 블록
```

### 협업 기능

**실시간 편집:**
```
협업 시나리오:

1. 문서 공유
   → 편집자 권한 부여
   → 팀원들에게 링크 전송

2. 동시 편집
   → 커서 색상으로 구분
   → 실시간 변경 확인
   → 충돌 없음

3. 댓글과 제안
   → Ctrl+Alt+M (댓글)
   → Ctrl+Alt+Shift+M (제안 모드)
   → @멘션으로 알림
```

**버전 관리:**
```
파일 → 버전 기록 → 버전 기록 보기

기능:
- 이전 버전 확인
- 특정 시점 복원
- 버전 이름 붙이기
- 변경 사항 비교
```

### 실용 템플릿

**회의록 템플릿:**
```
📅 [날짜] 주간 회의

👥 참석자
- 홍길동
- 김철수
- 박영희

📋 안건
1. 프로젝트 A 진행 상황
2. 예산 검토
3. 다음 주 일정

💬 논의 내용
1. 프로젝트 A
   - 완료된 작업: ...
   - 진행 중: ...
   - 이슈: ...

✅ 결정 사항
- [ ] 홍길동: OO까지 자료 준비
- [ ] 김철수: OO 검토

📅 다음 회의
- 3월 22일 (금) 오후 2시
```

## 4️⃣ Google Sheets로 데이터 관리

### 기초 공식

**자주 쓰는 함수:**
```
데이터 처리:
=SUM(A1:A10) → 합계
=AVERAGE(B1:B10) → 평균
=COUNT(C1:C10) → 개수
=MAX/MIN(D1:D10) → 최대/최소

조건부 계산:
=SUMIF(범위, 조건, 합계범위)
=COUNTIF(범위, 조건)
=IF(조건, 참일때, 거짓일때)

데이터 검색:
=VLOOKUP(찾을값, 범위, 열번호)
=INDEX(범위, 행, 열)
=MATCH(찾을값, 범위)
```

### 데이터 시각화

**차트 만들기:**
```
1. 데이터 범위 선택
2. 삽입 → 차트
3. 차트 유형 선택:
   - 막대형: 비교
   - 선형: 추이
   - 원형: 비율
   - 히트맵: 분포

실용 예시:
- 월간 매출 추이 (선형)
- 카테고리별 비용 (막대형)
- 예산 대비 실적 (히트맵)
```

### 협업 기능

**데이터 검증:**
```
데이터 → 데이터 검증

설정 예시:
- 허용: 목록
- 범위: 예, 아니오, 보류
→ 드롭다운 메뉴 생성

- 허용: 날짜
- 조건: 유효한 날짜
→ 잘못된 날짜 입력 방지
```

**보호 설정:**
```
도구 → 시트 보호

설정:
- 특정 셀만 편집 가능
- 수식 셀 보호
- 특정 사용자만 편집 허용
```

## 🔄 서비스 연동 팁

### Gmail → Drive → Calendar 연동

**시나리오 1: 이메일 첨부 파일 자동 저장**
```
Gmail에서:
1. 첨부 파일이 있는 이메일 확인
2. Drive 아이콘 클릭
3. 저장할 폴더 선택
→ Gmail 첨부가 Drive에 저장됨
```

**시나리오 2: 문서에서 일정 추가**
```
Google Docs에서:
1. "3월 15일 회의" 텍스트 선택
2. Smart Chips (@) 입력
3. "Date" 선택
4. 날짜 입력
→ 자동으로 캘린더 이벤트 생성 가능
```

**시나리오 3: 스프레드시트에서 이메일 발송**
```
Google Sheets에서:
1. 도구 → 스크립트 편집기
2. 간단한 코드로 이메일 자동화
3. 버튼 클릭으로 대량 메일 발송
```

## 📊 저장 공간 관리

### 용량 확인

```
drive.google.com/drive/quota

15GB 묶음:
- Gmail: 이메일 + 첨부 파일
- Drive: 모든 파일
- Photos: 원본 화질 사진

확인 방법:
→ Drive → 왼쪽 하단 저장용량
→ 클릭하면 상세 내역 확인
```

### 공간 확보 팁

```
1. 중복 파일 정리
   → 검색: type:document (또는 type:pdf)
   → 같은 이름 파일 확인 후 삭제

2. 대용량 파일 정리
   → 정렬: 파일 크기순
   → 불필요한 동영상/이미지 삭제

3. 휴지통 비우기
   → 휴지통 → 비우기
   → 30일 후 자동 삭제

4. 업그레이드 (필요시)
   Google One: 100GB (₩2,400/월)
   Google One: 2TB (₩11,900/월)
```
