## 1. AI 마케팅 개요

**자동화된 작업:**
```
✅ 광고 카피 생성
✅ 이미지/동영상 제작
✅ 타겟 오디언스 분석
✅ 입찰가 자동 조정
✅ 성과 예측
✅ 리포트 자동 생성
```

**AI의 장점:**
```
🚀 속도: 초당 수천 개 광고 테스트
🎯 정확도: 데이터 기반 의사결정
💰 비용: 효율적 예산 배분
⏰ 시간: 24/7 자동 운영
```

## 2. AI 광고 자동화

### 2-1. Meta (Facebook/Instagram) AI

**Advantage+ Shopping Campaign:**
```
AI가 알아서 하는 것:
- 타겟 오디언스 찾기
- 최적의 광고 조합 선택
- 입찰가 자동 조정
- 예산 자동 배분
- 시간대별 최적화
```

**설정 방법:**
```
1. 광고 관리자 → 캠페인 생성
2. "판매" 목표 선택
3. "Advantage+ 쇼핑" 선택
4. 소재 업로드 (이미지 10개+, 텍스트 5개+)
5. AI가 최적 조합 테스트
6. 예산 설정 (일 최소 3,000원)
```

**Advantage+ Creative:**
```
AI 최적화 항목:
- 이미지 자동 크롭
- 텍스트 자동 배치
- 필터 자동 적용
- 음악 자동 선택 (릴스)
```

### 2-2. Google Performance Max

**특징:**
```
하나의 캠페인으로:
- 검색
- 디스플레이
- 유튜브
- 디스커버
- Gmail
- 지도

AI가 알아서 분배
```

**AI 소재 생성:**
```
Google AI가:
- 입력된 자료로 광고 변형 생성
- 검색어에 맞게 헤드라인 조합
- 자동으로 성과 테스트
```

**설정:**
```
1. Google Ads → 캠페인 생성
2. "Performance Max" 선택
3. 마케팅 목표 설정
4. 소재 그룹 생성
   - 이미지 (5개)
   - 로고 (5개)
   - 동영상 (5개)
   - 텍스트 (5개 헤드라인, 5개 설명)
5. AI가 자동 최적화 시작
```

### 2-3. AI 입찰 전략

**스마트 입찰:**
```
Google Ads:
- 타겟 ROAS: 목표 수익률 달성
- 타겟 CPA: 목표 전환당 비용
- 최대 전환: 최대 전환량 추구

Meta:
- 최저 비용: 최저 CPC
- 비용 상한: 한도 내 최적화
- ROAS 목표: 수익률 목표
```

**선택 가이드:**
```
초보자: "최대 전환" 또는 "최저 비용"
중급자: "타겟 CPA"
전문가: "타겟 ROAS"
```

## 3. AI 콘텐츠 마케팅

### 3-1. AI 카피라이팅

**Jasper 활용:**
```
템플릿:
- AIDA 프레임워크
- PAS (Problem-Agitate-Solution)
- Before-After-Bridge

프롬프트 예시:
"Facebook 광고 카피 작성

제품: 홈트레이닝 앱
타겟: 30-40대 직장인
특징: 20분 운동, AI 코칭
톤: 동기부여 + 현실적

헤드라인 5개, 설명 3개"
```

**Copy.ai:**
```
특징:
- 블로그 포스트
- SNS 캡션
- 이메일 제목
- 광고 카피
- 묶 사용량 제한 있음
```

### 3-2. AI 디자인

**Canva Magic Design:**
```
기능:
- 텍스트 입력 → 디자인 자동 생성
- 브랜드 키트 자동 적용
- 크기 자동 변환
- AI 배경 제거

사용법:
1. "Magic Design" 클릭
2. 설명 입력
   "세일 프로모션 인스타그램 포스트,
   봄 시즌, 밝은 색상, 30% 할인"
3. AI가 여러 버전 생성
4. 선택 후 편집
```

**Creative Automation:**
```
Pixelcut, Creatopy:
- 하나의 디자인 → 여러 사이즈 자동
- A/B 테스트용 변형 자동 생성
- 다국어 자동 번역 + 디자인
```

### 3-3. AI 이메일 마케팅

**AI 최적화 항목:**

```
1. 발송 시간 최적화

   - 오픈율 높은 시간대

2. 제목 라인 최적화
   - AI가 클릭률 예측

3. 콘텐츠 개인화
   - 수신자 행동 기반

4. 세그먼테이션
   - AI가 자동 클러스터링
   - 예측 모델 기반
```

## 4. AI 분석 및 최적화

### 4-1. 예측 분석 (Predictive Analytics)

**GA4 예측 지표:**
```
AI가 예측하는 것:
- 구매 가능성 (Purchase Probability)
- 이탈 가능성 (Churn Probability)
- 예상 수익 (Predicted Revenue)

활용:
- 고위험 고객 조기 발견
- VIP 고객 예측
- 맞춤 마케팅 실행
```

**활용 예시:**
```
이탈 가능성 80% 이상 고객:
→ 자동으로 할인 쿠폰 발송
→ 리텐션 캠페인 실행
```

```
AI가:
```
