## 1. 비밀번호, 제대로 만들기

### 1-1. 나쁜 비밀번호 vs 좋은 비밀번호

❌ **절대 사용하면 안 되는 비밀번호:**
```
123456
password
qwerty
생일 (950101)
전화번호 뒷자리
아이디와 같은 것
"사랑해", "abc123"
```

✅ **좋은 비밀번호의 조건:**
- **12자 이상** (길수록 좋아요)
- **대문자 + 소문자 + 숫자 + 특수문자** 조합
- **의미 없는 조합** (단어 전체 사용 금지)
- **각 사이트마다 다르게**

### 1-2. 쉽게 만드는 방법

**방법 1: 문장 약자법**
```
문장: "나는 2024년에 디지털 보안을 완벽하게 지킬 거야!"

비밀번호: "n2024Edbwjk!"
(나+2024+에+디+보+완+지+!)
```

**방법 2: 키보드 패턴 + 단어**
```
단어: "Security"

비밀번호: "S3c!r1t9" 
(모양 비슷한 숫자와 특수문자로 치환)
```

**방법 3: 랜덤 생성기 사용**
- iCloud Keychain (아이폰)
- Google Password Manager (안드로이드)
- 1Password, LastPass 같은 앱

### 1-3. 비밀번호 관리의 황금률

**규칙 1: 사이트마다 다른 비밀번호**
```
❌ 모든 사이트: "Password123!"

✅ 네이버: "N4v3r#2024"
✅ 카카오: "K4k@0_Sec"
✅ 은행: "B4nk$Safe"
```

왜? 한 곳이 털리면 다른 곳도 위험해져요.

**규칙 2: 주기적 변경**
- 중요한 계정(은행, 이메일): **6개월마다**
- 일반 사이트: **1년마다**
- 해킹 의심 시: **즉시 변경**

**규칙 3: 비밀번호 관리자 사용**
```
추천 앱:
- 1Password (유료, 가장 추천)
- LastPass (묶/유료)
- Google Password Manager (묶)
- iCloud Keychain (아이폰 묶)
```

## 2. 2단계 인증 (2FA) 꼭 쓰기

### 2-1. 2단계 인증이 뭐예요?

**쉬운 설명:**
> 비밀번호(알고 있는 것) + 휴태폰(가진 것)
> 
> = 더블 잠금!

**예시:**
```
1단계: 비밀번호 입력
2단계: 휴태폰으로 온 6자리 숫자 입력
→ 로그인 성공!
```

### 2-2. 왜 중요할까요?

비밀번호가 털려도, 해커는 내 휴태폰이 없으니 로그인 못 해요!

**통계:**
- 2FA 사용 시 해킹 위험 **99% 감소**
- 구글 데이터: 2FA 사용자 중 해킹 당한 비율 **0.01%**

### 2-3. 설정하는 법

**네이버:**
```
내정보 → 보안설정 → 2단계 인증
→ 휴태폰 인증 설정
```

**카카오:**
```
더보기 → 설정 → 개인/보안 → 2단계 인증
→ 카카오톡 또는 문자 인증 선택
```

**구글:**
```
Google 계정 → 보안 → 2단계 인증
→ Google OTP 앱 설치 권장
```

**은행 앱들:**
- 대부분 설정 → 보안 → 추가 인증 메뉴
- 공인인증서 + OTP 앱 조합

### 2-4. 인증 앱 추천

| 앱 | 특징 | 플랫폼 |
|----|------|--------|
| **Google Authenticator** | 가장 보편적, 물로 | iOS/Android |
| **Microsoft Authenticator** | 마이크로소프트 계정에 최적 | iOS/Android |
| **카카오 인증기** | 한국 서비스 연동 | iOS/Android |
| **1Password** | 비번관리+인증 한번에 | iOS/Android |

**💡 팁:**  
인증 앱을 쓸 때는 **백업 코드**를 꼭 저장하세요. 휴태폰을 잃어버려도 계정을 찾을 수 있어요.

## 3. 피싱(Phishing) 사기 예방하기

### 3-1. 피싱이 뭔가요?

**정의:**  
가짜 이메일, 문자, 전화로 개인정보를 빼내는 사기

**쉬운 비유:**  
> 낚시꾼이 미끼를 던지면 물고기가 낚이듯,  
> 사기꾼이 미끼(가짜 메일)를 던지면 피해자가 속는 거예요.

### 3-2. 흔한 피싱 수법

**1) 가짜 문자 메시지 (스미싱)**
```
❌ [국민은행] 고객님 계좌가 제한되었습니다. 
   아래 링크에서 인증하세요.
   http://fake-bank.com

✅ 실제 은행은 문자에 링크를 별로 안 볂어요
✅ 의심되면 은행에 직접 전화해서 확인!
```

**2) 가짜 이메일**
```
❌ [네이버] 비밀번호 변경 안내
   From: naver-help@gmail.com
   (실제는 naver.com 도메인)

✅ 발신자 주소 꼭 확인!
✅ 정체 불명 첨부 파일 열지 말기
```

**3) 가짜 사이트**
```
❌ 주소: paypa1.com (숫자 1)
   실제: paypal.com (영문 l)

✅ 주소창 정확히 확인
✔ HTTPS (자물쇠 아이콘) 확인
```

**4) 가짜 전화 (보이스피싱)**
```
❌ "검찰입니다. 당신 계좌가 범죄에 연루되었습니다"
❌ "은행 직원입니다. 인증번호 알려주세요"

✅ 정부기관은 전화로 인증번호 안 물어요
✅ 의심되면 112 또는 해당 기관에 확인
```

### 3-3. 피싱 구별하는 법

**체크리스트:**
- [ ] 발신자 주소/번호 확인
- [ ] 링크 주소 확인 (마우스 올려보기)
- [ ] 맞춤법/띄어쓰기 확인 (피싱은 대체로 엉성)
- [ ] 요구하는 정보 확인 (은행은 비밀번호 안 물음)
- [ ] 긴급/협박 어조 ("지금 당장!") 주의

**의심되면:**
1. 일단 클릭하지 말기
2. 해당 기관에 직접 연락
3. 공식 앱/사이트에서 확인
4. 주변에 물어보기

## 4. 공개 Wi-Fi 사용 주의

### 4-1. 위험한 이유

**공공 Wi-Fi의 위험:**
- 같은 네트워크의 해커가 내 정보 훔쳐볼 수 있음
- 가짜 Wi-Fi에 연결되면 모든 데이터 유출
- 은행 거래, 쇼핑 시 특히 위험

### 4-2. 안전하게 사용하는 법

**✅ 안전:**
- 일반적인 웹서핑
- 유튜브 시청
- 뉴스 읽기

**❌ 위험 (절대 하지 마세요):**
- 인터넷 뱅킹
- 쇼핑 결제
- 중요한 이메일 확인
- 개인정보 입력

### 4-3. 꼭 써야 한다면?

**VPN 사용하기:**
```
추천 VPN:
- NordVPN (유료, 가장 안전)
- ExpressVPN (유료)
- ProtonVPN (묶/유료)
- Cloudflare WARP (묶)
```

**VPN이 뭐예요?**  
내 인터넷 연결을 암호화해서 해커가 못 보게 해주는 터널 같은 거예요.

### 4-4. 스마트폰 데이터로 하세요

공공 Wi-Fi 대신 ** LTE/5G 데이터**를 쓰는 게 더 안전해요.

## 5. 소프트웨어 업데이트는 필수

### 5-1. 왜 업데이트 해야 하나요?

**업데이트의 목적:**
- 새로운 기능 추가
- **보안 취약점 패치** ← 중요!
- 버그 수정
- 성능 개선

**예시:**  
Windows, iOS, 안드로이드에서 발견된 보안 구멍을 막기 위해 업데이트를 배포해요. 안 하면 해커가 그 구멍으로 들어올 수 있어요.

### 5-2. 어떤 걸 업데이트 해야 하나요?

**필수:**
- [ ] 운영체제 (Windows, macOS, iOS, Android)
- [ ] 웹 브라우저 (Chrome, Safari, Edge)
- [ ] 백신 프로그램
- [ ] 은행/금융 앱

**권장:**
- [ ] SNS 앱 (카카오톡, 인스타그램 등)
- [ ] 기타 앱들

### 5-3. 자동 업데이트 설정하기

**iPhone/iPad:**
```
설정 → 일반 → 소프트웨어 업데이트 
→ 자동 업데이트 → 켜기
```

**Android:**
```
설정 → 소프트웨어 업데이트 
→ 자동 다운로드 설정
```

**Windows:**
```
설정 → Windows 업데이트 
→ 고급 옵션 → 자동 업데이트 켜기
```

**macOS:**
```
시스템 설정 → 일반 → 소프트웨어 업데이트
→ 자동 업데이트 켜기
```

## 7. 백업은 안전의 기본

### 7-1. 왜 백업이 중요할까요?

** ransomware (랜섬웨어) 공격:**
- 해커가 파일을 암호화해버림
- 복구하려면 돈(비트코인) 요구
- 백업 있으면? 돈 안 내고 복구 가능!

**기기 고장/분실:**
- 휴태폰 분실해도 사진 안전
- 컴퓨터 고장나도 자료 살아있음

### 7-2. 3-2-1 백업 법칙

```
3: 데이터를 3개 복사
2: 2가지 다른 매체에 저장
1: 1개는 다른 장소에 보관
```

**실제 예시:**
```
원본: 스마트폰 사진
복사1: 외장하드
복사2: 구글 포토/네이버 클라우드
```

### 7-3. 쉬운 백업 방법

**스마트폰 사진:**
```
1) 구글 포토 자동 백업 설정
   - 설정 → 백업 및 동기화 → 켜기

2) iCloud 사진 (아이폰)
   - 설정 → 사진 → iCloud 사진 켜기

3) 네이버 MYBOX
   - 앱 설치 후 자동 백업 설정
```

**컴퓨터 문서:**
```
1) OneDrive/Google Drive/드롭박스
   - 폴더 동기화 설정

2) 외장하드 정기 복사
   - 매주 주말 한 번씩
```

## 8. 개인정보 보호 습관

### 8-1. SNS에서 주의할 점

**올리지 말 것:**
- ❌ 주민등록번호 (일부라도)
- ❌ 여권/신분증 사진
- ❌ 신용카드 (번호나 사진)
- ❌ 상세한 주소
- ❌ 현재 위치 실시간 공유
- ❌ 휴가 중 집 비었다는 정보

**조심해서 올릴 것:**
- ⚠️ 자녀 사진 (과도한 공유는 자제)
- ⚠️ 출퇴근 루트
- ⚠️ 회사 내부 사진

### 8-2. 앱 권한 관리

**불필요한 권한 끄기:**
```
[안드로이드]
설정 → 애플리케이션 → 권한 관리
→ 각 앱별로 권한 확인 및 조정

[iOS]
설정 → 개인 정보 보호 및 보안 
→ 권한 설정
```

**의심스러운 권한:**
- 계산기 앱이 연락처 접근?
- 손전등 앱이 위치 정보?
- 게임 앱이 통화 녹음?

→ **권한 거부!**

### 8-3. 계정 정리하기

**안 쓰는 계정은 삭제:**
- 오래된 쇼핑몰 회원정보
- 사용하지 않는 SNS 계정
- 필요없는 앱 회원가입

**정기 확인:**
- 가입한 사이트 목록 정리
- 개인정보 제공 동의 내역 확인
- 광고 수신 거부 설정

## 9. 문제가 생겼을 때

### 9-1. 의심스러운 상황

**이럴 때:**
- 비밀번호가 자꾸 틀린다
- 모르는 기기에서 로그인 알림
- 계좌에서 모르는 출금
- 이상한 이메일/SMS가 온다

### 9-2. 즉시 해야 할 일

**1단계: 차단**
- 해당 계정 비밀번호 즉시 변경
- 2FA 설정 확인
- 연결된 기기 로그아웃

**2단계: 확인**
- 계정 활동 기록 확인
- 이상 거래 내역 확인
- 다른 계정도 비밀번호 변경

**3단계: 신고**
- 해당 서비스 고객센터 연락
- 금융감독원 (금융 사기 시)
- 경찰청 사이버안전국 (112)
- 한국인터넷진흥원 (118)

### 9-3. 신고 연락처

| 기관 | 연락처 | 상황 |
|------|--------|------|
| **경찰청 사이버안전국** | 112 | 해킹, 사이버 범죄 |
| **금융감독원** | 1332 | 금융 사기 |
| **한국인터넷진흥원** | 118 | 피싱, 악성코드 |
| **개인정보침해 신고센터** | 118 | 개인정보 유출 |
