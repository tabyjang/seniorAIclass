## 1. AI 음성 기술의 종류

### 1.1 음성 → 텍스트 (STT: Speech-to-Text)

**원리:**
- 음성 파형을 분석
- AI가 음소(Phoneme) 인식
- 문맥을 고려한 단어 변환
- 문장 완성

**활용:**
- 음성 메모
- 회의록 작성
- 글쓰기
- 자막 생성

### 1.2 텍스트 → 음성 (TTS: Text-to-Speech)

**원리:**
- 텍스트를 음소로 분해
- AI가 자연스러운 억양 생성
- 음성 합성

**활용:**
- 오디오북 제작
- 유튜브 내레이션
- 접근성 향상 (시각장애인)
- IVR (전화 응대)

### 1.3 음성 → 음성 (실시간 번역)

**원리:**
- 음성 인식 → 텍스트 변환
- AI 번역
- 목소리 유지하며 음성 합성

**활용:**
- 글로벌 회의
- 여행 통역
- 라이브 스트리밍

## 3. 실전 활용 시나리오

### 3.1 음성으로 블로그/문서 작성

**프로세스:**
```
[음성 녹음] → [AI 인식] → [텍스트 변환] → [편집]
```

**도구:**
- **Mac**: 내장 음성 입력 (Dictation) + ChatGPT 정제
- **Windows**: Windows + H (음성 입력)
- **iPhone**: 키보드 마이크 아이콘
- **Android**: Gboard 음성 입력

**효율성 비교:**
```
타이핑 속도: 분당 40단어
음성 입력: 분당 100-150단어
개선율: 250-375%
```

**실전 팁:**
```
✅ 문단 단위로 말하기
✅ 구두점 말하기: "쉼표" "마침표" "줄바꿈"
✅ 명확하게 발음
✅ 조용한 환경에서 녹음
```

**AI 정제:**
```
ChatGPT/Claude에게:
"다음 음성 인식 텍스트를 다듬어줘.
불필요한 말더듬 제거하고, 
문장을 자연스럽게 연결해줘.

[음성 인식 원문]"
```

### 3.2 회의 자동 기록

**Naver Clova Note 활용:**

**설정:**
```
1. 클로바 노트 앱 설치
2. 회의 시작 시 '녹음' 클릭
3. 실시간으로 텍스트 변환
4. 회의 종료 후 AI 요약
```

**주요 기능:**
- 스피커 자동 구분 (A, B, C)
- 실시간 텍스트 표시
- 핵심 내용 하이라이트
- 요약 및 액션 아이템 추출

**팁:**
```
✅ 회의 시작 전 "참석자 소개" 
   → 스피커 구분 정확도 향상
✅ 중요 발언은 천천히 명확하게
✅ 전문 용어는 사전에 등록
```

## 4. 음성 품질 향상 기법

### 4.1 녹음 환경 최적화

**✅ 좋은 환경:**
```
- 조용한 공간
- 마이크와 15-30cm 거리
- 에코가 적은 방
- 배경 소음 최소화
- 적절한 볼륨 (녹음기 피크 -12dB ~ -6dB)
```

**장비 추천:**
```
입문: 스마트폰 이어폰 마이크
중급: USB 마이크 (Blue Yeti, Audio-Technica ATR2500)
고급: XLR 마이크 + 오디오 인터페이스
```

### 4.2 발음 및 억양 팁

**정확도 높이기:**
```
✅ 적당한 속도로 말하기
   - 너무 빠르면 인식률 저하
   - 너무 느리면 끊김 현상

✅ 명확한 발음
   - 앞뒤 단어 구분
   - 받침 정확히

✅ 구두점 발음
   - "쉼표" → ,
   - "마침표" → .
   - "줄바꿈" → 새 문단
   - "따옴표" → ""
```

### 4.3 AI 사전 학습

**Custom Vocabulary:**
```
회사/제품 특화 용어 등록:
- 회사명
- 제품명
- 전문 용어
- 사람 이름

등록 방법:
Otter.ai → Manage Vocabulary
Naver Clova → 단어장 등록
```

## 5. 고급 활용: Whisper API

### 5.1 개발자를 위한 자동화

**Python 코드 예시:**
```python
import openai

# 오디오 파일을 텍스트로 변환
audio_file = open("meeting.mp3", "rb")
transcript = openai.Audio.transcribe(
    model="whisper-1",
    file=audio_file,
    language="ko",
    response_format="srt"  # 자막 형식
)

print(transcript.text)
```

**자동화 시나리오:**
```
1. 회의 녹음 파일 자동 수집
2. Whisper API로 텍스트 변환
3. ChatGPT로 요약
4. Notion/Slack으로 자동 전송
5. 액션 아이템 추적
```

### 5.2 실시간 스트리밍

**WebRTC + Whisper:**
```
온라인 회의 실시간 자막:
1. 회의 오디오 스트림 캡처
2. Whisper API 실시간 전송
3. 자막 실시간 표시
4. 다국어 번역 추가
```
