## 데이터, 그냥 모으면 쓸모없어요

"데이터는 많은데... 뭘 어떻게 봐야 하죠?"  
"엑셀 그냥 표 만들기만 하는데..."  
"AI가 분석해준다는데, 어떻게 시켜야 할지 몰라요"

데이터는 **현대의 원유**예요. 그런데 그냥 두면 아무 쓸모가 없죠.  
**정제하고, 분석하고, 인사이트를 뽑아내야** 비로소 가치가 생깁니다.

이 강의에서는 엑셀과 AI를 활용해 **실전 데이터 분석**을 배웁니다.

## 1. 데이터 분석이 뭔가요?

### 1-1. 정의와 목적

**정의:**  
데이터를 수집 → 정제 → 분석 → **의사결정에 활용**하는 과정

**쉬운 비유:**
> 데이터 분석은 **탐정이 단서를 찾는 것**과 같아요.  
> 범인(문제)을 찾기 위해 현장(데이터)을 뒤지고,  
> 지문(패턴)을 채취하고, 증거(인사이트)를 수집해서  
> 사건(상황)을 해결하는 거죠.

### 1-2. 왜 필요할까요?

**직관 vs 데이터:**
```
❌ 직관: "이번 달 매출이 떨어진 것 같아"
✅ 데이터: "이번 달 매출은 지난 달 대비 15% 감소했고,
           주요 원인은 신규 고객 유입 30% 감소입니다.
           해결책은 마케팅 채널 A의 예산을 20% 증액하는 것입니다."
```

**데이터 분석의 힘:**
- 객관적 근거 제시
- 문제의 원인 파악
- 미래 예측 가능
- 효율적 자원 배분

## 2. 데이터 분석 5단계

### Step 1: 데이터 수집

**수집 방법:**
```
📊 낶부 데이터:
- ERP/CRM 시스템
- 웹사이트 애널리틱스 (GA4)
- 엑셀 파일
- 데이터베이스

🌐 외부 데이터:
- 공공 데이터 포털
- 업계 보고서
- API 연동
- 설문조사
```

**주의사항:**
```
✅ 데이터 품질 확인
- 누락값 (Missing data)
- 이상값 (Outlier)
- 중복값
- 오타/오류
```

### Step 2: 데이터 정제 (Cleaning)

**정제 작업:**
```
🧹 누락값 처리:
- 삭제 (값이 적을 때)
- 대체 (평균, 중앙값 등)
- 추정 (회귀분석 등)

🧹 이상값 처리:
- 확인 (정말 오류인지)
- 변환 (로그 변환 등)
- 삭제 (극단적일 때)

🧹 형식 통일:
- 날짜: 2024-01-01 vs 2024.01.01
- 단위: 원 vs 만원
- 카테고리: 남/여 vs 남성/여성
```

**엑셀 활용:**
```
함수:
=TRIM() - 공백 제거
=CLEAN() - 특수문자 제거
=IF(ISBLANK(), "N/A", A1) - 빈칸 처리
=REMOVE_DUPLICATES - 중복 제거
```

### Step 3: 데이터 탐색 (EDA)

**EDA (Exploratory Data Analysis):**
```
데이터를 둘러복며 패턴을 찾는 과정
```

**기본 통계량:**
```
📈 중심 경향:
- 평균 (Mean): 전체 평균
- 중앙값 (Median): 중간 값
- 최빈값 (Mode): 가장 많이 나온 값

📈 산포도:
- 최소/최대값
- 범위 (Range)
- 표준편차 (Standard Deviation)
```

**엑셀 활용:**
```
데이터 → 데이터 분석 → 기술 통계
또는 피벗 테이블 활용
```

### Step 4: 분석 및 시각화

**분석 방법 선택:**
```
🔍 기술 통계: 현재 상황 파악
🔍 추론 통계: 원인 분석
🔍 예측 분석: 미래 예측
🔍 상관 분석: 관계 파악
```

**시각화:**
```
📊 막대 그래프: 비교
📈 선 그래프: 추세
🥧 파이 차트: 비율
🔥 히트맵: 밀도
📉 산점도: 상관관계
```

### Step 5: 인사이트 도출 및 공유

**So What? 테스트:**
```
"그래서 뭐요?" (So what?)
→ 이 데이터가 의사결정에 어떤 도움을 주나?

예:
"이번 달 매출이 20% 감소했습니다"
→ So what?
→ "신규 마케팅 채널 투자가 필요합니다"
→ "예산을 A 채널에서 B 채널로 옮겨야 합니다"
```

## 3. 엑셀로 실전 분석하기

### 3-1. 피벗 테이블 (Pivot Table)

**피벗 테이블이 뭔가요?**
```
대용량 데이터를 빠르게 요약/분석하는 도구
```

**만드는 법:**
```
1. 데이터 선택 (Ctrl+A)
2. 삽입 → 피벗 테이블
3. 행/열/값/필터 설정
4. 원하는 형태로 드래그
```

**실전 예시:**
```
📊 매출 분석:
- 행: 월별
- 열: 제품 카테고리
- 값: 매출 (합계)

📊 고객 분석:
- 행: 지역
- 열: 가입 연도
- 값: 고객 수 (개수)
```

**피벗 테이블 팁:**
```
✅ 값 표시 방식 변경:
- % of Grand Total (전체 대비 비율)
- Difference From (전월 대비 증감)
- Running Total (누계)

✅ 슬라이서 추가:
- 필터를 버튼 형태로
- 시각적 필터링
```

### 3-2. VLOOKUP/XLOOKUP

**데이터 찾기:**
```
시나리오:
- 주문 데이터에 고객 ID만 있음
- 고객 정보는 다른 시트에 있음
- 고객 이름을 주문 데이터에 추가하고 싶음
```

**XLOOKUP 함수:**
```
=XLOOKUP(찾을값, 찾을범위, 반환범위, [대체값])

예:
=XLOOKUP(A2, 고객정보!A:A, 고객정보!B:B, "미확인")
→ A2의 고객 ID로 고객정보 시트에서 이름 찾기
```

### 3-3. 조건부 서식

**시각적 강조:**
```
데이터: 매출 목표 달성률
- 100% 이상: 초록색
- 80-99%: 노란색
- 80% 미만: 빨간색
```

**적용 방법:**
```
홈 → 조건부 서식 → 색조합
또는 수식 기반 규칙
```

### 3-4. 데이터 시각화

**차트 만드는 법:**
```
1. 데이터 선택
2. 삽입 → 차트
3. 차트 종류 선택
4. 제목/축 레이블 추가
5. 디자인 수정
```

**차트 선택 가이드:**
```
📊 비교: 막대/세로 막대
📈 추세: 선 그래프/영역
🥧 비율: 파이/도넛
📉 분포: 히스토그램/상자
🔥 관계: 산점도/버블
```

**데이터 업로드:**
```
ChatGPT Plus/Claude Pro:

```

**프롬프트 예시:**
```

3. 카테고리별 비교
4. 이상값 확인
5. 상관관계 분석

핵심 인사이트 3가지를 도출해주세요."
```

**특징:**
```
- 데이터 분석 전용 AI
```

**사용법:**
```

```


**활용 예시:**

## 5. 실전 분석 사례

### 사례 1: 고객 이탈 분석

**상황:**
```
문제: 최근 3개월간 고객 이탈 증가
데이터: 고객 ID, 가입일, 마지막 구매일, 구매금액, 구매횟수
```

**분석 과정:**
```
Step 1: 이탈 정의
→ 6개월 이상 구매 없음 = 이탈

Step 2: 이탈률 계산
→ 월별 이탈 고객 수 / 전체 고객 수

Step 3: 특성 분석
→ 이탈 고객 vs 유지 고객 비교
→ 평균 구매액, 구매 주기 등

Step 4: 원인 분석
→ 가입 후 첫 30일 내 재구매 여부
→ VIP 고객 이탈 패턴

Step 5: 인사이트
→ "첫 구매 후 30일 내 재구매 유도가 핵심"
→ "VIP 고객 별도 관리 필요"
```

### 사례 2: 마케팅 채널 ROI 분석

**상황:**
```
문제: 마케팅 예산을 어디에 써야 할까?
데이터: 채널별 비용, 노출, 클릭, 전환, 매출
```

**분석 과정:**
```
Step 1: 기본 지표 계산
- CTR (클릭률) = 클릭 / 노출
- CVR (전환률) = 전환 / 클릭
- CPA (전환당 비용) = 비용 / 전환
- ROAS (광고대비매출) = 매출 / 비용

Step 2: 채널별 비교
- 피벗 테이블로 정리
- 히트맵으로 시각화

Step 3: 효율성 분석
- 비용 대비 전환 효율
- 매출 대비 성과

Step 4: 인사이트
→ "A 채널은 비싸지만 전환율 높음"
→ "B 채널은 저렴하지만 품질 낮음"
→ "예산을 A 60%, C 40%로 조정 필요"
```

### 사례 3: 재고 최적화

**상황:**
```
문제: 재고가 너무 많거나 부족함
데이터: 상품별 일별 판매량, 재고량, 입고일
```

**분석 과정:**
```
Step 1: 회전율 계산
- 재고회전율 = 판매량 / 평균재고

Step 2: ABC 분류
- A: 매출의 80%를 차지하는 상품 20%
- B: 중간
- C: 나머지

Step 3: 안전재고 계산
- 리드타임 × 평균일판매량

Step 4: 인사이트
→ "A 상품은 안전재고 2주분 유지"
→ "C 상품은 주문 후 생산 방식으로"
→ "계절성 고려한 예측 필요"
```

## 6. 보고서 작성하기

### 6-1. Executive Summary (요약)

**구성:**
```
1. 핵심 결론 (1문장)
   "이번 분기 매출이 15% 증가했으며, 
    주요 원인은 신규 채널 투자입니다."

2. 주요 수치 (3-4개)
   - 전체 매출: 5억원 (+15%)
   - 신규 고객: 300명 (+50%)
   - CAC: 10만원 (-20%)
   - ROAS: 300%

3. 권고사항 (1-2개)
   - 신규 채널 예산 30% 증액
   - 기존 채널은 유지
```

### 6-2. 상세 보고서 구조

```
📄 1. 개요
   - 배경 및 목적
   - 분석 기간 및 범위

📄 2. 방법론
   - 데이터 출처
   - 분석 방법
   - 한계점

📄 3. 결과
   - 주요 발견 (그래프와 함께)
   - 세부 분석

📄 4. 인사이트
   - 핵심 결론
   - 시사점

📄 5. 권고사항
   - Action Items
   - 우선순위
   - 예상 효과

📄 6. 부록
   - 상세 데이터
   - 추가 차트
```
