## 2️⃣ Coursera로 체계적으로

### Coursera란?

**Coursera**는 스탠퍼드, 예일 등 세계 명문대학의 강의를 온라인으로 제공하는 플랫폼입니다.

### 시작하기

**Step 1: 계정 생성**
```
coursera.org

□ 이메일로 가입
□ Google/Facebook 계정 연동
□ 학습 목표 설정
□ 관심 분야 선택
```

**Step 2: 묶은 강의 찾기**
```
Auditing (묶은 수강):
- Audit 클릭
- 강의 자료 접근 가능
- 과제/인증서는 유료

Financial Aid:
- 수강료 면제 신청
- 소득 증명 필요
- 15일 이내 승인
```

**Step 3: Learning Path 따라하기**
```
Google UX Design:
1. UX Design Fundamentals
2. Wireframes and Prototypes
3. UX Research
4. Portfolio Project

소요 시간: 6개월
묶음/월 $49
```

## 🎯 학습 계획 세우기

### SMART 목표 설정

```
❌ 나쁜 예:
"Python 배우기"

✅ 좋은 예:
"3개월 동안 매일 1시간 Python 공부해서
기본 문법 익히고 간단한 웹사이트 만들기"

S - Specific: Python 기본 문법
M - Measurable: 매일 1시간, 3개월
A - Achievable: 현실적인 시간
R - Relevant: 웹사이트 만들기 목표
T - Time-bound: 3개월
```

### 주간 학습 루틴

```
월요일: 새로운 개념 학습
- 강의 1개 시청
- 예제 따라하기

화-목: 실습
- 프로젝트 작업
- 문제 풀이

금요일: 복습
- 이번 주 내용 정리
- 블로그/노션에 기록

주말: 프로젝트
- 주간 내용 응용
- 나만의 프로젝트
- GitHub에 업로드
```

### 학습 진도 관리

**템플릿:**
```
📅 2024년 3월 학습 기록

🎯 이번 달 목표:
- Python 기초 완성
- 3개 미니 프로젝트

📚 진행 중인 강의:
1. 생활코딩 Python
   - 진도: 50/100강
   - 완료 예상: 3월 말

2. Udemy Web Bootcamp
   - 진도: 25/200강
   - 완료 예상: 5월 말

✅ 완료한 것:
□ Python 기초 문법
□ 변수와 자료형
□ 조건문과 반복문

📝 미니 프로젝트:
1. 계산기 (완료)
   - GitHub: 링크
   
2. 할 일 앱 (진행 중)
   - 예상 완료: 3월 20일

💭 느낀 점:
- 리스트 개념이 어려움
→ 추가 예제 풀이 필요
```
