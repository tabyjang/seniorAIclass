---

## 1. 집안일 AI

### 1.1 오늘 뭐 먹지? 해결

```
"냉장고에 OOO 있어. 30분 안에 만들 수 있는 저녁 메뉴 추천해줘"
```

**결과**: 재료 기반 레시피 + 조리 시간 + 난이도까지

### 1.2 장보기 리스트

```
"4인 가족 일주일 식단 짜고 장보기 목록 만들어줘. 예산 15만원 이내로."
```

**결과**: 요일별 식단 + 필요한 재료 + 예상 가격

### 1.3 청소 스케줄

```
"맞벌이 부부 기준으로 평일/주말 청소 스케줄 만들어줘"
```

**결과**: 요일별 10-15분 청소 루틴

---

## 2. 육아 AI

### 2.1 아이 숙제 도우미

```
"초등학교 3학년이 이해할 수 있게 '광합성'을 설명해줘"
```

**주의**: 아이 대신 숙제하는 게 아니라, **설명 방법**을 배우는 용도!

### 2.2 독후감 구조 잡기

```
"'어린왕자' 읽고 독후감 쓰려는데, 어떤 구조로 쓰면 좋을지 알려줘"
```

**결과**: 서론-본론-결론 구조 + 포함할 내용 가이드

### 2.3 주말 활동 추천

```
"6살, 8살 아이와 서울에서 할 수 있는 주말 활동 추천해줘. 교육적이면서 재미있는 거로."
```

---

## 3. 건강 AI

### 3.1 운동 루틴

```
"40대 남성, 허리 안 좋음, 집에서 할 수 있는 15분 스트레칭 루틴 만들어줘"
```

### 3.2 식단 조언

```
"당뇨 전단계인데 피해야 할 음식과 좋은 음식 알려줘"
```

> **의료 조언은 참고만! 전문의 상담 필수!**

### 3.3 증상 사전 체크

```
"머리 뒷부분이 뻐근하고 어깨가 결려. 어떤 원인이 있을 수 있어?"
```

병원 가기 전 **예비 정보** 수집 용도

---

## 4. 쇼핑 AI

### 4.1 상품 비교

```
"로봇청소기 추천해줘. 예산 30만원, 물걸레 기능 필수. 3개 비교해줘."
```

**결과**: 제품명 + 가격 + 장단점 표로 정리

### 4.2 리뷰 요약

```
(긴 상품 리뷰 복사 붙여넣기)
"이 리뷰들 요약해서 장단점 정리해줘"
```

### 4.3 협상/교섭 멘트

```
"인터넷 약정 끝났는데, 재계약 할인 요청하는 멘트 알려줘"
```

---

## 5. 여행 AI

### 5.1 여행 일정

```
"부산 2박 3일 여행 일정 짜줘. 30대 커플, 맛집 위주, 하루 예산 15만원"
```

**결과**: 시간대별 일정 + 맛집 + 예상 비용

### 5.2 현지 정보

```
"일본 오사카 여행 시 알아두면 좋은 팁 10가지"
```

### 5.3 여행 체크리스트

```
"겨울 일본 여행 짐 싸기 체크리스트 만들어줘"
```

---

## 6. 금융 AI

### 6.1 가계부 분석

```
"이번 달 지출 내역이야. 분석하고 절약할 수 있는 부분 알려줘.
식비 80만원, 교통비 15만원, 쇼핑 30만원..."
```

### 6.2 금융 상품 비교

```
"적금 가입하려는데, 최근 금리 좋은 상품 비교해줘"
```

> **투자 조언은 참고만! 최종 결정은 본인이!**

### 6.3 연말정산 준비

```
"연말정산 때 챙겨야 할 것들 체크리스트로 정리해줘"
```

## 7. 관계 AI

### 7.1 메시지 작성

```
"오랜만에 연락하는 친구한테 안부 묻는 카톡 메시지 써줘. 너무 딱딱하지 않게."
```

### 7.2 축하/위로 메시지

```
"친구가 승진했는데 진심 어린 축하 메시지 써줘"
```

### 7.3 어려운 대화 준비

```
"상사한테 연봉 협상 요청하는 메일 써줘. 공손하지만 단호하게."
```

## 8. 자기계발 AI

### 8.1 책 추천

```
"30대 직장인, 리더십 키우고 싶은데 책 3권 추천해줘"
```

### 8.2 학습 계획

```
"영어 회화 3개월 독학 계획 세워줘. 하루 30분 가능"
```

### 8.3 자격증 공부

```
"정보처리기사 실기 공부 방법 알려줘. 비전공자야."
```
