## 클라우드가 뭔가요?

"클라우드가 도대체 뭐예요?"  
"하늘에 저장하는 건가요?"  
"어렵지 않아요?"

걱정 마세요! 클라우드는 **인터넷 저장소**예요. USB나 외장하드처럼 생긴 건 아니지만, 인터넷으로 접속해서 파일을 저장하고 꺼내 쓰는 거죠.

**쉬운 비유:**
> USB는 내 서랍에 넣어두는 일기장이에요.  
> 클라우드는 어디서든 빌릴 수 있는 도서관이에요.  
> 도서관에 가면 내 책을 읽을 수 있죠? 클라우드도 똑같아요!

## 1. 클라우드 스토리지의 장점

### 1-1. 어디서든 접근 가능

**상황:**
- 회사에서 작업하다가 집에서 이어서 하고 싶어요
- 여행 중인데 중요한 문서가 필요해요
- 휴태폰 사진을 컴퓨터에서 보고 싶어요

**해결:**
```
인터넷만 연결되면!
- 회사 PC
- 집 노트북
- 스마트폰
- 태블릿
- 친구 집 컴퓨터

어디서든 내 파일에 접근 가능 ✨
```

### 1-2. 용량 걱정 없음

**내 장치:**
- 휴태폰: 128GB
- 노트북: 256GB

**클라우드:**
- 기본: 15GB ~ 1TB
- 유료: 무제한에 가까움

→ **사진첩이 꽉 차도 걱정 없어요!**

### 1-3. 공유가 쉬움

**과거:**
```
1. USB에 파일 복사
2. 친구한테 건네줌
3. 친구는 USB 꽂아서 확인
4. (USB 잃어버릴 위험 😱)
```

**클라우드:**
```
1. 링크 생성
2. 카카오톡으로 전송
3. 친구는 클릭 한 번으로 확인
4. 끝! ✨
```

### 1-4. 백업 효과

**휴태폰 분실해도:**
- 클라우드에 사진 백업됨 → 사진 안전
- 연락처 동기화 → 연락처 안전
- 문자 백업 → 대화내용 안전

**컴퓨터 고장나도:**
- 클라우드 문서는 그대로
- 새 컴퓨터에서 접속하면 OK

## 3. 네이버 MYBOX 설정하기 (한국인 추천)

### 3-1. 가입 및 설치

**1) 앱 설치:**
```
[Android]
Play Store → "MYBOX" 검색 → 설치

[iPhone]
App Store → "네이버 MYBOX" 검색 → 설치
```

**2) 로그인:**
```
- 네이버 아이디로 로그인
- 없으면 회원가입 (5분)
```

**3) 용량 확인:**
```
가입하면 기본 250GB 무료!
친구 추천하면 추가 250GB
총 500GB 무료로 사용 가능 🎉
```

### 3-2. 사진 자동 백업 설정

**Android:**
```
1. MYBOX 앱 실행
2. 하단 "사진" 탭
3. 우측 상단 설정(⚙️) 클릭
4. "사진 자동 백업" 켜기
5. "Wi-Fi에서만 업로드" 권장 (데이터 절약)
```

**iPhone:**
```
1. MYBOX 앱 실행
2. 하단 "사진" 탭
3. 상단 "백업" 클릭
4. "자동 백업" 켜기
5. "원본 백업" 선택 (화질 유지)
```

**⚙️ 중요 설정:**
- [ ] "Wi-Fi에서만 업로드" - 데이터 요금 절약
- [ ] "충전 중에만 업로드" - 배터리 소모 감소
- [ ] "원본 백업" - 사진 화질 그대로
- [ ] "동영상 포함" - 짧은 영상도 백업

### 3-3. 파일 업로드하기

**스마트폰에서:**
```
1. MYBOX 앱 실행
2. 하단 "+" 버튼 클릭
3. "업로드" 선택
4. 사진/동영상/파일 선택
5. 업로드 완료!
```

**컴퓨터에서:**
```
1. mybox.naver.com 접속
2. 로그인
3. "업로드" 버튼 클릭
4. 파일 선택 또는 드래그 앤 드롭
```

### 3-4. 파일 공유하기

**링크 생성:**
```
1. 공유할 파일 선택
2. "공유" 버튼 클릭
3. "링크 공유" 선택
4. 링크 생성!
5. 카카오톡/문자/이메일로 전송
```

**공유 옵션:**
- [ ] 다운로드 가능
- [ ] 보기만 가능
- [ ] 비밀번호 설정
- [ ] 유효기간 설정 (7일, 30일 등)

```

   - 앱 데이터
   - 통화 기록

```

```
2. Gmail 계정으로 로그인
3. 사진 백업 설정
```

## 5. iCloud 설정하기 (iPhone 사용자 필수)

### 5-1. iCloud란?

iPhone, iPad, Mac 사용자를 위한 Apple의 클라우드 서비스예요. Apple 기기끼리는 iCloud가 가장 편리해요.

### 5-2. 기본 설정

**iPhone/iPad:**
```
설정 → [내 이름] → iCloud
→ "iCloud Drive" 켜기
→ "사진" → "iCloud 사진" 켜기
→ 각 앱별 iCloud 동기화 설정
```

**백업할 항목:**
- [ ] 사진
- [ ] 연락처
- [ ] 캘린더
- [ ] 메모
- [ ] Safari 북마크
- [ ] 비밀번호 (키체인)
- [ ] 홈 화면 설정

### 5-3. iCloud 용량 관리

**무료 5GB는 부족해요:**
- 사진 몇 장이면 꽉 참
- 유료 업그레이드 필요

**용량 정리:**
```
설정 → [내 이름] → iCloud → 저장 공간 관리
→ 백업 확인
→ 큰 파일 삭제
→ 불필요한 앱 데이터 삭제
```

## 6. 클라우드 활용 팁

### 6-1. 스마트한 정리법

**폴더 구조 예시:**
```
📁 MYBOX (또는 Google Drive)
├── 📁 01_사진
│   ├── 📁 2024년
│   │   ├── 📁 01월
│   │   ├── 📁 02월
│   │   └── 📁 여행_제주도
│   └── 📁 2023년
├── 📁 02_문서
│   ├── 📁 회사
│   ├── 📁 개인_서류
│   └── 📁 영수증
├── 📁 03_받은_파일
│   └── (임시 저장 후 정리)
└── 📁 04_공유
    └── (다른 사람과 공유 중인 파일)
```

**파일명 규칙:**
```
✅ 좋은 예:
2024-02-05_프로젝트_기획안_최종.pdf
2024-01-여행-부산-사진.jpg
영수증_2024년1월.zip

❌ 나쁜 예:
최종.pdf (어떤 최종?)
사진.jpg (몇 번째 사진?)
Untitled 1.pdf (무슨 파일?)
```

### 6-2. 용량 절약법

**사진 정리:**
- 흔들린 사진 삭제
- 스크린샷 정리 (필요 없는 것 삭제)
- 중복 사진 제거 (Remo Duplicate 등 앱 사용)
- WhatsApp/카카오톡 자동 다운로드 끄기

**동영상 관리:**
- 긴 영상은 유튜브 비공개로 업로드
- 클라우드에는 중요한 것만
- 해상도 조정 (4K → 1080p)

**문서 압축:**
- 큰 PDF는 작게 압축 (ilovepdf.com)
- 오래된 파일은 "아카이브" 폴더로

### 6-3. 보안 설정

**2단계 인증:**
```
모든 클라우드 계정에 2FA 설정!
→ 이전 포스트 참고
```

**공유 주의:**
- 민감한 문서는 비밀번호 설정
- 공유 링크는 필요한 사람에게만
- 완료 후 공유 중지

**정기 점검:**
- [ ] 로그인 기록 확인
- [ ] 연결된 기기 관리
- [ ] 공유 중인 파일 점검
- [ ] 휴지통 비우기

## 8. 문제 해결

### 8-1. 동기화가 안 돼요

**확인 사항:**
```
1. 인터넷 연결 확인
2. 클라우드 앱이 최신 버전인지
3. 저장 공간이 부족하지 않은지
4. "Wi-Fi에서만" 설정이 켜져있는지
5. 로그인이 풀리지 않았는지
```

**해결법:**
```
- 앱 강제 종료 후 재실행
- 로그아웃 후 다시 로그인
- 동기화 일시정지 후 재개
- Wi-Fi/데이터 전환
```

### 8-2. 파일이 사라졌어요

**확인할 곳:**
```
1. 휴지통/삭제함 확인
2. 최근 삭제된 파일 복원
3. 다른 폴더에 있는지 검색
4. 웹사이트에서 로그인 후 확인
```

**복원 방법:**
```
네이버 MYBOX:
- 삭제함 → 파일 선택 → 복원
(30일간 보관)

Google Drive:
- 휴지통 → 복원
(30일간 보관)

iCloud:
- iCloud.com → 설정 → 데이터 복원
(30일간 보관)
```

### 8-3. 용량이 부족해요

**정리 방법:**
```
1. 큰 파일 찾기
   - 용량순 정렬
   - 필요 없는 영상 삭제

2. 중복 파일 제거
   - 중복 사진 정리 앱 사용
   - Gemini Photos (iPhone)
   - Files by Google (Android)

3. 오래된 파일 정리
   - 1년 이상 안 본 파일
   - "아카이브" 폴더로 이동

4. 유료 업그레이드
   - 월 2,400원이면 100GB
   - 사진 많은 분은 필수
```
