## 2. ElevenLabs 마스터하기

### 2-1. ElevenLabs란?

**특징:**
```
✅ 가장 자연스러운 AI 음성
✅ 목소리 클론 (Voice Cloning)
✅ 29개 언어 지원
✅ 감정/억양 조절
✅ 실시간 생성
```

**가격:**
```
🆓 Starter: 10K 문자/월 (묶)
💼 Creator: $5/월 (30K 문자)
🎙️ Independent: $22/월 (100K 문자)
```

### 2-2. 기본 사용법

**음성 생성:**
```
1. ElevenLabs.io 가입
2. "Speech Synthesis" 선택
3. 목소리 선택 (한국어: Adam, Bella 등)
4. 텍스트 입력
5. Settings 조절:
   - Stability: 0.5 (안정성)
   - Clarity + Similarity: 0.75 (선명도)
6. Generate 클릭
7. 다운로드
```

**프롬프트 팁:**
```
❌ 그냥 텍스트:
"안녕하세요 오늘은 팟캐스트 시작합니다"

✅ SSML 태그 활용:
"<break time=\"0.5s\"/>안녕하세요.
<break time=\"0.3s\"/>오늘은 팟캐스트 시작합니다!"
→ 자연스러운 호흡
```

### 2-3. Voice Cloning (목소리 복제)

**내 목소리 복제:**
```
1. "Voices" → "Add Generative or Cloned Voice"
2. "Instant Voice Cloning" 선택
3. 녹음 파일 업로드
   - 1-5분 분량
   - 조용한 환경
   - 마이크 가까이
4. 이름 설정 (예: "MyVoice")
5. Generate
6. 완료! 내 목소리로 TTS 가능
```

**주의사항:**
```
⚠️ 타인 목소리 복제 금지
⚠️ 동의 없는 공인 목소리 사용 금지
⚠️ 사기/악용 목적 금지
```

### 2-4. Projects (장문 음성)

**팟캐스트/오디오북 제작:**
```
1. "Projects" 생성
2. 스크립트 붙여넣기
3. 목소리 선택
4. 문단별로 설정 조절:
   - Pause 추가
   - Speed 조절
   - Pitch 조절
5. Generate
6. 한 번에 긴 음성 생성
```

## 3. Murf.ai로 비즈니스 음성


## 5. 팟캐스트 제작 워크플로우

### 5-1. AI로 팟캐스트 만들기

**Step 1: 기획 (AI 활용)**
```
ChatGPT에게:
"[주제] 팟캐스트 10회 분 기획해줘

회차별:
- 제목
- 주요 내용
- 키워드 5개
- 예상 분량 (20분)"
```

**Step 2: 스크립트 작성**
```
ChatGPT:
"[주제]에 대한 팟캐스트 스크립트 작성

형식:
- 인트로 (30초)
- 본론 3부분
- 아웃트로 (30초)
- 톤: 친근하고 전문적
- 분량: 3000단어 (20분)"
```

**Step 3: AI 음성 생성**
```
ElevenLabs:
1. 스크립트 입력
2. 목소리: Adam (뉴스형)
3. Settings: Stability 0.35
4. Generate
5. 다운로드
```

**Step 4: 편집 (Descript)**
```
1. 음성 파일 업로드
2. filler word 제거 (음, 그)
3. 공백 자동 제거
4. BGM 추가 (Suno AI)
5. Export
```

**Step 5: 배포**
```
- Spotify for Podcasters
- Apple Podcasts
- Google Podcasts
- 네이버 오디오클립
```

### 5-2. AI DJ 만들기

**Stationhead 활용:**
```
1. Stationhead 앱 설치
2. 방송국 생성
3. AI 음성으로 진행
4. Spotify/Apple Music 연동
5. 라이브 방송
```

## 6. 실전 활용 사례

### 사례 1: 유튜브 낭독 채널

**시나리오:**
```
- 블로그 글 → 음성 변환
- 매일 1편 업로드
- 구독자 10만 목표
```

**워크플로우:**
```
1. 블로그 글 선정
2. 스크립트 편집 (구어체로)
3. ElevenLabs로 음성 생성
4. 썸네일 (Canva AI)
5. 유튜브 업로드
6. 자막 생성 (YouTube AI)

시간: 1시간/편
```

### 사례 2: 오디오북 제작

**시나리오:**
```
- 자기계발서 오디오북
- 10챕터, 총 5시간
```

**워크플로우:**
```
1. 원고 정리
2. 챕터별 스크립트
3. 목소리 클론 (작가 목소리)
4. ElevenLabs Projects 사용
5. 감정 표현 추가:
   - 중요한 부분: 강조
   - 스토리: 생동감
6. 편집 및 마스터링
7. Audible/교보문고 업로드
```

### 사례 3: 다국어 콘텐츠

**시나리오:**
```
- 한국어 콘텐츠 → 영어/중국어/일본어 변환
```

**워크플로우:**
```
1. 원본 스크립트
2. ChatGPT로 번역
3. ElevenLabs에서 해당 언어 목소리 선택
4. 각 언어별 음성 생성
5. 영상과 합성
6. 각 국가 채널 업로드
```
