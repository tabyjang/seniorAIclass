"AI로 뭘 할 수 있어요?"라는 질문에 대한 가장 좋은 답은 "뭐든지요"입니다. 하지만 그게 너무 막연하죠. 그래서 오늘은 **당장 오늘부터 써먹을 수 있는** 실생활 AI 활용법 30가지를 준비했습니다. 이론 없이, 바로 복사해서 쓸 수 있는 꿀팁만 모았습니다.

## 1. 요리/식단 (5가지)

### 1-1. 냉장고 파먹기 레시피

```
[복사해서 사용하세요]

냉장고에 있는 재료로 저녁 만들어줘.
재료: 두부, 계란 3개, 파, 마늘, 간장, 참기름
조건: 30분 이내, 어른 2명 + 아이 1명분
```

### 1-2. 맞춤형 식단 짜기

```
[복사해서 사용하세요]

일주일 식단 짜줘.
조건:
- 아침은 간단하게 (10분 이내)
- 점심은 도시락 쌀 수 있는 것
- 저녁은 가족이 같이 먹는 한식 위주
- 예산: 1인당 하루 1만원
```

### 1-3. 레시피 간소화

```
[복사해서 사용하세요]

이 레시피 더 쉽게 바꿔줘. 초보자도 따라할 수 있게.
[기존 복잡한 레시피 붙여넣기]
```

### 1-4. 칼로리 계산

```
오늘 먹은 거 칼로리 계산해줘:
아침: 토스트 2장, 계란후라이 1개, 우유 1잔
점심: 김치찌개, 밥 1공기, 김치
간식: 아메리카노, 마카롱 1개
저녁: 치킨 5조각, 맥주 1캔
```

### 1-5. 재료 대체품 찾기

```
버터 없이 쿠키 만들 수 있어?
버터 대신 넣을 수 있는 재료 알려줘.
```

## 2. 건강/운동 (5가지)

### 2-1. 맞춤형 운동 루틴

```
[복사해서 사용하세요]

홈트레이닝 루틴 짜줘.
조건:
- 운동 초보
- 하루 20분
- 기구 없음
- 목표: 뱃살 빼기
- 주 4회
```

### 2-2. 스트레칭 가이드

```
하루 종일 앉아서 일하는 사람을 위한 스트레칭 알려줘.
사무실에서 할 수 있는 것으로, 5분짜리로.
```

### 2-3. 증상별 대처법

```
[주의: 심각한 증상은 반드시 병원 가세요]

두통이 있을 때 집에서 할 수 있는 방법 알려줘.
약 먹기 전에 시도해볼 수 있는 것들로.
```

### 2-4. 영양제 정보

```
40대 여성에게 추천하는 영양제 알려줘.
각 영양제가 왜 필요한지 설명해줘.
```

### 2-5. 수면 개선

```
잠이 잘 안 와. 수면의 질 높이는 방법 알려줘.
잠들기 전 루틴도 추천해줘.
```

## 3. 육아/교육 (5가지)

### 3-1. 아이 숙제 도우미

```
[복사해서 사용하세요]

초등학교 3학년 수준으로 광합성 설명해줘.
아이가 이해하기 쉽게, 예시 들어서.
```

### 3-2. 놀이 아이디어

```
비 오는 날 집에서 7살 아이랑 할 수 있는 놀이 10가지 알려줘.
준비물 없거나 집에 있는 것으로만.
```

### 3-3. 아이 식단

```
편식하는 5살 아이가 먹을 만한 야채 요리 알려줘.
야채가 안 보이게 숨겨서 만드는 레시피로.
```

### 3-4. 독서 추천

```
책 읽기 싫어하는 초등학생이 재밌게 읽을 수 있는 책 추천해줘.
학년: 4학년
관심사: 축구, 게임
```

### 3-5. 공부법 코칭

```
중학생 아이가 영어 단어 외우기 힘들어해.
효과적인 영단어 암기 방법 알려줘.
재미있게 할 수 있는 것으로.
```

## 4. 여행/외출 (5가지)

### 4-1. 여행 일정 짜기

```
[복사해서 사용하세요]

제주도 2박 3일 여행 일정 짜줘.
- 구성: 부모님(60대) + 나
- 이동: 렌트카
- 숙소: 서귀포 근처
- 원하는 것: 맛집, 경치 좋은 곳, 많이 안 걸어도 되는 곳
```

### 4-2. 여행 짐 체크리스트

```
태국 4박 5일 여행 짐 싸야 해.
11월에 가는데 체크리스트 만들어줘.
빠뜨리기 쉬운 것 위주로.
```

### 4-3. 현지 회화

```
일본 여행 갈 때 필요한 기본 일본어 알려줘.
식당, 쇼핑, 교통에서 쓸 수 있는 것.
발음도 한글로 적어줘.
```

### 4-4. 주변 데이트 코스

```
강남역 근처에서 저녁 데이트 코스 짜줘.
예산: 10만원
시간: 저녁 6시부터 4시간
포함: 저녁 식사 + 카페 또는 활동
```

### 4-5. 맛집 분석

```
이 맛집 리뷰들 요약해줘. 진짜 가볼 만한지 판단해줘.
[네이버/카카오 리뷰 복사해서 붙여넣기]
```

## 5. 돈/재테크 (5가지)

### 5-1. 가계부 분석

```
이번 달 지출 분석해줘.
[카드 사용 내역 복사해서 붙여넣기]
줄일 수 있는 부분 알려줘.
```

### 5-2. 저축 계획

```
월급 300만원 받는 1인 가구야.
월 100만원 저축하려면 어떻게 해야 해?
현실적인 예산 배분 알려줘.
```

### 5-3. 보험 이해하기

```
이 보험 약관 쉽게 설명해줘.
[보험 약관 일부 붙여넣기]
가입해도 괜찮은 건지 의견도 말해줘.
```

### 5-4. 세금 계산

```
연봉 5천만원이면 실수령액 얼마야?
4대보험, 소득세 다 계산해서 알려줘.
```

### 5-5. 투자 용어 설명

```
주식 초보야.
PER, PBR, ROE가 뭔지 쉽게 설명해줘.
실제 예시 들어서.
```

## 6. 업무/자기계발 (5가지)

### 6-1. 이메일 작성

```
[복사해서 사용하세요]

거래처에 미팅 요청 이메일 써줘.
- 상대: ABC 회사 김부장님
- 목적: 신규 프로젝트 제안
- 희망 일시: 다음 주 화-목 중
- 톤: 정중하지만 딱딱하지 않게
```

### 6-2. 보고서 요약

```
이 보고서 핵심만 3줄로 요약해줘.
[긴 보고서 붙여넣기]
```

### 6-3. 발표 준비

```
신입사원 교육 발표 자료 개요 만들어줘.
주제: 효과적인 시간 관리
시간: 30분 발표
슬라이드: 10장 내외
```

### 6-4. 자기소개서

```
이직 준비 중이야. 자기소개서 첨삭해줘.
[자기소개서 붙여넣기]
더 인상적으로 바꿔줘.
```

### 6-5. 외국어 번역/교정

```
이 영어 이메일 자연스럽게 고쳐줘.
[영어 이메일 붙여넣기]
비즈니스 상황에 맞게.
```

## 7. 쇼핑/생활 (5가지)

### 7-1. 제품 비교

```
에어프라이어 사려고 해.
삼성 vs LG vs 필립스 비교해줘.
가격대: 20만원 내외
중요한 것: 용량, 세척 편의성
```

### 7-2. 청소 꿀팁

```
욕실 곰팡이 제거하는 방법 알려줘.
천연 재료로 할 수 있는 것으로.
```

### 7-3. 옷 코디

```
면접 가는데 뭐 입어야 해?
IT 스타트업, 30대 남자, 비즈니스 캐주얼
갖고 있는 옷: 남색 재킷, 흰 셔츠, 검정 바지, 청바지
```

### 7-4. 선물 추천

```
60대 아버지 생신 선물 추천해줘.
예산: 10-20만원
아버지 관심사: 등산, 뉴스 보기
피해야 할 것: 건강식품 (이미 많음)
```

### 7-5. 정리정돈

```
옷장 정리하는 방법 알려줘.
옷이 너무 많아서 뭐가 있는지도 모르겠어.
버리는 기준도 알려줘.
```

## 사용 팁

### 더 좋은 답변 받는 법

1. **구체적으로 말하기**
   - ❌ "여행지 추천해줘"
   - ✅ "7월에 4살 아이랑 갈 만한 국내 여행지 추천해줘"

2. **조건 붙이기**
   - ❌ "저녁 메뉴"
   - ✅ "30분 안에, 재료 5개 이하로 만들 수 있는 저녁 메뉴"

3. **역할 부여하기**
   - ❌ "운동 알려줘"
   - ✅ "넌 퍼스널 트레이너야. 초보자인 나한테 운동 알려줘"

4. **마음에 안 들면 다시 요청**
   - "다른 거 추천해줘"
   - "좀 더 쉬운 걸로"
   - "더 자세하게 설명해줘"
