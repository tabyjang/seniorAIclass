## 1️⃣ 디지털 인맥 관리

### 인맥 정리하기

**Step 1: 현재 인맥 분석**
```
친구/팔로워 분류:

🟢 핵심 인맥 (50-100명)
- 가족/친한 친구
- 정기적으로 소통
- 중요한 소식 공유

🟡 지인 (200-500명)
- 아는 사람들
- 필요시 연락
- 생일/명절 인사

🔴 정리 대상
- 기억 안 나는 사람
- 소통 없는 사람
- 부정적인 사람
```

**Step 2: 정리 실행**
```
Facebook:
□ 친구 목록 확인
□ 친구 숨김/차단 설정
□ 그룹 정리
□ 페이지 좋아요 취소

Instagram:
□ 팔로우 목록 정리
□ 팔로잉 취소 (관심 없는 계정)
□ 저장된 콘텐츠 정리

LinkedIn:
□ 커넥션 정리
□ 의미 있는 인맥에 집중
□ 프로필 최신화
```

### 의미있는 연결 만들기

**네트워킹 전략:**
```
주 1회 의식적인 연결:

월요일: LinkedIn
→ 3명에게 인사 메시지
→ 1개 포스트에 의견 남기기

수요일: Instagram
→ 5명의 스토리에 반응
→ 1개 게시물에 진심 댓글

금요일: Facebook
→ 가족/친구 소식 확인
→ 생일인 친구 축하
```

## 2️⃣ 콘텐츠 관리법

### 내 콘텐츠 전략

**개인 브랜딩:**
```
포스팅 주제 정하기:

전문 분야:
- 업계 뉴스 공유
- 전문 지식 공유
- 프로젝트 소개

관심사:
- 취미 활동
- 독서 리뷰
- 여행 기록

일상:
- 적절한 수준의 공유
- 긍정적인 내용 중심
- 프라이버시 존중
```

**포스팅 템플릿:**
```
좋은 게시물 구성:

1. Hook (관심 끌기)
   "3년 만에 드디어..."
   "놀라운 발견을 했어요"

2. 내용 (가치 제공)
   - 구체적인 정보
   - 개인적인 경험
   - 유용한 팁

3. Engagement (참여 유도)
   "여러분은 어떻게 생각하세요?"
   "댓글로 공유해주세요"
   "저장하고 나중에 보세요"

4. 해시태그 (노출 확대)
   #관련키워드 #업계명 #취미
   (5-10개 적절히)
```

### 팔로우 관리

**정보 다이어트:**
```
팔로우 정리 기준:

✅ 유지할 계정:
- 업계 전문가
- 긍정적인 영향
- 실용적인 정보
- 친구/가족

❌ 정리할 계정:
- 부정적인 내용만
- 광고 과다
- 관심 없는 분야
- 비교만 유발

정리 방법:
□ 팔로우 취소
□ 뮤트 (관계 유지하되 피드에서 숨김)
□ 차단 (극단적 경우)
```

## 3️⃣ 시간 관리와 디지털 디톡스

### SNS 사용 시간 줄이기

**Step 1: 현황 파악**
```
스마트폰 설정:

iPhone: 설정 → 화면 사용 시간
Android: 설정 → 디지털 웰빙

확인 사항:
□ 하루 평균 사용 시간
□ 가장 많이 쓰는 앱
□ 픽업 횟수
□ 알림 횟수
```

**Step 2: 제한 설정**
```
앱 타이머 설정:

Instagram: 30분/일
Facebook: 20분/일
Twitter: 15분/일
TikTok: 20분/일

총 SNS: 1시간 25분/일

타이머 효과:
- 시간 초과 시 알림
- 초과 후 앱 사용 제한
- 의식적인 사용
```

### 디지털 디톡스 루틴

**일상 디톡스:**
```
아침 루틴 (30분):
□ SNS 금지
□ 명상/욕조
□ 오늘의 목표 확인
□ 아침 식사

업무 시간:
□ SNS 알림 끄기
□ 특정 시간만 확인
  (점심시간, 퇴근 후)

저녁 루틴 (2시간):
□ 21:00 이후 SNS 금지
□ 독서/욕조
□ 내일 준비
□ 23:00 취침
```

**주간 디톡스:**
```
주말 무SNS:

토요일 오전:
□ SNS 사용 금지
□ 야외 활동
□ 사람들과 직접 대화

일요일:
□ SNS 30분만 사용
□ 일주일 정리
□ 다음 주 계획
```

## 4️⃣ 개인정보 보호

### 프라이버시 설정

**Facebook:**
```
설정 → 개인정보:

□ 게시물 공개 범위
  - 친구만
  - 특정 친구
  - 비공개

□ 프로필 정보
  - 생년월일 숨김
  - 연락처 비공개
  - 직장 정볼만 표시

□ 타임라인 태그
  - 태그 승인 후 게시
  - 타인의 게시물 내 타임라인 차단
```

**Instagram:**
```
설정 → 개인정보:

□ 비공개 계정 설정
  - 승인된 팔로워만 게시물 확인

□ 스토리 공유 설정
  - 친한 친구만
  - 특정인 숨김

□ 활동 상태
  - 온라인 상태 숨김

□ DM 설정
  - 모르는 사람 메시지 차단
```

**LinkedIn:**
```
설정 → 개인정보:

□ 프로필 열람 설정
  - 익명으로 열람
  - 일부 정보만 공개

□ 공개 프로필
  - 구글 검색 결과 제외
  - 기본 정보만 공개
```

### 정보 유출 방지

**주의사항:**
```
공유하지 말 것:
❌ 집 주소/거주 지역
❌ 여행 중인 날짜
❌ 신분증 번호
❌ 금융 정보
❌ 비밀번호 힌트

신중하게 공유:
⚠️ 출퇴근 시간/경로
⚠️ 회사 내부 사진
⚠️ 자녀 사진/정보
⚠️ 정치적/종교적 의견
```

## 5️⃣ SNS를 성장 도구로 활용

### 학습에 활용

**정보 수집:**
```
팔로우 추천:

업계 전문가:
- 관련 업계 리더
- 전문가 계정
- 협회/단체 계정

교육 채널:
- 유튜브 채널
- 인스타그램 에듀계정
- 트위터 교육자

뉴스/정보:
- 뉴스 매체
- 전문 미디어
- 팟캐스트
```

**저장 활용:**
```
Instagram 저장:
- 배운 내용 정리
- 나중에 볼 콘텐츠
- 주제별 폴더 정리

Facebook 저장:
- 아티클 저장
- 동영상 저장
- 나중에 읽기

Twitter 북마크:
- 중요한 트윗
- 스레드/정리글
- 유용한 링크
```

### 커리어에 활용

**LinkedIn 활용:**
```
프로필 최적화:

□ 헤드라인
  - 직책 + 전문분야
  - 예: "디지털 마케팅 전문가 | 
         데이터 기반 성과 창출"

□ About
  - 3-5문장 자기소개
  - 핵심 성과 2-3가지
  - 연락처

□ Experience
  - 성과 중심 작성
  - 수치화된 결과

□ Engagement
  - 주 2회 포스트
  - 업계 뉴스 공유
  - 의견 제시
```
