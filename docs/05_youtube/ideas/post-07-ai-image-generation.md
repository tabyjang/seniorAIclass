## 1. AI 이미지 생성의 기본 개념

### 1.1 AI가 이미지를 만드는 원리

**텍스트를 이미지로 (Text-to-Image):**
- 사용자가 자연어로 설명을 입력
- AI가 수십억 개의 이미지 데이터를 학습해 새로운 이미지 생성
- 10-60초 내에 결과물 출력

**주요 기술:**
- **Diffusion Model**: 점진적으로 노이즈를 제거하며 이미지 생성
- **GAN (Generative Adversarial Network)**: 생성자와 판별자가 경쟁하며 품질 향상
- **Transformer**: 자연어 이해와 이미지 생성의 연결

### 1.2 대표적인 AI 이미지 생성 도구

| 도구 | 특징 | 가격 | 난이도 |
|------|------|------|--------|
| **Midjourney** | 예술적 품질 최고 | $10/월 | 중급 |
| **DALL-E 3** | ChatGPT 연동, 사용 편리 | API 과금 | 초급 |
| **Stable Diffusion** | 오픈소스, 커스터마이징 가능 | 묶은 (셀프호스팅) | 고급 |
| **Canva AI** | 디자인 통합, 간편함 | 묶은/Pro | 초급 |
| **Adobe Firefly** | 상업적 안전성, Photoshop 연동 | Creative Cloud | 초급 |

## 2. 초보자를 위한 도구 선택 가이드

### 2.1 첫 도구로 추천: Canva AI

**선택 이유:**
- 한국어 지원 완벽
- 별도 설치 불필요 (웹 브라우저)
- 생성된 이미지를 바로 디자인에 활용 가능
- 묶은 버전으로도 충분한 기능 제공

**시작하기:**
1. canva.com 접속 → 회원가입
2. "Magic Media" 또는 "텍스트에서 이미지로" 선택
3. 한국어로 원하는 이미지 설명 입력
4. 스타일, 화멸 비율 선택 후 생성

### 2.2 업그레이드: DALL-E 3 (ChatGPT Plus)

**ChatGPT Plus 구독 시 ($20/월):**
- ChatGPT 대화창에서 바로 이미지 생성
- 자연스러운 대화로 이미지 수정 가능
- "이미지를 더 밝게 해줘", "배경을 바꿔줘" 등 지시 가능

**사용 예시:**
```
"SNS용 명언 이미지를 만들어줘. 
배경은 따뜻한 색상의 그라데이션이고, 
텍스트는 '오늘도 화이팅'이라고 써줘. 
미니멀한 스타일로 부탁해."
```

### 2.3 고급 사용자: Midjourney

**특징:**
- 디스코드 기반 커뮤니티
- 가장 예술적이고 고퀄리티한 결과물
- 세밀한 파라미터 조정 가능

**기본 프롬프트 구조:**
```
[주제] + [세부 묘사] + [스타일/분위기] + [기술적 파라미터]

예시:
"A cozy coffee shop interior, warm lighting, 
vintage furniture, plants by the window, 
photorealistic style, 8k resolution, 
--ar 16:9 --v 6"
```

## 3. 효과적인 프롬프트 작성법

### 3.1 프롬프트의 5요소

**1) 주제 (Subject)**
- 무엇을 그릴 것인가?
- 예: "고양이", "도시 풍경", "사과"

**2) 세부 묘사 (Details)**
- 색상, 크기, 재질, 위치 관계
- 예: "주황색 털의 작은 고양이가 창가에 앉아 있다"

**3) 스타일/분위기 (Style/Mood)**
- 예술적 스타일, 조명, 분위기
- 예: "수채화 스타일, 따뜻한 오후의 햇살, 평화로운 분위기"

**4) 구도/구성 (Composition)**
- 칩라 앵글, 프레이밍
- 예: "클로즈업 샷, 배경 흐림, 룰오브써스 구도"

**5) 기술적 사양 (Technical)**
- 해상도, 화멸 비율, 렌더링 품질
- 예: "4K, 16:9 비율, 사실적 렌더링"

### 3.2 프롬프트 템플릿

**템플릿 1: 인물/캐릭터**
```
[인물 묘사] + [옷차림/스타일] + [행동/포즈] + 
[배경] + [조명/분위기] + [칩라 정보]

예시:
"20대 아시아 여성, 캐주얼한 청바지와 흰 셔츠, 
노트북으로 일하는 모습, 현대적인 카페 배경, 
자연광, 사이드 앵글, 4K 사실적 스타일"
```

**템플릿 2: 제품/오브젝트**
```
[제품명] + [특징/디테일] + [배경/세팅] + 
[조명] + [스타일]

예시:
"무선 이어폰, 화이트 컬러, 미니멀 디자인, 
나무 탁자 위에 놓임, 부드러운 자연광, 
제품 사진 스타일, 깨끗한 배경"
```

**템플릿 3: 배경/풍경**
```
[장소] + [시간/날씨] + [분위기] + 
[스타일] + [화멸 비율]

예시:
"봄의 벚꽃길, 맑은 날씨, 평화로운 분위기, 
애니메이션 스타일, 와이드 앵글, 16:9"
```

### 3.3 효과를 높이는 키워드 모음

**품질 향상 키워드:**
```
고퀄리티: "high quality", "highly detailed", "8k", "photorealistic"
전문적: "professional", "masterpiece", "award-winning"
세밀함: "intricate details", "sharp focus", "ultra detailed"
```

**스타일 키워드:**
```
예술: "oil painting", "watercolor", "pencil sketch", "digital art"
사진: "photography", "DSLR", "35mm", "cinematic"
일러스트: "anime style", "cartoon", "3D render", "vector art"
```

**조명/분위기:**
```
조명: "natural lighting", "studio lighting", "golden hour", "soft light"
분위기: "cozy", "mysterious", "romantic", "futuristic", "vintage"
```

## 4. 실전 활용 시나리오

### 4.1 블로그/콘텐츠용 이미지

**시나리오: 블로그 썸네일 만들기**

**단계 1: 주제 정하기**
- 블로그 주제: "재테크 초보자를 위한 5가지 팁"

**단계 2: 프롬프트 작성**
```
"재테크 블로그 썸네일 이미지, 
동전과 성장하는 식물, 깨끗한 흰색 배경, 
미니멀하고 현대적인 스타일, 밝은 색상, 
16:9 비율, 전문적인 일러스트"
```

**단계 3: 생성 및 수정**
- Canva AI로 생성
- 텍스트 추가: "재테크 초보자 가이드"
- 색상 조정 및 필터 적용

### 4.2 SNS 인스타그램 포스트

**시나리오: 명언 카드 만들기**

**단계 1: 배경 이미지 생성**
```
"인스타그램 명언 카드 배경, 
파스텔 톤의 그라데이션, 
부드러운 구름 패턴, 
1:1 비율, 감성적인 분위기"
```

**단계 2: Canva에서 텍스트 추가**
- 생성된 배경 이미지 업로드
- "오늘도 최선을 다하는 당신이 멋집니다" 텍스트 추가
- 폰트: 스포카한산스 또는 Noto Sans

### 4.3 프레젠테이션 자료

**시나리오: 보고서용 차트 이미지**

**AI 활용법:**
- AI로 전문적인 비즈니스 이미지 생성
- "비즈니스 미팅, 현대적인 오피스, 
  프로페셔널한 분위기, 16:9"
- 생성된 이미지를 슬라이드 배경으로 활용

## 5. 각 도구별 실전 팁

### 5.1 Canva AI 팁

**텍스트 포함 이미지 생성:**
- Canva AI는 한글 텍스트 생성이 불안정
- 텍스트 없는 배경 이미지만 생성 권장
- 텍스트는 Canva 편집 기능으로 추가

**Magic Edit 기능:**
1. 기존 이미지 업로드
2. 수정하고 싶은 영역 브러시로 선택
3. "이 부분을 ~로 바꿔줘" 입력
4. AI가 자연스럽게 수정

### 5.2 DALL-E 3 팁

**대화형 수정:**
```
첫 생성: "고양이 이미지"
↓
수정1: "더 귀여운 표정으로 바꿔줘"
↓
수정2: "배경을 벚꽃이 있는 공원으로 바꿔줘"
↓
수정3: "애니메이션 스타일로 변환해줘"
```

**스타일 일관성 유지:**
- "위 이미지와 같은 스타일로 ~도 만들어줘"
- 시리즈 콘텐츠 제작 시 유용

### 5.3 Midjourney 팁

**파라미터 활용:**
```
--ar 16:9     # 화멸 비율 (16:9)
--ar 1:1      # 정사각형
--ar 9:16     # 세로형 (인스타그램 스토리)
--v 6         # 버전 6 사용
--stylize 250 # 스타일화 정도 (0-1000)
--chaos 50    # 다양성 정도 (0-100)
```

**이미지 참조 (Image Prompt):**
1. 참조할 이미지 URL 복사
2. 프롬프트 앞에 URL 입력
3. "이 이미지와 비슷한 스타일로 ~ 만들어줘"

## 6. 저작권과 윤리적 사용

### 6.1 상업적 사용 가능 여부

| 도구 | 상업적 사용 | 주의사항 |
|------|-------------|----------|
| **Canva Pro** | 가능 | Pro 요금제 가입 시 |
| **DALL-E 3** | 가능 | OpenAI 정책 준수 |
| **Midjourney** | 가능 | 유료 플랜 가입 시 |
| **Stable Diffusion** | 가능 | 오픈소스 라이선스 |

### 6.2 주의사항

**⚠️ 피해야 할 것:**
- 실존 인물의 얼굴 생성 (명예훼손 위험)
- 저작권 있는 캐릭터/브랜드 모방
- 민감한 콘텐츠 생성
- AI 생성 이미지를 "직접 그림"이라고 주장

**✅ 올바른 사용:**
- "AI로 생성한 이미지" 명시 (선택사항이나 권장)
- 원본 이미지에 충분히 변형 가하기
- 공개 도메인 스타일 참고
