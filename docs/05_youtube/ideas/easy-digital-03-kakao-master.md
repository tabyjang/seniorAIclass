# 카카오톡 숨은 기능 10가지: 이것만 알면 카톡 고수

**발행일**: 2024-03-10
**카테고리**: 쉬운 디지털 생활
**난이도**: ⭐ (입문)
**대상**: 카카오톡을 더 편리하게 사용하고 싶은 분
**읽는 시간**: 15분

## 여는 말: 카카오톡, 채팅만 하고 계신가요?

카카오톡은 우리나라 국민 메신저예요. 하루에도 수십 번 카톡을 확인하지만, 대부분의 분들은 **메시지 보내기**와 **사진 공유** 정도만 사용하고 계실 거예요.

그런데 사실 카카오톡에는 놀라울 정도로 유용한 숨은 기능들이 정말 많아요! 오늘 알려드리는 10가지 기능만 알면, 카카오톡을 200% 활용할 수 있을 거예요. 하나씩 따라해 보면서 익혀볼까요?

---

## 기능 1: 예약 메시지 - 원하는 시간에 자동으로 카톡 보내기

생일 축하 메시지를 자정에 보내고 싶은데 잠이 올 때, 업무 연락을 아침에 보내고 싶을 때! 예약 메시지 기능을 사용하면 돼요.

### 따라하기

1. 메시지를 보낼 **채팅방**에 들어가요
2. 메시지 입력창에 보내고 싶은 **메시지를 입력**해요
3. 전송 버튼을 **길게 눌러요** (짧게 누르면 바로 전송되니 주의!)
4. **"예약 메시지"** 옵션이 나타나요
5. 원하는 **날짜와 시간**을 설정해요
6. **"예약"** 버튼을 누르면 설정한 시간에 자동으로 전송돼요!

### 활용 팁

- 생일 축하 메시지를 자정에 예약해두면 가장 먼저 축하할 수 있어요
- 업무 관련 메시지를 근무 시간에 맞춰 예약할 수 있어요
- 기념일 메시지를 미리 준비해둘 수 있어요

**참고**: 예약 메시지는 설정한 시간에 인터넷이 연결되어 있어야 전송돼요. 비행기 모드에서는 전송되지 않으니 주의해주세요.

## 기능 2: 카카오페이 송금 - 친구에게 간편하게 돈 보내기

밥값 나눠 내기, 회비 모으기, 용돈 보내기... 카카오톡 안에서 바로 송금할 수 있어요.

### 따라하기

1. 송금할 친구의 **채팅방**에 들어가요
2. 메시지 입력창 왼쪽의 **+ 버튼**을 눌러요
3. **"송금"**을 선택해요
4. **금액**을 입력해요
5. **비밀번호**를 입력하면 즉시 송금 완료!

### 여러 명에게 동시에 보내기

1. **그룹 채팅방**에서 + 버튼 > 송금을 눌러요
2. 금액을 입력한 후 **"N분의 1 송금"**을 선택할 수도 있어요
3. 받을 사람을 선택하면 각자에게 보내져요

### 돈 받기 (정산하기)

밥값을 나눠야 할 때, **"송금 요청"** 기능도 있어요.

1. 채팅방에서 + 버튼 > **"송금 요청"**을 선택해요
2. 금액을 입력하고 보내면, 상대방이 확인 후 바로 보내줄 수 있어요

## 기능 3: 투표 만들기 - 모임 날짜 정할 때 딱!

"언제 만날까요?" "뭘 먹을까요?" 그룹 채팅방에서 의견을 모을 때, 투표 기능을 사용하면 깔끔해요.

### 따라하기

1. **그룹 채팅방**에 들어가요
2. 메시지 입력창 왼쪽의 **+ 버튼**을 눌러요
3. **"투표"**를 선택해요
4. **투표 제목**을 입력해요 (예: "이번 주 모임 날짜")
5. **선택 항목**을 추가해요 (예: "금요일", "토요일", "일요일")
6. 필요하면 **"복수 선택 허용"**을 켜요
7. **"완료"**를 누르면 투표가 채팅방에 올라가요

### 투표 결과 확인

- 투표를 누르면 실시간으로 **누가 어디에 투표했는지** 볼 수 있어요
- 투표가 끝나면 **결과를 한눈에** 확인할 수 있어요

## 기능 4: 캘린더 공유 - 가족, 친구와 일정 함께 관리하기

카카오톡 캘린더를 사용하면 가족이나 친구와 일정을 공유할 수 있어요. 따로 전화하거나 물어볼 필요 없이, 모두가 같은 캘린더를 보면서 일정을 관리할 수 있어요.

### 따라하기

1. 카카오톡 하단의 **더보기(...)**를 눌러요
2. **"캘린더"**를 선택해요
3. 원하는 날짜를 눌러서 **일정을 추가**해요
4. 일정 입력 화면에서 **"참여자 추가"**를 눌러요
5. 함께 공유하고 싶은 **카카오톡 친구를 선택**해요
6. **"저장"**을 누르면 상대방에게도 일정 알림이 가요!

### 공유 캘린더 만들기

특정 그룹(가족, 동호회 등)과 계속 일정을 공유하고 싶다면 공유 캘린더를 만들어보세요.

1. 캘린더 > 좌측 상단의 **햄버거 메뉴(세 줄)**를 눌러요
2. **"캘린더 추가"** > **"공유 캘린더 만들기"**를 선택해요
3. 이름을 입력하고 (예: "우리 가족 일정") 멤버를 초대해요
4. 이제 이 캘린더에 일정을 추가하면 멤버 모두가 볼 수 있어요

## 기능 5: 멀티프로필 - 상대에 따라 다른 프로필 보여주기

회사 동료에게는 전문적인 프로필을, 친구에게는 재미있는 프로필을 보여주고 싶을 때! 멀티프로필 기능을 사용하면 돼요.

### 따라하기

1. 카카오톡 > **더보기(...)** > 상단의 **내 프로필**을 눌러요
2. **"멀티프로필 관리"**를 선택해요
3. **"멀티프로필 만들기"**를 눌러요
4. 새로운 프로필 사진과 이름, 상태 메시지를 설정해요
5. 이 프로필을 보여줄 **친구들을 선택**해요
6. **"저장"**을 누르면 설정 완료!

### 활용 예시

- **회사용 프로필**: 깔끔한 사진 + 직함 상태 메시지
- **친구용 프로필**: 재미있는 사진 + 재치 있는 상태 메시지
- **가족용 프로필**: 가족 사진 + 따뜻한 상태 메시지

## 기능 6: 채팅방 검색 - 과거 대화 내용 바로 찾기

"그때 그 식당 이름이 뭐였지?", "지난번에 보낸 주소가 어디였더라?" 이런 경험, 많으시죠? 채팅방 검색 기능을 사용하면 과거 대화를 쉽게 찾을 수 있어요.

### 따라하기

1. 찾고 싶은 대화가 있는 **채팅방**에 들어가요
2. 우측 상단의 **돋보기(검색) 아이콘**을 눌러요
3. 찾고 싶은 **키워드**를 입력해요 (예: "식당", "주소", "약속")
4. 관련 메시지가 쭉 나타나요
5. 원하는 메시지를 누르면 해당 **대화 위치로 이동**해요

### 전체 채팅방 검색

특정 채팅방이 아니라 모든 채팅방에서 한꺼번에 검색하고 싶다면?

1. 카카오톡 **채팅 탭** 상단의 **검색창**을 눌러요
2. 키워드를 입력하면 **모든 채팅방**에서 관련 대화를 찾아줘요
3. 사진, 링크, 파일도 검색할 수 있어요

## 기능 7: 책갈피 - 중요한 메시지 저장하기

중요한 메시지, 주소, 약속 시간 등을 나중에 쉽게 찾을 수 있도록 **책갈피(즐겨찾기)**로 저장할 수 있어요.

### 따라하기

1. 저장하고 싶은 **메시지를 길게 눌러요**
2. 옵션 메뉴에서 **"책갈피"** 또는 **"즐겨찾기"**를 선택해요
3. 메시지에 별표(책갈피) 표시가 돼요

### 저장한 책갈피 확인하기

1. 해당 채팅방에서 우측 상단의 **메뉴(세 줄)**를 눌러요
2. **"책갈피"** 또는 **"즐겨찾기"**를 선택해요
3. 저장해둔 메시지 목록을 한눈에 볼 수 있어요

### 활용 팁

- 친구가 보내준 **맛집 정보**를 책갈피에 저장
- **약속 장소와 시간**을 책갈피에 저장
- 중요한 **업무 연락 내용**을 책갈피에 저장
- **계좌번호**나 **주소** 등 자주 필요한 정보를 저장

## 기능 8: 라이브톡 - 실시간 영상으로 소통하기

그룹 채팅방에서 실시간 영상을 공유하며 소통할 수 있는 기능이에요. 화상 통화와는 조금 달라요. 한 명이 방송하면 다른 사람들이 시청하면서 채팅으로 소통하는 방식이에요.

### 따라하기

1. **그룹 채팅방**에 들어가요
2. 메시지 입력창 왼쪽의 **+ 버튼**을 눌러요
3. **"라이브톡"**을 선택해요
4. **"시작"**을 누르면 실시간 방송이 시작돼요
5. 참여자들은 알림을 받고 들어와서 시청할 수 있어요
6. 시청자들은 **채팅**으로 반응할 수 있어요

### 활용 예시

- 여행 중 실시간으로 풍경 보여주기
- 모임에 못 온 친구에게 현장 분위기 전달하기
- 언박싱이나 요리 과정 공유하기
- 손주의 학예회를 할아버지 할머니에게 실시간으로 보여주기

## 기능 9: 공지 설정 - 중요한 메시지 고정하기

그룹 채팅방에서 중요한 공지사항을 올렸는데, 다른 대화에 묻혀서 아무도 못 봤다면? **공지 설정** 기능을 사용하면 메시지가 채팅방 상단에 고정돼요.

### 따라하기

1. 공지로 설정하고 싶은 **메시지를 길게 눌러요**
2. **"공지로 등록"**을 선택해요
3. 해당 메시지가 채팅방 **상단에 고정**돼요
4. 모든 참여자가 채팅방에 들어올 때마다 공지 내용을 볼 수 있어요

### 공지 해제하기

1. 상단에 고정된 공지 메시지를 눌러요
2. **"공지 해제"**를 선택하면 고정이 풀려요

### 활용 예시

- 모임 날짜/장소 공지
- 회비 입금 계좌 안내
- 행사 준비물 목록
- 동호회 규칙이나 안내사항

## 기능 10: 키워드 알림 - 내 이름이 언급되면 알려줘요

그룹 채팅방에서 대화가 많이 오가면, 나와 관련된 메시지를 놓치기 쉬워요. 키워드 알림을 설정해두면, 특정 단어가 포함된 메시지가 올 때 따로 알림을 받을 수 있어요.

### 따라하기

1. 카카오톡 > **더보기(...)** 탭 > 우측 상단 **설정(톱니바퀴)**을 눌러요
2. **"알림"**을 선택해요
3. **"키워드 알림"**을 눌러요
4. **"키워드 추가"**를 눌러요
5. 알림 받고 싶은 **키워드를 입력**해요 (예: 내 이름, "회의", "공지", "투표")
6. **"저장"**을 누르면 설정 완료!

### 활용 팁

- **내 이름**을 키워드로 등록하면, 누군가 나를 언급할 때 바로 알림을 받아요
- **"회의"**, **"미팅"** 같은 업무 관련 키워드를 등록해두면 편리해요
- 최대 **30개**까지 키워드를 등록할 수 있어요

## 보너스 꿀팁 3가지

### 나에게 보내기 (메모장처럼 활용)

1. 카카오톡 채팅 탭에서 **"나와의 채팅"**을 찾아요 (검색창에 내 이름 검색)
2. 여기에 메모, 링크, 사진 등을 보내두면 나만의 메모장으로 활용할 수 있어요
3. 컴퓨터 카카오톡과 연동되니까, 폰에서 보낸 내용을 PC에서 바로 확인할 수 있어요

### 대화 내용 백업하기

소중한 대화 내용을 잃어버리지 않으려면 백업을 해두세요.

1. 카카오톡 > 더보기 > 설정 > **"채팅"** > **"대화 백업"**
2. 백업을 해두면 폰을 바꿔도 대화 내용을 복원할 수 있어요

### 글자 크기 조절

카카오톡 글씨가 너무 작다면?

1. 카카오톡 > 더보기 > 설정 > **"화면"**
2. **"글자 크기"**에서 원하는 크기로 조절할 수 있어요
