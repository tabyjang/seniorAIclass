## 1. AI 시대의 직업 변화

### 1-1. 어떤 일이 바뀌고 있나요?

**사라지는 직업:**
```
⚠️ 단순 반복 업무:
- 데이터 입력
- 기초 번역
- 단순 디자인
- 기초 코딩
- 일반 사무
```

**새로 생기는 직업:**
```
✨ AI 시대 신직업:
- 프롬프트 엔지니어
- AI 트레이너
- AI 윤리 전문가
- 데이터 큐레이터
- AI-human 협업 매니저
```

**변화하는 직업:**
```
🔄 마케터:
Before: 광고 카피 작성
After: AI 도구로 10배 많은 콘텐츠 기획

🔄 개발자:
Before: 코드 한 줄씩 작성
After: AI가 생성한 코드 리뷰 및 아키텍처 설계

🔄 디자이너:
Before: 그림 그리기
After: AI 이미지 큐레이션 및 브랜드 전략
```

## 2. 미래에 필요한 5가지 핵심 역량

### 역량 1: AI 리터러시 (AI Literacy)

**의미:**
```
AI를 이해하고 활용하는 능력
```

**구체적 스킬:**
```
✅ 프롬프트 엔지니어링
✅ AI 도구 활용 능력
✅ AI의 한계와 편향 이해
✅ AI 결과물 평가 능력
```

**학습 방법:**
```
1. 매일 AI 도구 사용 (30분)
2. 프롬프트 연습
3. AI 윤리 교육
4. 최신 트렌드 팔로우
```

### 역량 2: 데이터 리터러시

**의미:**
```
데이터로 의사결정하는 능력
```

**구체적 스킬:**
```
✅ 기초 통계 이해
✅ 데이터 시각화
✅ 가설 검증 능력
✅ 데이터 기반 스토리텔링
```

**학습 방법:**
```
1. 엑셀/스프레드시트 고급 기능
2. Python 또는 SQL 기초
3. Tableau 또는 Power BI
4. 통계학 온라인 강의
```

### 역량 3: 창의적 문제해결

**의미:**
```
AI가 못 하는 복잡한 문제 해결
```

**구체적 스킬:**
```
✅ 비판적 사고
✅ 융합적 사고
✅ 직관과 통찰
✅ 감성 지능 (EQ)
```

**학습 방법:**
```
1. 다양한 분야 독서
2. 디자인 씽킹 워크숍
3. 크로스 펑셔널 프로젝트
4. 예술/창작 활동
```

### 역량 4: 디지털 커뮤니케이션

**의미:**
```
온라인에서 효과적으로 소통
```

**구체적 스킬:**
```
✅ 비동기 커뮤니케이션
✅ 원격 협업 툴 활용
✅ 디지털 프레젠테이션
✅ 온라인 네트워킹
```

**학습 방법:**
```
1. Notion, Slack 마스터
2. 화상회의 스킬
3. 디지털 콘텐츠 제작
4. 온라인 커뮤니티 참여
```

### 역량 5: 지속적 학습 능력

**의미:**
```
끊임없이 새로운 것을 배우는 능력
```

**구체적 스킬:**
```
✅ 학습하는 법을 아는 것
✅ 자기 주도 학습
✅ 실패에서 배우기
✅ 메타인지
```

**학습 방법:**
```
1. 온라인 강의 플랫폼 활용
2. 개인 학습 로드맵 작성
3. 멘토링/코칭
4. 학습 커뮤니티 참여
```

## 4. 커리어 전환 전략

### 4-1. 현재 상태 분석

**강점/약점 파악:**
```
내 강점:
- 현재 직무의 전문성
- 고유한 경험
- 네트워크

보완할 점:
- AI 도구 활용 능력
- 디지털 스킬
- 새로운 분야 지식
```

### 4-2. 1년 성장 계획

**분기별 목표:**
```
Q1: AI 기초 다지기
- ChatGPT 마스터
- 프롬프트 엔지니어링 학습
- AI 도구 5개 활용

Q2: 데이터 역량 강화
- Excel 고급 기능
- Python 또는 SQL 기초
- 데이터 분석 프로젝트 1개

Q3: 실전 적용
- 현재 업무에 AI 도입
- 자동화 프로젝트 완료
- 포트폴리오 작성

Q4: 전문성 확장
- 새로운 분야 학습
- 사이드 프로젝트
- 네트워킹 강화
```

### 4-3. 사이드 프로젝트 아이디어

**경험 쌓기:**
```
💡 AI 콘텐츠 제작:
- AI로 블로그 운영
- 유튜브 채널 (AI 편집)
- 뉴스레터 발행

💡 데이터 프로젝트:
- 공공 데이터 분석
- 개인 재무 분석 대시보드
- 취미 데이터 수집/분석

💡 자동화 프로젝트:
- 개인 업무 자동화
- 가계부 자동 정리
- 소셜 미디어 자동화

💡 AI 앱/서비스:
- 노코드로 웹앱 만들기
- AI 챗봇 만들기
- 자동화 워크플로우
```
