## 2. Runway Gen-2 마스터하기

### 2-1. Runway가 뭔가요?

**특징:**
```
✅ 텍스트로 영상 생성 (Text to Video)
✅ 이미지로 영상 생성 (Image to Video)
✅ 배경 제거/변경 (Green Screen 없이)
✅ 객체 추적/제거
✅ 무한 확장 (Infinite Image)
```

**가격:**
```
🆓 묶: 125 크레딧/월 (약 25초 영상)
💼 Standard: $12/월 (625 크레딧)
🎬 Pro: $28/월 (2250 크레딧)
```

### 2-2. Gen-2로 영상 생성

**텍스트로 영상 만들기:**
```
1. RunwayML.com 접속
2. "Text to Video" 선택
3. 프롬프트 입력

예시:
"A drone flying over a tropical beach at sunset, 
cinematic lighting, 4K quality, smooth motion"

4. 설정 조정
   - General/Motion 적절히
   - 침자로 Fixed Seed (재현성)
5. Generate 클릭
6. 4개 버전 중 선택
```

**이미지로 영상 만들기:**
```
1. "Image to Video" 선택
2. 이미지 업로드
3. Motion Brush로 움직일 영역 선택
   - 물결: Motion 3
   - 구름: Motion 2
   - 인물: Motion 1-2
4. 침자로 Fixed Seed
5. Generate
```

### 2-3. Green Screen (배경 제거)

**기능:**
```
- 자동 배경 제거 (크로마키 불필요)
- 인물만 분리
- 새 배경 합성
```

**사용법:**
```
1. 영상 업로드
2. "Green Screen" 선택
3. "Remove Background" 클릭
4. AI가 자동으로 인물 분리
5. 새 배경 업로드 또는 색상 선택
6. Export
```

### 2-4. Inpainting (객체 제거)

**기능:**
```
- 영상 속 원하지 않는 객체 제거
- 보행자, 차량 등 제거
- 자연스럽게 채움
```

**사용법:**
```
1. 영상 업로드
2. "Inpainting" 선택
3. 브러시로 제거할 영역 칠하기
4. Generate
5. 객체가 사라진 영상 생성
```

## 3. CapCut Pro로 모바일 편집

### 3-1. CapCut의 강점

**특징:**
```
✅ 완전 물론 (기본 기능)
✅ AI 자막 자동 생성
✅ 템플릿 풍부
✅ 모바일/PC 동기화
✅ 틱톡/인스타 최적화
```

**AI 기능:**
```
- 자동 자막 (한국어 지원)
- 음성 변환 (TTS)
- 노이즈 제거
- 배경 제거
- 스마트 컷 (자동 편집)
```

### 3-2. AI 자막 자동 생성

**사용법:**
```
1. CapCut 앱 실행
2. 영상 가져오기
3. "텍스트" → "자동 자막"
4. 언어 선택 (한국어)
5. "생성" 클릭
6. 자동으로 자막 생성됨
7. 수정 필요한 부분만 편집
```

**스타일 설정:**
```
- 폰트: Noto Sans KR 등
- 크기: 24-32pt
- 색상: 흰색 + 검은 테두리
- 애니메이션: Fade in/out
```

### 3-3. 템플릿 활용

**빠른 제작:**
```
1. "템플릿" 탭
2. 카테고리 선택 (vlog/여행/음식 등)
3. 마음에 드는 템플릿 선택
4. "사용하기" 클릭
5. 내 영상/사진으로 교체
6. 텍스트 수정
7. 완성!
```

### 3-4. AI 음성 (TTS)

**자연스러운 나레이션:**
```
1. "오디오" → "텍스트 음성 변환"
2. 스크립트 입력
3. 음성 선택 (한국어 목소리 다양)
4. 속도/톤 조절
5. 생성
```

## 4. Descript로 텍스트 기반 편집

### 4-1. Descript의 혁신

**개념:**
```
영상을 텍스트처럼 편집!
→ 글자를 지우면 영상도 삭제
→ 글자를 수정하면 음성도 수정
```

**기능:**
```
✅ 오버딕 (Overdub): 텍스트로 음성 생성
✅ filler word 제거 (음, 그러니까 등)
✅ 공백 자동 제거
✅ 화자 분리
✅ 스크린 레코딩
```

### 4-2. 텍스트 기반 편집

**사용법:**
```
1. 영상 업로드
2. 자동 전사 (transcription)
3. 텍스트 보기
4. 삭제하고 싶은 부분 텍스트 선택
5. Delete 키 → 영상도 삭제
6. 텍스트 수정 → 음성도 수정
```

**오버딕 (Overdub):**
```
1. "Overdub" 활성화
2. 텍스트 수정
3. AI가 내 목소리로 복원
4. 재녹음 없이 수정 완료
```

## 5. 실전 워크플로우

### 5-1. 유튜브 브이로그 제작

**단계별 제작:**
```
Step 1: 영 (촬영)
→ 폰으로 영
→ 4K 30fps 권장

Step 2: AI 자막 (CapCut)
→ 자동 자막 생성
→ 스타일 적용

Step 3: 하이라이트 (Descript)
→ filler word 제거
→ 공백 자동 제거

Step 4: B-roll (Runway)
→ AI로 장면 생성
→ 배경 제거

Step 5: 음악 (Suno AI)
→ BGM 생성
→ 저작권 걱정 없음

Step 6: 썸네일 (Midjourney + Canva)
→ AI 이미지
→ 텍스트 추가

총 소요 시간: 2시간 (기존 8시간)
```

### 5-2. 숏폼 콘텐츠 (쇼츠/릴스)

**Opus Clip 활용:**
```
1. 긴 영상 업로드 (유튜브 URL도 가능)
2. AI가 하이라이트 추출
3. 10개 후보 생성
4. 선택 후 편집
5. 자막 자동 생성
6. Export

→ 1시간 영상 → 10개 숏폼 (15분)
```

### 5-3. 강의/교육 영상

**Descript 워크플로우:**
```
1. 녹화 (Descript 내장 스튜디오)
2. 자동 전사
3. 텍스트 편집으로 영상 편집
4. 오버딕으로 실수 수정
5. 스크린샷/자료 삽입
6. 자막 생성
7. Export
```
