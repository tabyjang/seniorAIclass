## 서론: 회의록 작성, 이제 AI에게 맡기세요

"회의는 끝났는데 회의록은 언제 쓰죠?"

많은 직장인이 공감하는 고민입니다. 1시간 회의 후 30분씩 회의록을 정리하면 하루 업무 시간의 상당 부분이 사라집니다. 더욱이 중요한 내용을 놓치거나, 주관적인 해석이 들어가기도 하죠.

이제 AI가 회의를 실시간으로 녹음하고, 핵심 내용을 요약하며, 할 일 목록까지 자동으로 만들어줍니다. 본 가이드에서는 AI 회의록 도구의 활용법과 실전 팁을 소개합니다.

## 2. AI 회의록 작성 프로세스

### 2.1 전체 워크플로우

```
┌─────────────────────────────────────────────────────┐
│  1. 회의 녹음/녹화                                    │
│     └─ AI 도구로 실시간 음성인식                      │
├─────────────────────────────────────────────────────┤
│  2. 자동 전사 (Transcription)                         │
│     └─ 음성 → 텍스트 변환                             │
├─────────────────────────────────────────────────────┤
│  3. AI 요약 및 정리                                   │
│     └─ 핵심 내용, 결정사항, 액션 아이템 추출           │
├─────────────────────────────────────────────────────┤
│  4. 후처리 및 배포                                    │
│     └─ 형식화, 공유, 작업 관리 연동                    │
└─────────────────────────────────────────────────────┘
```

### 2.2 도구별 상세 사용법

#### 네이버 클로바 노트 (한국어 추천)

**설정 방법:**
1. clovadubbing.naver.com 접속
2. 클로바 노트 앱 설치 (iOS/Android)
3. 회의 시 '녹음 시작' 클릭
4. 자동으로 음성인식 및 텍스트화
5. 회의 종료 후 AI 요약 확인

**주요 기능:**
- 실시간 음성인식 (한국어 인식률 95%+)
- 스피커 자동 구분 (A, B, C)
- 핵심 내용 자동 하이라이트
- 키워드 추출
- 공유 기능 (PDF, 텍스트)

**팁:**
- 회의 시작 전 "참석자: 홍길동, 김철수, 박영희"라고 말하면 스피커 구분이 쉬움
- 중요 발언은 "중요: ~"이라고 말하면 AI가 하이라이트

#### Otter.ai (영어/글로벌)

**설정 방법:**
1. otter.ai 회원가입
2. 앱 설치 또는 웹 사용
3. 회의 시 'Record' 클릭
4. 실시간으로 전사 및 요약
5. 회의 후 공유

**고급 기능:**
- 캘린더 연동 (자동 녹음)
- Zoom/Teams 연동
- 맞춤형 어휘 사전 (회사/제품명 학습)
- 협업 기능 (팀원과 동시 편집)

**팁:**
- Custom Vocabulary에 회사 용어 등록
- 스피커 이름 태깅 (Learn Speaker Names)
- 하이라이트 기능으로 중요 부분 표시

## 3. AI로 회의록 정리하기

### 3.1 원본 텍스트 정제

**상황:** 녹음된 텍스트가 어수선함

**AI에게 요청:**
```
"다음 회의 녹취록을 정리해주세요.

요구사항:
1. 불필요한 말더듬, 반복 제거
2. 문장을 자연스럽게 연결
3. 같은 사람의 발언을 하나로 묶기
4. 시간 표기 추가 [10:15], [10:23]

[녹취 원본]"
```

### 3.2 핵심 내용 요약

**ChatGPT/Claude 프롬프트:**
```
"다음 회의 내용을 요약해주세요.

요약 형식:
📅 회의 정보
- 일시: 2026년 2월 5일
- 참석자: [추출]
- 주제: [추출]

🎯 핵심 논의 사항 (3-5개)
1. [주제]: [결정/논의 내용]
2. ...

✅ 결정사항 (Decision)
- [결정1]
- [결정2]

📋 액션 아이템 (Action Items)
| 담당자 | 할 일 | 마감일 | 우선순위 |
|--------|-------|--------|----------|
| [이름] | [할일] | [날짜] | High/Mid/Low |

❓ 다음 회의에서 논의할 사항
- [주제1]
- [주제2]

[회의 원문]"
```

### 3.3 실제 예시

**원본 (녹음 텍스트):**
```
어... 홍길동입니다. 그... 이번 프로젝트 일정인데요.
음... 개발팀이랑 디자인팀이랑 얘기를 좀 해봤는데...
아마 다음 주 금요일까지는 좀 어려울 것 같아요.
그래서... 음... 다음 주 월요일로 하면 어떨까 싶은데...
김철수 대리님 의견은 어떠세요?

네 김철수입니다. 저도 그게 좋을 것 같습니다.
그런데 테스트 기간을 얼마나 잡아야 할까요?
일주일이면 충분할까요? 아니면 더 필요할까요?

음... 아마 일주일이면 될 것 같은데...
박영희 과장님은 어떻게 생각하세요?
...
```

**AI 정제 후:**
```
📅 회의 정보
- 일시: 2026년 2월 5일 오전 10:00
- 참석자: 홍길동, 김철수, 박영희
- 주제: 프로젝트 일정 조정

🎯 핵심 논의 사항
1. 개발 완료 일정: 원래 금요일 → 다음 주 월요일로 연기
2. 테스트 기간: 일주일로 결정
3. 최종 배포 일정: 2월 17일로 확정

✅ 결정사항
- 개발 완료: 2월 10일(월)까지
- 테스트 기간: 2월 11일(화) ~ 17일(월)
- 최종 배포: 2월 17일(월)

📋 액션 아이템
| 담당자 | 할 일 | 마감일 | 우선순위 |
|--------|-------|--------|----------|
| 홍길동 | 개발 완료 및 내부 테스트 | 2/10 | High |
| 김철수 | 테스트 시나리오 작성 | 2/7 | High |
| 박영희 | 배포 환경 준비 | 2/14 | Mid |

❓ 다음 회의에서 논의할 사항
- 테스트 결과 리뷰 (2/14)
- 배포 체크리스트 최종 확인
```

## 4. 자동화 및 연동

### 4.1 워크플로우 자동화

**시나리오:** 회의 종료 후 자동으로 정리

**Zapier/Make 활용:**
```
트리거: Otter.ai에서 새 회의록 생성
↓
액션1: ChatGPT로 요약 요청
↓
액션2: Notion에 회의록 저장
↓
액션3: Slack으로 팀원에게 공유
↓
액션4: 구글 캘린더에 액션 아이템 등록
```

### 4.2 Notion 연동

**자동화 설정:**
1. Notion 데이터베이스 템플릿 생성
2. Zapier로 Otter.ai와 연동
3. 회의 종료 시 자동으로 Notion에 생성
4. AI 요약 자동 삽입

**Notion 템플릿 예시:**
```
# 회의록 템플릿

## 기본 정보
- 일시: {{회의일시}}
- 참석자: {{참석자}}
- 장소: {{장소/온라인}}

## 회의 목적
{{회의 목적}}

## 핵심 내용
{{AI 요약}}

## 액션 아이템
- [ ] {{할일1}}
- [ ] {{할일2}}

## 원문 보기
{{전체 녹취록 링크}}
```

### 4.3 작업 관리 도구 연동

**Asana/Monday/Trello 연동:**
```

↓

↓
마감일 설정
↓
우선순위 설정

## 5. 실전 팁 및 주의사항

### 5.1 녹음 품질 향상

**✅ 좋은 환경:**
- 조용한 공간
- 마이크 가까이
- 한 명씩 차례로 발언
- 중요 내용은 천천히, 명확하게

**❌ 피해야 할 것:**
- 여러 사람이 동시에 말하기
- 배경 소음 많은 장소
- 마이크 먼 곳에서 말하기
- 빠르거나 unclear한 발음

### 5.2 개인정보 보호

**주의사항:**
- 녹음 전 반드시 동의 구하기
- 민감한 정보(개인정보, 기밀)는 녹음하지 않기
- 녹음 파일 안전하게 저장
- 불필요한 녹음은 즉시 삭제

**동의 문구 예시:**
```
"본 회의는 AI를 활용해 회의록을 작성하기 위해 
녹음됩니다. 동의하시는 분은 '네'라고 말씀해주세요."
```

### 5.3 한계와 보완

**AI의 한계:**
- 전문 용어 인식 오류
- 동음이의어 구분 어려움
- 감정/맥락 이해 한계
- 방언/억양 인식 어려움

**보완 방법:**
- 중요 내용은 반복해서 말하기
- 키워드 명확히 발음하기
- 회의 후 핵심 내용 인간 검수
- 액션 아이템 별도 확인

```
