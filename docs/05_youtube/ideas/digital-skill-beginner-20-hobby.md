## 1️⃣ 스마트폰 사진/영상

### 스마트폰 사진 기초

**기본 설정:**
```
카메라 설정:
□ 그리드 켜기 (삼분할 법칙)
□ HDR 자동
□ 고화질 저장
□ 위치 정보 켜기

촬영 팁:
☑ 자연광 활용
☑ 배경 정리
☑ 피사체에 집중
☑ 여러 각도 시도
```

**구도와 앵글:**
```
삼분할 법칙:
- 화면을 가로세로 3등분
- 교차점에 피사체 배치
- 자연스러운 느낌

대칭 구도:
- 수평/수직 맞추기
- 반영 활용
- 건축물에 효과적

시선 유도:
- 선 활용 (도로, 난간)
- 피사체 방향 여백
- 프레임 안의 프레임
```

### 사진 편집 앱

**VSCO (묶음/Pro):**
```
기능:
- 필터 200+
- 기본 보정
- 그레인/빈티지 효과

사용법:
1. 사진 불러오기
2. 필터 선택 (A6, C1 추천)
3. 강도 조절
4. 추가 보정
5. 저장
```

**Lightroom Mobile (묶음):**
```
기능:
- RAW 편집
- 전문적 보정
- 프리셋 적용

보정 항목:
- 노출 (밝기)
- 대비
- 하이라이트/그림자
- 색온도
- 선명도
```

**Snapseed (묶음):**
```
Google의 묶은 사진 편집 앱

기능:
- 선택적 보정
- 힐링 (사물 제거)
- 투시 보정
- HDR 효과
```

### 브이로그/쇼츠 만들기

**CapCut (묶음):**
```
기능:
- 자동 자막
- 트렌디한 효과
- 음악 라이브러리
- 템플릿 제공

쇼츠 제작:
1. 15-60초 영상 촬영
2. CapCut으로 편집
3. 자막 추가
4. 음악 삽입
5. 해시태그와 업로드
```

**브이로그 콘텐츠 아이디어:**
```
일상 브이로그:
- 아침 루틴
- 직장인 일상
- 주말 일상
- 쇼핑/장보기

취미 브이로그:
- 독서 기록
- 요리 과정
- 운동 루틴
- 공부 기록

여행 브이로그:
- 당일치기 여행
- 카페 투어
- 맛집 탐방
- 야경 투어
```

### 시작하기

**필요한 것:**
```

□ 스타일러스 (Apple Pencil/S Pen)
□ 드로잉 앱

예산별:
저예산: 스마트폰 + 스타일러스 (₩50,000)
중간: iPad + Apple Pencil (₩500,000)
고급: iPad Pro + Procreate (₩1,000,000+)
```

```
가격: ₩15,000 (일회성)

기능:
- 200+ 브러시
- 레이어 시스템
- 타임랩스 녹화
- PSD 낮으기

입문자용 브러시:
- 6B Pencil (스케치)
- Studio Pen (선화)
- Soft Brush (채색)
```

```
만화/웹툰 전용

기능:
- 톤/효과 라인

가격:

- 연간: ₩45,000
```

## 3️⃣ 디지털 음악

### 시작하기

**GarageBand (iOS/macOS 묶음):**
```
기능:
- 가상 악기
- 드럼 루프
- 멀티트랙 녹음
- 공유 기능

시작하기:
1. 앱 실행
2. "Empty Project" 선택
3. Software Instrument
4. 피아노/기타 선택
5. 연주 시작
```

**FL Studio Mobile:**
```
가격: ₩20,000

기능:
- 시퀀서
- 신디사이저
- 드럼 머신
- 믹싱

장르:
- EDM
- 힙합
- 팝
- Lo-fi
```

### 음악 이론 기초

```
기본 개념:

코드:
- Major: 밝은 느낌 (C, G, F)
- Minor: 어두운 느낌 (Am, Em, Dm)
- 3가지 코드로 수백곡 가능

리듬:
- 4/4박자 (가장 일반적)
- BPM (분당 비트)
- 드럼 패턴

멜로디:
- 스케일 (C Major, A Minor)
- 화음 진행
- 멜로디 라인

학습 자료:
- YouTube: "기초 음악 이론"
- Hooktheory (웹사이트)
- 연주하면서 배우기
```
