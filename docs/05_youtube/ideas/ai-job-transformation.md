## 1. 2030년 일자리 전망

### 1.1 숫자로 보는 변화

**세계경제포럼(WEF) 2023 보고서**:
- 향후 5년 내 8,300만 개 일자리 감소
- 6,900만 개 새로운 일자리 생성
- 순감소: 1,400만 개

**맥킨지 글로벌 연구소**:
- 2030년까지 전 세계 3억 7,500만 명 직업 전환 필요
- 현재 업무의 30%가 자동화 가능
- 생성형 AI로 추가 자동화 가능 업무: 60-70%

## 4. 직업별 전환 로드맵

### 4.1 사무/행정직 → 데이터 분석가

**현재 스킬**: 엑셀, 문서 작성, 업무 프로세스 이해
**전환 필요 스킬**: Python, SQL, 데이터 시각화

**로드맵 (1년)**:
```
Month 1-3: 엑셀 고급 기능 + SQL 기초
Month 4-6: Python 데이터 분석 (Pandas, NumPy)
Month 7-9: 데이터 시각화 (Tableau, Power BI)
Month 10-12: 실전 프로젝트 + 포트폴리오
```

**학습 리소스**:
- Coursera: Google Data Analytics Certificate
- 노마드코더: 파이썬 무료 강의
- 데이터리안: SQL 첫걸음

### 4.2 마케터 → AI 마케팅 전문가

**현재 스킬**: 기획, 콘텐츠 제작, 광고 운영
**전환 필요 스킬**: AI 도구 활용, 데이터 분석, 자동화

**로드맵 (6개월)**:
```
Month 1-2: ChatGPT/Claude 마스터 + 프롬프트 엔지니어링
Month 3-4: AI 이미지/영상 도구 (Midjourney, Runway)
Month 5-6: 마케팅 자동화 + AI 기반 분석
```

**핵심 변화**:
- 카피 작성 → AI 프롬프트 + 에디팅
- 이미지 의뢰 → AI 이미지 직접 생성
- 수동 분석 → AI 기반 인사이트 도출

### 4.3 개발자 → AI 엔지니어

**현재 스킬**: 프로그래밍, 시스템 설계
**전환 필요 스킬**: ML/DL, AI 프레임워크, MLOps

**로드맵 (1년)**:
```
Month 1-3: 머신러닝 기초 (Andrew Ng 강의)

Month 7-9: LLM 활용 및 파인튜닝
Month 10-12: MLOps + 프로덕션 배포
```

### 4.4 디자이너 → AI 크리에이티브 디렉터

**현재 스킬**: 비주얼 디자인, 도구 활용
**전환 필요 스킬**: AI 이미지 도구, 브랜드 전략

**로드맵 (6개월)**:
```
Month 1-2: Midjourney/DALL-E/Stable Diffusion 마스터
Month 3-4: AI 워크플로우 통합 (포토샵 AI, Figma AI)
Month 5-6: 브랜드/전략 역량 강화
```

### 4.5 작가/에디터 → AI 콘텐츠 전략가

**현재 스킬**: 글쓰기, 스토리텔링, 편집
**전환 필요 스킬**: AI 글쓰기 도구, SEO, 콘텐츠 전략

```
Month 1-2: AI 글쓰기 도구 완전 정복

```

## 5. 전환 성공을 위한 실전 조언

### 5.1 T자형 인재 전략

```
        │
        │ 깊은 전문성
        │ (AI 활용 능력)
━━━━━━━━┿━━━━━━━━ 넓은 이해
        │ (비즈니스, 커뮤니케이션, 리더십)
        │
```

**전략**: 하나의 전문 분야 + 다양한 분야의 기본 이해

### 5.2 포트폴리오 우선 전략

학위/자격증보다 **실제로 만든 것**이 중요합니다.

**포트폴리오 예시**:
- 데이터 분석: Kaggle 프로젝트 3개
- 마케팅: AI로 진행한 캠페인 케이스
- 개발: GitHub에 AI 프로젝트
- 디자인: AI 활용 작업물 컬렉션

### 5.3 사이드 프로젝트로 시작

직장 다니면서 전환 준비하기:

```
평일 저녁: 1시간 학습
주말: 3-4시간 프로젝트
→ 6개월 후 포트폴리오 완성
→ 이직 또는 부서 이동
```

### 5.4 네트워크 활용

**가입할 커뮤니티**:
- 관심 분야 오픈 채팅/슬랙
- LinkedIn 그룹
- 오프라인 밋업/세미나
- 온라인 스터디 그룹
