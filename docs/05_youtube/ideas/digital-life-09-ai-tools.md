## 1. AI 이미지 생성

### 1-1. 이미지 생성 AI가 뭐예요?

**정의:**  
글로 설명하면 AI가 그림을 그려주는 기술

**쉬운 비유:**
> 마술사가 "빨간 풍선을 들고 있는 고양이를 그려줘"라고 부탁하면  
> 순식간에 그림을 그려주는 거예요.  
> 그것도 여러 가지 스타일로!

### 1-2. 추천 도구

**DALL-E 3 (ChatGPT Plus)**
```
장점:
- ChatGPT 안에서 바로 사용
- 한국어도 잘 이해
- 다양한 스타일

사용법:
1. ChatGPT Plus 가입
2. "DALL-E" 모드 선택
3. 원하는 이미지 설명
4. 4가지 버전 생성

예시 프롬프트:
"강아지가 핫도그를 먹는 모습을 
픽사 애니메이션 스타일로 그려줘"
```

**Midjourney (미드저니)**
```
장점:
- 예술적인 품질
- 디테일이 뛰어남
- 커뮤니티 활발

사용법:
1. Discord 가입
2. Midjourney 서버 접속
3. 채팅창에 /imagine 입력
4. 영어로 설명 입력

예시:
/imagine a cat wearing sunglasses, 
pop art style, colorful --ar 16:9

※ Discord 사용법이 필요해요
※ 유료 ($10/월부터)
```

**Bing Image Creator (빙 이미지)**
```
장점:
- 물론
- Microsoft 계정만 있으면 OK
- 한국어 지원
- DALL-E 기술 사용

사용법:
1. bing.com/create 접속
2. Microsoft 로그인
3. 한국어로 설명 입력
4. 이미지 생성

예시:
"겨울 숲 속에서 눈사람을 만드는 아이들, 
수채화 스타일"
```

**Stable Diffusion (스테이블 디퓨전)**
```
장점:
- 완전 물론 (오픈소스)
- 컴퓨터에 설치해서 사용
- 무제한 생성

단점:
- 컴퓨터 사양 필요
- 설치가 복잡

사용법:
1. stable-diffusion-webui 설치
2. 모델 다운로드
3. 프롬프트 입력
4. 이미지 생성

※ 기술 지식 필요
```

### 1-3. 프롬프트 작성법

**좋은 프롬프트의 구성:**
```
[주제] + [세부사항] + [스타일] + [품질]
```

**예시:**
```
❌ 나쁜 예:
"고양이 그려줘"

✅ 좋은 예:
"귀여운 주황색 고양이가 창가에서 
햇살을 받으며 자고 있는 모습, 
일본 애니메이션 스타일, 
따뜻한 색감, 고화질"
```

**스타일 키워드:**
```
🎨 미술 스타일:
- 수채화 (watercolor)
- 유화 (oil painting)
- 픽셀 아트 (pixel art)
- 팝아트 (pop art)
- 인상파 (impressionist)

🎬 애니메이션:
- 지브리 스타일
- 디즈니/픽사 스타일
- 3D 렌더링
- 벡터 일러스트

📸 사진 스타일:
- DSLR 사진
- 필름 사진
- 인스타그램 필터
- 블랙앤화이트
```

### 1-4. 활용 예시

**일상에서:**
```
- 블로그 썸네일 제작
- 프레젠테이션 이미지
- 생일 축하 카드
- 동화책 일러스트
- 캐릭터 디자인
```

**주의사항:**
```
⚠️ 저작권:
- 상업용으로 사용 시 주의
- 학습된 이미지의 저작권 문제
- AI 생성 이미지 표기 권장

⚠️ 진위:
- AI 생성 이미지를 실제 사진처럼 속이면 안 됨
- 딥페이크 등 악용 금지
```

## 7. AI 도구 사용 시 주의사항

### 7-1. 저작권과 윤리

**⚠️ 주의할 점:**
```
1. 학습 데이터의 저작권
   - AI가 학습한 데이터에 대한 권리
   - 상업적 사용 시 주의

2. 생성물의 저작권
   - AI 생성 콘텐츠의 저작권归属
   - 국가/플랫폼별 정책 확인

3. 윤리적 사용
   - 타인의 초상권 침해 금지
   - 허위 정보 생성 금지
   - 악의적인 딥페이크 금지
```

### 7-2. 플랫폼별 정책

**유튜브:**
```
- AI 생성 콘텐츠 표기 권장
- "altered or synthetic content" 설정
```

**인스타그램/페이스북:**
```
- AI 생성 콘텐츠 라벨 자동 부착
- 정책 위반 시 제재
```

**상업용 사용:**
```
- 각 도구의 라이선스 확인
- 유료 플랜의 상업용 허용 범위
- 고용 계약서 확인
```

### 7-3. 품질 관리

**AI 결과물은 완벽하지 않아요:**
```
- 이미지: 손가띝 개수, 이상한 부분 확인
- 음성: 발음 확인, 자연스러움 확인
- 영상: 화질, 어색한 부분 확인
- 글: 사실 확인, 문맥 확인
```

**필수 후처리:**
```
- AI 생성 → 검토 → 수정 → 완성
- 100% AI 의존 NO
- 인간의 검증 필수
```
