# 온라인 보안 기초: 디지털 자산 지키기

## 🤔 이런 분들에게 추천해요

- 비밀번호 관리가 어려우신 분
- 해킹/피싱에 대한 걱정이 있으신 분
- 온라인 쇼핑/뱅킹을 하시는 분
- 개인정보 보호에 관심 있으신 분

---

## 🔐 보안의 3대 원칙

| 원칙 | 설명 | 방법 |
|------|------|------|
| **기밀성** | 권한 있는 사람만 접근 | 비밀번호, 암호화 |
| **무결성** | 정보가 변조되지 않음 | 백업, 검증 |
| **가용성** | 필요할 때 접근 가능 | 백업, 복구 계획 |

## 1️⃣ 비밀번호 관리

### 강력한 비밀번호 만들기

**비밀번호 원칙:**
```
강력한 비밀번호:
✅ 12자 이상
✅ 대/소문자 + 숫자 + 특수문자
✅ 사전 단어 사용 안 함
✅ 개인정보 포함 안 함
✅ 각 서비스별 다른 비밀번호

예시:
❌ password123
❌ 12345678
❌ qwer1234
❌ 생일/전화번호

✅ Tr0ub4dor&3 (Xkcd 방식)
✅ correct-horse-battery-staple
✅ 7H#pL$9vQw@2
```

**비밀번호 생성법:**
```
암기하기 쉬운 복잡한 비밀번호:

1. 문장 사용법:
   "나는 2024년에 부자가 될 것이다!"
   → n2bEb24!
   
2. 키보드 패턴:
   1qaz@WSX3edc
   → 왼쪽 키보드 세로줄
   
3. 치환법:
   "I love coffee"
   → 1L0v3c0ff33
   
4. 랜덤 단어 조합:
   "파란 하늘 아래 고양이"
   → 파란_하늘_고양이_2024
```

### 비밀번호 관리자 사용

**추천 도구:**
```
비밀번호 관리자:

🔐 Bitwarden (묶음/Pro)
- 오픈소스
- 크로스 플랫폼
- 가족 공유

🔐 1Password ($2.99/월)
- 사용자 친화적
- 여행 모드
- 팀 기능

🔐 Google Password Manager
- Chrome 내장
- 묶음
- Google 계정 연동

🔐 iCloud Keychain
- Apple 생태계
- 묶음
- Face ID/Touch ID
```

**설정 방법:**
```
Bitwarden 설정:

Step 1: 가입 및 설치
□ bitwarden.com 가입
□ 브라우저 확장 설치
□ 모바일 앱 설치

Step 2: 마스터 비밀번호
□ 12자 이상
□ 기억할 수 있으면서 복잡하게
□ 절대 잃어버리지 않도록

Step 3: 기존 비밀번호 import
□ 브라우저에서 export
□ CSV 파일로 Bitwarden에 import
□ 중복/취약 비밀번호 확인

Step 4: 자동 생성 사용
□ 새 계정 생성 시
→ Bitwarden이 강력한 비밀번호 생성
→ 자동 저장
```

## 2️⃣ 2단계 인증 (2FA)

### 2FA 설정하기

**필수 설정 서비스:**
```
우선순위:
🥇 금융 (은행, 카드, 증권)
🥇 이메일 (Gmail, 네이버)
🥈 쇼핑 (쿠팡, 네이버)
🥈 클라우드 (Google Drive, iCloud)
🥉 SNS (Facebook, Instagram)
```

**설정 방법:**
```
Google 계정:
1. myaccount.google.com
2. 보안 → 2단계 인증
3. 시작
4. 전화번호 등록
5. 백업 코드 저장 (중요!)

네이버 계정:
1. 내정보 → 보안설정
2. 2단계 인증
3. OTP 앱 또는 문자 인증
4. 등록 완료

은행/카드:
- 각 은행 앱에서 설정
- 보통 생체인증 + PIN
- OTP 카드 또는 앱
```

### 인증 앱 추천

```
Google Authenticator:
- 묶음
- 간단함
- 기기 변경 시 마이그레이션 필요

Microsoft Authenticator:
- 묶음
- 백업/복구 기능
- Windows 연동

Authy:
- 묶음
- 클라우드 백업
- 다중 기기 지원

네이버 OTP:
- 네이버 전용
- 묶음
- 간편함
```

## 3️⃣ 피싱 공격 방어

### 피싱 인식하기

**피싱의 종류:**
```
이메일 피싱:
❌ 보낸 사람 주소 확인
   - 공식 도메인인가?
   - 오타가 있는가?
   
❌ 긴급/협박성 문구
   - "계정이 해킹됐습니다"
   - "24시간 내에 확인하세요"
   - "당첨되셨습니다"

❌ 의심스러운 링크
   - URL 확인 (hover)
   - https://가 아닌 http://
   - 이상한 도메인

❌ 개인정보 요구
   - 은행이 비밀번호를 묻지 않음
   - 카드 번호 요구
   - 주민등록번호 요구
```

**메시지 피싱 (스미싱):**
```
의심 증상:
❌ 모르는 번호에서 온 문자
❌ URL 포함된 메시지
❌ "택배", "인증", "당첨" 등 키워드
❌ 앱 설치 요구

대처법:
✅ 링크 클릭 금지
✅ 공식 앱/사이트에서 직접 확인
✅ 의심되면 삭제
✅ 118로 신고
```

**전화 피싱 (보이스 피싱):**
```
의심 증상:
❌ 금융기관 사칭
❌ 검찰/경찰 사칭
❌ 자녀 사칭 ("딸/아들이에요")
❌ 대출/투자 권유
❌ 긴급하다는 압박

대처법:
✅ 전화 끊고 공식 번호로 확인
✅ 금융기관은 전화로 비밀번호 안 물음
✅ 가족에게 연락 확인
✅ 112 신고
```

### 안전한 클릭 습관

```
링크 확인 체크리스트:

□ URL 직접 확인
  - 주소창에 직접 입력
  - 북마크 사용
  - 구글 검색 후 접속

□ HTTPS 확인
  - 자물쇠 아이콘
  - https:// 로 시작

□ 도메인 주의
  - naver.com (정상)
  - naver-login.com (가짜)
  - amazon.com (정상)
  - amazon-security.com (가짜)

□ 짧은 URL 주의
  - bit.ly, tinyurl 등
  - 실제 목적지 확인
  - 확장 서비스 사용
```

## 4️⃣ 소프트웨어 업데이트

### 자동 업데이트 설정

**OS 업데이트:**
```
Windows:
설정 → Windows 업데이트
→ 고급 옵션
→ 자동 업데이트 켜기
→ 활성 시간 설정

macOS:
시스템 설정 → 일반 → 소프트웨어 업데이트
→ 자동 업데이트 켜기

iOS:
설정 → 일반 → 소프트웨어 업데이트
→ 자동 업데이트 → 다운로드 및 설치

Android:
설정 → 시스템 → 시스템 업데이트
→ 자동 업데이트 설정
```

**앱 업데이트:**
```
iPhone:
App Store → 프로필
→ 자동 업데이트 켜기

Android:
Play Store → 프로필
→ 설정 → 네트워크 환경설정
→ 자동 업데이트 → Wi-Fi에서만
```

### 보안 소프트웨어

**필수 보안:**
```
Windows Defender:
- Windows 내장
- 묶음
- 실시간 보호
- 정기 검사

macOS:
- Gatekeeper
- XProtect
- 추가 백신 필요 없음 (일반 사용자)

iOS/Android:
- 공식 앱스토어만 사용
- 불필요한 백신 앱
- 기본 보안으로 충분
```

## 5️⃣ 백업 전략

### 3-2-1 백업 법칙

```
3: 3개의 복사본
  - 원본 + 백업1 + 백업2

2: 2개의 다른 매체
  - 외장하드 + 클라우드
  - USB + NAS

1: 1개는 오프사이트
  - 집 밖의 장소
  - 클라우드 저장소
  - 친구/가족 집
```

### 백업 도구

**클라우드 백업:**
```
Google Drive (15GB 묶음):
□ 중요 문서
□ 사진/동영상
□ 설정 백업

iCloud (5GB 묶음):
□ iPhone 백업
□ 사진
□ 문서

OneDrive (5GB 묶음):
□ Windows 통합
□ Office 문서
□ 설정 동기화

Dropbox (2GB 묶음):
□ 간단한 파일 공유
□ 히스토리 관리
```

**로컬 백업:**
```
외장하드:
- 주 1회 정기 백업
- Time Machine (Mac)
- 파일 히스토리 (Windows)

NAS (Network Attached Storage):
- 자동 백업
- RAID 구성
- 원격 접근

USB 메모리:
- 중요 문서
- 휴대용
- 암호화 필수
```

### 백업 자동화

```
Windows 파일 히스토리:
설정 → 시스템 → 저장공간
→ 고급 저장소 설정
→ 파일 히스토리 백업

macOS Time Machine:
시스템 설정 → 일반 → Time Machine
→ 외장하드 선택
→ 자동 백업

Google One:
설정 → Google 백업
→ 기기 데이터 백업
→ 자동화 설정
```

## 🚨 비상 대응 계획

### 계정 해킹 시

**즉시 조치:**
```
1. 비밀번호 변경
   - 해당 서비스
   - 다른 서비스(동일 비밀번호 사용 시)

2. 2단계 인증 확인
   - 설정된 2FA 확인
   - 해커가 추가했는지 확인

3. 활동 기록 확인
   - 로그인 기록
   - 수상한 활동

4. 연결된 서비스 확인
   - 제3자 앱 연결 해제
   - 권한 회수

5. 고객센터 연락
   - 계정 정지 요청
   - 복구 절차 안내
```

### 피해 발생 시

```
금전적 피해:
□ 즉시 은행/카드사 연락
□ 카드 정지 요청
□ 거래 취소 요청
□ 경찰 신고 (112)

개인정보 유출:
□ 한국인터넷진흥원 신고 (118)
□ 개인정보보호위원회 신고
□ 신용조회회사에 fraud alert 설정
□ 주민등록번호 변경 검토
```
