## 2. AI로 수식 마스터하기

### 2.1 기본 수식 생성

**시나리오: 합계와 평균 계산**

**AI에게 물어보기:**
```
"A1:A10에 있는 숫자의 합계와 평균을 
구하는 엑셀 수식을 알려줘"
```

**AI의 답변:**
```
합계: =SUM(A1:A10)
평균: =AVERAGE(A1:A10)

설명:
- SUM: 지정한 범위의 모든 숫자를 더함
- AVERAGE: 지정한 범위의 평균값 계산
```

### 2.2 조건부 수식

**시나리오: 성적 등급 부여**

**AI에게 물어보기:**
```
"B열에 점수가 있을 때,
90점 이상 A, 80점 이상 B, 70점 이상 C, 
나머지 F를 부여하는 수식을 만들어줘"
```

**AI의 답변:**
```excel
=IF(B2>=90,"A",IF(B2>=80,"B",IF(B2>=70,"C","F")))

또는 IFS 함수 사용 (Excel 2019 이상):
=IFS(B2>=90,"A",B2>=80,"B",B2>=70,"C",B2<70,"F")
```

### 2.3 복잡한 수식

**시나리오: VLOOKUP + 조건**

**AI에게 물어보기:**
```
"A열은 사원번호, B열은 이름, C열은 부서인 
표가 있을 때, F2 셀에 사원번호를 입력하면 
이름과 부서를 자동으로 가져오는 수식을 만들어줘"
```

**AI의 답변:**
```excel
이름: =VLOOKUP(F2,A:C,2,FALSE)
부서: =VLOOKUP(F2,A:C,3,FALSE)

또는 XLOOKUP 사용 (더 유연함):
이름: =XLOOKUP(F2,A:A,B:B)
부서: =XLOOKUP(F2,A:A,C:C)
```

## 3. 데이터 정리 자동화

### 3.1 데이터 클렌징

**시나리오: 더러운 데이터 정리**

**원본 데이터 문제:**
- 이름에 공백이 불규칙하게 들어감
- 전화번호 형식이 일정하지 않음
- 대소문자가 섞여 있음

**AI에게 물어보기:**
```
"엑셀에서 다음 작업을 하는 방법을 알려줘:
1. A열의 텍스트 앞뒤 공백 제거
2. B열의 전화번호를 000-0000-0000 형식으로 통일
3. C열을 모두 대문자로 변환"
```

**AI의 답변:**
```excel
1. 공백 제거: =TRIM(A2)
2. 전화번호 정리: =TEXT(B2,"000-0000-0000")
3. 대문자 변환: =UPPER(C2)
```

### 3.2 데이터 분할/병합

**시나리오: 성명 분리/병합**

**AI에게 물어보기:**
```
"A열에 '홍길동'처럼 이름이 붙어있을 때 
성과 이름을 분리하는 방법과
B열과 C열에 나뉘어진 성명을 
합치는 방법을 알려줘"
```

**AI의 답변:**
```excel
성과 이름 분리:
- 성: =LEFT(A2,1)
- 이름: =RIGHT(A2,LEN(A2)-1)

성과 이름 합치기:
=CONCAT(B2,C2)
또는 =B2&C2
```

### 3.3 중복 제거 및 필터링

**AI에게 물어보기:**
```
"A열에서 중복된 값을 제거하고 
고유값만 추출하는 방법을 알려줘"
```

**AI의 답변:**
```
방법 1: 중복 제거 기능 사용
1. A열 선택
2. 데이터 → 중복 항목 제거
3. 확인

방법 2: 고유 함수 사용 (Excel 365)
=UNIQUE(A:A)

방법 3: 조건부 서식으로 중복 표시
=COUNTIF($A$2:$A$100,A2)>1
```

## 4. 보고서 자동화

### 4.1 피벗 테이블 생성

**시나리오: 부서별 매출 집계**

**AI에게 물어보기:**
```
"부서별로 매출을 집계하는 피벗 테이블을 
만드는 단계를 알려줘.
A열: 날짜, B열: 부서, C열: 매출액"
```

**AI의 답변:**
```
1. 데이터 범위 선택 (A1:C100)
2. 삽입 → 피벗 테이블
3. 새 워크시트에 생성 선택
4. 필드 목록에서:
   - 행: 부서
   - 값: 매출액 (합계)
5. 완료

피벗 테이블 옵션:
- 값 표시 방식: %로 표시 가능
- 필터: 특정 부서만 표시 가능
- 정렬: 매출액 순 정렬 가능
```

### 4.2 차트 자동 생성

**AI에게 물어보기:**
```
"월별 매출 추이를 보여주는 라인 차트를 
만드는 방법을 알려줘"
```

**AI의 답변:**
```
1. 날짜와 매출 데이터 선택
2. 삽입 → 차트 → 라인 차트
3. 차트 디자인 선택
4. 차트 제목 추가: '월별 매출 추이'
5. 데이터 레이블 추가 (선택사항)

팁:
- 추세선 추가: 우클릭 → 추세선 추가
- 이중 축: 두 가지 데이터 비교 시 유용
```

### 4.3 자동화된 보고서 템플릿

**AI에게 물어보기:**
```
"매주 자동으로 업데이트되는 보고서를 
만들려면 어떤 기능을 사용해야 해?"
```

**AI의 답변:**
```
자동화 방법:

1. 데이터 연결
   - 데이터 → 데이터 가져오기
   - 외부 파일이나 데이터베이스 연결

2. 파워 쿼리 (Power Query)
   - 데이터 변환 자동화
   - 새로 고침으로 업데이트

3. 매크로 (VBA)
   - 반복 작업 자동화
   - 개발자 도구 → 매크로 기록

4. 수식 활용
   - TODAY(), NOW() 함수로 날짜 자동 입력
   - OFFSET()으로 동적 범위 설정
```

## 5. 실전 활용 시나리오

### 5.1 영업 보고서 자동화

**상황:** 매일 업데이트되는 영업 데이터

**자동화 프로세스:**
1. 원본 데이터 시트에 새 데이터 입력
2. AI가 생성한 수식으로 자동 계산
3. 피벗 테이블에서 자동 집계
4. 차트에서 자동 업데이트
5. 요약 보고서 자동 생성

**핵심 수식:**
```excel
이번 달 매출 합계:
=SUMIFS(C:C,A:A,">="&DATE(YEAR(TODAY()),MONTH(TODAY()),1),
A:A,"<"&DATE(YEAR(TODAY()),MONTH(TODAY())+1,1))

전월 대비 증감률:
=(이번달-전월)/전월*100
```

### 5.2 인사 관리 시트

**상황:** 직원 정보 관리 및 휴가 계산

**AI 활용:**
```
"입사일을 기준으로 근속 연수를 계산하고
남은 연차를 자동으로 계산하는 
수식을 만들어줘"
```

**생성된 수식:**
```excel
근속 연수:
=DATEDIF(입사일,TODAY(),"Y")

남은 연차 (15일 기준):
=15-사용연차

올해 발생 연차:
=MIN(15+DATEDIF(입사일,TODAY(),"Y"),25)
```

### 5.3 프로젝트 관리

**상황:** 업무 진행 상황 추적

**자동화 기능:**
- 진행률 자동 계산
- 마감일 임박 알림
- 완료된 작업 자동 정리

**조건부 서식 활용:**
```
마감일이 3일 이내: 빨간색 강조
진행률 100%: 회색 처리
지연된 작업: 노란색 배경
```

## 6. 고급 자동화: VBA와 함께

### 6.1 AI로 VBA 코드 생성

**시나리오: 반복 작업 매크로**

**AI에게 물어보기:**
```
"엑셀에서 다음 작업을 자동으로 실행하는 
VBA 매크로 코드를 만들어줘:
1. A1 셀에 현재 날짜 입력
2. 데이터 범위를 복사해서 새 시트에 붙여넣기
3. 파일을 PDF로 저장"
```

**AI가 생성한 코드:**
```vba
Sub 일일보고서생성()
    ' 현재 날짜 입력
    Range("A1").Value = Date
    
    ' 데이터 복사
    Sheets("원본데이터").Range("A1:D100").Copy
    
    ' 새 시트 생성 및 붙여넣기
    Sheets.Add
    ActiveSheet.Name = Format(Date, "yyyy-mm-dd")
    Range("A1").PasteSpecial
    
    ' PDF 저장
    ActiveSheet.ExportAsFixedFormat _
        Type:=xlTypePDF, _
        Filename:="C:\Reports\일일보고서_" & Format(Date, "yyyymmdd") & ".pdf"
    
    MsgBox "보고서 생성 완료!"
End Sub
```

### 6.2 AI로 데이터 분석

**Claude 활용 예시:**
```
"다음 데이터를 분석해줘:
- 1월 매출: 1500만원
- 2월 매출: 1800만원
- 3월 매출: 1650만원
- 4월 매출: 2100만원

엑셀에서 분석할 수 있는 차트와 
인사이트를 알려줘"
```

**AI의 분석:**
```
분석 결과:
1. 성장률: 평균 월 12% 성장
2. 최고 성장: 3월→4월 (27% 증가)
3. 권장 차트: 
   - 월별 매출 막대 차트
   - 성장률 라인 차트 (이중 축)
   - 누적 매출 영역 차트

인사이트:
- 4월에 특별한 이벤트나 캠페인이 있었는지 확인 필요
- 3월 하락의 원인 분석 필요
```

## 7. 자주 묻는 질문

**Q: AI가 만든 수식이 안 돼요**
- 데이터 범위가 올바른지 확인
- 텍스트/숫자 형식 확인
- 지역 설정(콤마/세미콜론) 확인

**Q: 한글 데이터가 깨져요**
- UTF-8 인코딩으로 저장
- 한글 버전 Excel 사용 권장
- CONCAT 대신 CONCATENATE 사용

**Q: 대용량 데이터 처리가 느려요**
- 수식 대신 파워 쿼리 사용
- 배열 수식 사용 자제
- 데이터 필터링 후 작업
