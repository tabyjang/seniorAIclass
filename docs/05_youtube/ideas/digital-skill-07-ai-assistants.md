## 1. AI 어시스턴트 비교

### 1-1. 주요 AI 도구 특징

| 도구 | 강점 | 최적 사용처 | 가격 |
|------|------|------------|------|
| **ChatGPT** | 범용성 | 아이디어, 작성 | $20/월 |
| **Claude** | 긴 분석 | 문서, 코드 | $20/월 |
| **Copilot** | 통합 | MS 365 | $30/월 |

**상황별 추천:**
```

→ Perplexity (실시간 정보)

→ ChatGPT
```

## 2. Claude 마스터하기

### 2-1. Claude가 뭔가요?

**특징:**
```
✅ Anthropic 개발
✅ 100K 토큰 (75,000단어) 처리
✅ 문서 분석 특화
✅ 안전성 중시
✅ 코드 작성 우수
```

**ChatGPT와 차이:**
```
Claude 장점:
- 더 긴 문서 처리
- 더 신중한 답변
- 코드 설명 상세
- 거부율 낮음

ChatGPT 장점:
- 더 빠른 응답
- 더 창의적
- 플러그인 생태계
- DALL-E 통합
```

### 2-2. 긴 문서 분석

**논문/계약서 분석:**
```
프롬프트:
"다음 문서를 분석해주세요.

[PDF 또는 텍스트 붙여넣기]

분석 요청:
1. 핵심 결론 3가지
2. 주요 arguments
3. 잠재적 문제점
4. 내 입장에서 고려할 점
5. 실행 가능한 action items"
```

**여러 문서 비교:**
```
프롬프트:
"다음 3개의 견적서를 비교 분석해주세요.

[문서 1]
[문서 2]
[문서 3]

비교 항목:
- 가격
- 포함 서비스
- 제외 사항
- 결제 조건
- 추천 (이유 포함)"
```

### 2-3. 코드 분석 및 작성

**코드 리뷰:**
```
프롬프트:
"다음 코드를 리뷰해주세요.

[코드 붙여넣기]

요청사항:
1. 버그/오류 찾기
2. 성능 개선점
3. 보안 이슈
4. 가독성 개선 제안
5. 주석 추가 버전"
```

**코드 설명:**
```
프롬프트:
"이 코드가 뭘 하는지 초보자도 이해할 수 있게 설명해주세요.
한 줄씩 설명하고, 전체 흐름도 알려주세요."
```

### 2-4. Claude의 Projects 기능

**프로젝트 활용:**
```
1. 프로젝트 생성 (예: "마케팅 전략")
2. 관련 문서 업로드
   - 브랜드 가이드
   - 시장 조사
   - 경쟁사 분석
3. Claude가 컨텍스트 유지
4. 일관된 답변 제공
```

## 3. Microsoft Copilot 활용

### 3-1. Copilot 종류

**Copilot Pro ($20/월):**
```
- Word: 문서 작성/편집
- Excel: 데이터 분석/공식
- PowerPoint: 슬라이드 제작
- Outlook: 이메일 작성
- OneNote: 노트 정리
```

**Copilot 365 ($30/월):**
```
- Teams: 회의 요약/액션 아이템
- SharePoint: 문서 검색
- 기업용 보안/관리
```

### 3-2. Word에서 활용

**문서 작성:**
```
"현재 문서를 바탕으로
제품 소개서 초안을 작성해주세요.

요구사항:
- 2페이지 분량
- 전문적이지만 친근한 톤
- 장점 5가지 강조
- CTA 포함"
```

**문서 편집:**
```
"이 문서를 더 간결하게 다듬어주세요.
- 문장 길이 줄이기
- 중복 제거
- 가독성 향상"
```

### 3-3. Excel에서 활용

**데이터 분석:**
```
"이 테이블을 분석해주세요.

분석 요청:
1. 월별 추세 파악
2. 이상값 찾기
3. 상위 3개 항목
4. 피벗 테이블 제안
5. 차트 추천"
```

**공식 작성:**
```
"C2:C100의 평균을 구하고,
50 이상이면 '우수', 미만이면 '개선'이라고 표시하는
IF 공식을 만들어주세요."
```

### 3-4. PowerPoint에서 활용

**슬라이드 제작:**
```
"[주제]에 대한 프레젠테이션을 10장으로 만들어주세요.

1장: 제목
2장: 문제 상황
3-5장: 해결책
6-8장: 사례
9장: 기대효과
10장: CTA

각 슬라이드에 적절한 레이아웃과
더미 텍스트를 넣어주세요."
```

**디자인 개선:**
```
"현재 슬라이드를 더 전문적으로 보이게 디자인 개선해주세요.
- 색상 팔레트 제안
- 폰트 추천
- 이미지 배치"
```

## 4. Perplexity로 리서치하기

### 4-1. Perplexity가 뭔가요?

**특징:**
```
✅ AI + 실시간 검색
✅ 출처 표시
✅ 신뢰할 수 있는 정보
✅ 물론 사용 가능
```

**ChatGPT와 차이:**
```
ChatGPT: 학습된 정보만 (2024년 4월까지)
Perplexity: 실시간 검색 + AI 요약
```

### 4-2. 효과적인 리서치

**최신 정보 검색:**
```
질문:
"2024년 2월 기준
한국 AI 스타트업 투자 현황과
주요 투자 유치 기업을 알려주세요."

결과:
- 실시간 검색 결과
- 뉴스 출처 링크
- 요약 정리
- 관련 질문 추천
```

**심층 분석:**
```
질문:
"ChatGPT vs Claude 비교
- 가격
- 성능
- 특징
- 추천 사용처"

→ 표로 정리된 비교
→ 각 항목의 출처
→ 결론 및 권고
```

### 4-3. Copilot(Copilot) vs Perplexity

**비교:**
```
Perplexity:
- 검색 중심
- 출처 중요시
- 리서치 특화

Gemini:
- Google 서비스 통합
- YouTube 요약
- Gmail/캘린더 연동
```

## 5. AI 어시스턴트 통합 전략

### 5-1. 업무별 AI 선택 가이드

**마케팅:**
```
1. Perplexity: 시장 조사
2. ChatGPT: 카피 작성
3. Claude: 긴 보고서 분석
4. Copilot: PPT 제작
```

**개발:**
```
1. Copilot: 코드 작성
2. Claude: 코드 리뷰
3. ChatGPT: 아키텍처 설계
4. Perplexity: 기술 조사
```

**경영/기획:**
```
1. Claude: 계약서/문서 검토
2. Perplexity: 경쟁사 분석
3. Copilot: 보고서 작성
4. ChatGPT: 브레인스토밍
```

### 5-2. 워크플로우 예시

**콘텐츠 제작:**
```
Step 1: Perplexity
→ 최신 트렌드 리서치

Step 2: ChatGPT
→ 아이디어 발산

Step 3: Claude
→ 긴 글 작성/편집

Step 4: Copilot
→ PPT/문서 변환
```

**업무 자동화:**
```
Step 1: Claude
→ 요구사항 문서 분석

Step 2: ChatGPT
→ 자동화 방안 설계

Step 3: Copilot
→ Excel/문서 작성

Step 4: Claude
→ 코드 검토
```

### 6-1. 무료로 최대 활용

**무료 플랜 조합:**
```
ChatGPT 묶: 일상 대화, 간단한 작성

Perplexity 묶: 리서치 (제한적)
Copilot 묶: MS 기본 기능
```

**효율적 사용:**
```
- 할 일 큐에 쌓아두기
- 유료 버전은 집중 사용
- 여러 계정 활용 (개인/업무)
```

### 6-2. 유료 구독 선택 가이드

**한 개만 선택한다면:**
```
💼 업무 전반: Copilot 365
✍️ 작성/분석: Claude
🎯 범용: ChatGPT Plus
🔍 리서치: Perplexity Pro
```

**두 개 조합 추천:**
```
Best 조합 1:
ChatGPT Plus ($20) + Perplexity ($20) = $40
→ 범용 + 리서치

Best 조합 2:
Claude Pro ($20) + Copilot ($20) = $40
→ 분석 + MS 통합

Best 조합 3:
ChatGPT Plus ($20) + Copilot ($20) = $40
→ 범용 + 업무 효율
```
