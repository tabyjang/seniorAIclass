# 유튜브 채널 기본 설정 체크리스트

**작성일:** 2026-02-02  
**목적:** 유튜브 채널 시작 전 기본 요소 준비

---

## 1. 채널 기본 설정

### 채널 아이덴티티
- [ ] 채널명 확정
- [ ] 채널 슬로건/태그라인
- [ ] 채널 카테고리 설정 (교육/기술)
- [ ] 타겟 시청자 정의 (50대 이상 시니어)

**예시 아이디어:**
1. **채널명:** "시니어AI클래스", "50대AI학교", "AI돈벌이연구소"
2. **슬로건:** "50대부터 시작하는 AI 돈벌이", "AI는 어렵지 않아요", "당신의 첫 AI 코치"
3. **타겟:** 50-70대 사업가, 퇴직 후 재취업 희망자, 온라인 마케팅 시작하는 중년

### 시각적 요소
- [ ] 프로필 사진 (로고 또는 대표 이미지)
- [ ] 채널 배너 (2560 x 1440)
- [ ] 채널 소개 문구 작성
- [ ] 링크 설정 (홈페이지, 문의처)

**예시 아이디어:**
1. **프로필:** 골드(#D4A52C) 포인트의 원형 로고 + "AI" 텍스트
2. **배너:** 좌측에 강사 사진, 우측에 "50대 AI 돈벌이의 시작" 문구
3. **소개 문구:** "50대 이상을 위한 AI 실전 교육. ChatGPT부터 AI 이미지 생성까지, 사업에 바로 적용하는 AI 활용법을 알려드립니다."

---

## 2. 인트로 (0-5초)

### 기술적 요소
- [ ] 인트로 영상/애니메이션 제작
- [ ] 배경음악 선정 (저작권 문제 없는 음원)
- [ ] 인트로 길이: 3-5초
- [ ] 로고 애니메이션 스타일 확정

### 콘텐츠 요소
- [ ] 채널 로고 노출
- [ ] 채널명 노출
- [ ] 슬로건 노출 (선택사항)

### 인트로 스타일 옵션
- [ ] 심플 타이포그래피
- [ ] 모션 그래픽
- [ ] 실사 촬영 기반
- [ ] AI 생성 이미지 기반

**예시 아이디어:**
1. **심플 타이포:** 검정 배경 → 골드색 채널명 페이드인 → "AI로 돈 벌기 시작!" 서브타이틀
2. **모션 그래픽:** 반짝이는 파티클 효과 중앙에 로고 등장 + 경쾌한 효과음 3번
3. **실사 기반:** 강사가 카메라를 보며 미소짓는 장면 2초 → 로고 전환 1초

## 3. 오프닝 멘트 (5-15초)

### 인사말 구성
- [ ] 기본 인사말 작성
  - 예: "안녕하세요, AI 강사 [이름]입니다"
  - 예: "50대부터 시작하는 AI, [채널명]입니다"
  
- [ ] 채널/영상 소개 한줄
  - 예: "오늘은 ChatGPT로 PPT 만드는 방법 알려드릴게요"

### 오프닝 멘트 템플릿
```
[인사말] + [오늘의 주제] + [왜 봐야하는지]
```

### 오프닝 멘트 예시
- **버전 A (친근한):** "안녕하세요! 오늘도 AI로 돈 버는 방법, 바로 알려드릴게요"
- **버전 B (전문적인):** "안녕하세요. 50대 사업가를 위한 AI 활용법, [주제] 시작하겠습니다"
- **버전 C (궁금증 유발):** "ChatGPT로 한 달에 100만원 벌 수 있다는데, 어떻게 하는 걸까요?"

**주제별 오프닝 예시:**
1. **ChatGPT 영상:** "ChatGPT 쓰고 계시죠? 근데 제대로 쓰고 계신가요? 오늘은 사업에 바로 적용하는 프롬프트 3가지 알려드릴게요"
2. **AI 이미지 생성:** "포스터, 블로그 썸네일, SNS 이미지... 디자이너 안 쓰고 10초 만에 만드는 방법, 궁금하시죠?"
3. **AI로 돈 버는 방법:** "이 영상 보고 한 달에 100만원씩 부수입 얻으신 분들이 계세요. 어떻게 했는지 바로 보여드릴게요"

---

## 4. 중간 멘트 (전환/강조)

### 전환 멘트
- [ ] 섹션 이동 시 멘트
  - 예: "자, 그럼 본격적으로 시작해볼게요"
  - 예: "이제 핵심 내용입니다"
  - 예: "잠깐, 여기서 중요한 팁 하나"

### 강조 멘트
- [ ] 중요 포인트 강조
  - 예: "여기서 꼭 기억하세요"
  - 예: "이 부분이 가장 중요합니다"
  - 예: "다시 한번 강조드릴게요"

### 호응 유도 멘트
- [ ] 구독 요청
  - 예: "마음에 드셨다면 구독 부탁드려요"
  - 예: "앞으로도 매주 AI 꿀팁 올라옵니다"

- [ ] 좋아요/댓글 요청
  - 예: "도움되셨으면 좋아요 한번"
  - 예: "궁금한 점은 댓글로 남겨주세요"

**예시 아이디어:**
1. **3분 구독 유도:** "여기까지 잘 따라오셨나요? 이런 실전 팁 매주 올라와요. 구독하고 놓치지 마세요"
2. **문제 해결 유도:** "이 방법 말고 다른 방법도 궁금하신가요? 댓글에 남겨주시면 다음 영상으로 만들어드릴게요"
3. **커뮤니티 유도:** "이 팁 써보셨다면 댓글에 결과 좀 알려주세요. 성공 사례 모아서 다음에 소개해드릴게요"

## 5. 클로징/엔딩 (마무리)

### 마무리 멘트
- [ ] 요약 멘트
  - 예: "오늘 [주제] 알려드렸는데, 잘 이해되셨나요?"

- [ ] 다음 영상 예고
  - 예: "다음 시간에는 [다음주제] 다룰게요"

- [ ] 마무리 인사
  - 예: "그럼 오늘도 성공적인 하루 되세요"

### CTA (Call To Action)
- [ ] 구독 요청
- [ ] 댓글 참여 요청
- [ ] 홈페이지/문의 링크 안내

### 엔딩 화면
- [ ] 구독 버튼 애니메이션
- [ ] 추천 영상 2개 배치
- [ ] 채널 로고/정보

**예시 아이디어:**
1. **구독 유도 화면:** 좌측에 구독 버튼 애니메이션 + 우측 상단 "다음 영상: ChatGPT 프롬프트 10선" + 우측 하단 "관련 영상: AI로 블로그 글 쓰기"
2. **CTA 구성:** "오늘 배운 내용 실전에서 써보세요" + 구독 버튼 클릭 애니메이션 + 배경에 채널 로고 워터마크
3. **맞춤형 엔딩:** 시청자가 본 영상 주제와 관련된 2개 영상 썸네일 노출 (예: ChatGPT 영상 → "프롬프트 고급편", "ChatGPT로 홍보글 쓰기")

## 7. 준비할 추가 요소

### 썸네일
- [ ] 썸네일 템플릿 제작
- [ ] 폰트/색상 규칙 확정
- [ ] 썸네일 문구 패턴 (숫자, 질문, 감정)

**예시 아이디어:**
1. **숫자 썸네일:** 큰 숫자 "3" + "ChatGPT로 돈 버는 법" + 강사 얼굴 우측 하단
2. **비포/애프터:** 좌측 "전" (복잡한 화면), 우측 "후" (깔끔한 결과물) + "AI로 10초 해결!"
3. **질문형:** "ChatGPT 이걸 몰라서 손해보셨나요?" + 충격 표정의 강사 컷

### 자막 스타일
- [ ] 자막 폰트 선택
- [ ] 자막 색상/위치 규정
- [ ] 강조 자막 스타일

**예시 아이디어:**
1. **기본 자막:** Pretendard 폰트, 흰색 바탕에 검정 테두리, 화면 하단 1/3 지점
2. **강조 자막:** 채널 골드색(#D4A52C) 배경 + 흰색 글씨 + 화면 중앙에 3초간 노출
3. **키워드 자막:** 중요 단어는 빨간색 또는 노란색으로 강조 + 폰트 크기 1.5배

### 효과음
- [ ] 구독 알림 효과음
- [ ] 좋아요 효과음
- [ ] 전환 효과음

**예시 아이디어:**
1. **구독 효과음:** "띵~" 소리의 알림음 + 벨소리
2. **좋아요 효과음:** "딸깍" 소리의 클릭음
3. **전환 효과음:** "슈우웅" 경쾌한 전환음 (섹션 이동 시)

## 8. 촬영/녹화 체크리스트

### 장비
- [ ] 카메라/휴대폰 설정
- [ ] 마이크 테스트
- [ ] 조명 설정
- [ ] 배경 정리

### 복장
- [ ] 촬영 복장 확정
- [ ] 색상 규칙 (금색/검정색 계열 추천)

**예시 아이디어:**
1. **정장 스타일:** 네이비 정장 + 흰 셔츠 + 골드 포켓스퀘어 (신뢰감 주는 전문가 이미지)
2. **스마트 캐주얼:** 검정 카라티 + 베이지 재킷 (친근함 + 전문성)
3. **일관된 색상:** 채널 골드색(#D4A52C) 포인트 넥타이 또는 브로치 착용

### 진행
- [ ] 대본 리허설
- [ ] 말하기 속도 조절 (느리게)
- [ ] 시선 처리 (렌즈 보기)

**예시 아이디어:**
1. **리허설 방법:** 거울 앞에서 3회 연습 → 휴대폰으로 녹화해서 확인 → 실제 촬영
2. **말하기 속도:** 일반 대화보다 20% 느리게, 중요 포인트는 1초 정지 후 계속
3. **시선 팁:** 렌즈 바로 위/아래 보지 말고 정중앙 보며, 화면 끝에 스티커 붙여두고 그곳 보기

## 9. 첫 영상 기획

### 첫 영상 아이디어 (택 1)
- [ ] 채널 소개 + 왜 시작했는지
- [ ] 가장 쉬운 AI 도구 소개
- [ ] 시니어 성공 사례 소개
- [ ] 자주 묻는 질문 TOP 5

**구체적인 첫 영상 3가지 옵션:**
1. **옵션 A: 채널 소개형**
   - 제목: "50대 AI 돈벌이 시작합니다"
   - 구성: 왜 이 채널 시작했는지(1분) + 내가 AI로 성공한 이야기(1분) + 앞으로 알려줄 것들(1분)
   
2. **옵션 B: 바로 꿀팁형**
   - 제목: "ChatGPT 처음 쓰시는 분들, 이 3가지만 기억하세요"
   - 구성: 인사(10초) → 첫 번째 팁(1분) → 두 번째 팁(1분) → 세 번째 팁(1분) → 구독 유도(30초)
   
3. **옵션 C: 공감 유발형**
   - 제목: "50대라서 AI 어렵다고요? 저도 그랬습니다"
   - 구성: 어려웠던 경험 공유(1분) → 극복 과정(1분) → 지금 제가 사용하는 방법(1분) → 응원 멘트(30초)

### 첫 영상 구조
```
1. 인트로
2. 자기소개 (10초)
3. 채널 소개 (20초)
4. 앞으로의 방향 (20초)
5. 첫 번째 꿀팁 (3분)
6. 마무리 + 구독 요청
```
