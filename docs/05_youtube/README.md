## 폴더 구조

```
05_youtube/
├── README.md                    # 이 파일
├── scripts/                     # 대본 모음
│   ├── shorts/                  # 쇼츠 대본
│   └── longform/                # 장편 영상 대본
├── thumbnails/                  # 썸네일 기획
├── ideas/                       # 영상 아이디어
└── analytics/                   # 성과 분석
```

## 파일명 규칙
