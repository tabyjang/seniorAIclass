## 🎯 영상 제작 가이드라인

### 필수 원칙

1. **느리게 말하기** - 일반 유튜브의 70% 속도
2. **큰 글씨** - 화면 글자 최소 24pt
3. **반복 설명** - 중요한 건 2-3번 반복
4. **실수 OK** - 완벽하지 않아도 친근하게
5. **칭찬 많이** - "잘하셨어요!", "대단해요!"

### 영상 구조 템플릿

```
[0:00-0:10] 후킹 (오늘 뭘 배울지)
[0:10-0:30] 인사 + 공감 ("처음엔 저도 어려웠어요")
[0:30-끝] 본 내용 (단계별, 천천히)
[마지막 30초] 요약 + 다음 예고 + 구독 유도
```

### 썸네일 공식

- **큰 글씨** (3-5단어)
- **밝은 색상** (노랑, 주황, 하늘색)
- **표정** (놀람 😲, 미소 😊)
- **before/after** (가능하면)
