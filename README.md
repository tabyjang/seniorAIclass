# 시니어 AI 클래스 프로젝트

## 📋 프로젝트 개요

**프로젝트명**: 시니어 AI 클래스 (Senior AI Class)  
**대상**: 50-70대 시니어  
**목표**: 12주 만에 AI 전문가로 성장하는 완벽한 가이드  
**총 강의수**: 50강  
**개발 기간**: 12주

---

## 🎯 프로젝트 목표

1. **커리큘럼**: 5단계 레벨별 50강 상세 매뉴얼
2. **플랫폼**: 홈페이지 + 블로그 + 유튜브 연동
3. **콘텐츠**: 영상 스크립트, 블로그 글, PDF 자료
4. **비즈니스**: 무료 → 유료 멤버십 → 강사 양성

---

### Phase 4: 정식 런칭 (Week 9-10)

## 💰 비즈니스 모델

### 수익 구조

1. **무료 체험**
   - Lv.1-2 일부 강의 (YouTube)
   - 블로그 기본 팁
   - 무료 체험 수업

2. **월간 멤버십: 29,000원**
   - 전체 50강 무제한 수강
   - 실습 자료 다운로드
   - Q&A 게시판 접근

3. **연간 멤버십: 290,000원** (17% 할인)
   - 전체 강의 무제한
   - 특별 자료 제공
   - 우선 상담

4. **고급 패키지**
   - Lv.1-3: 150,000원
   - Lv.4-5 전문가: 500,000원
   - 1:1 코칭: 100,000원/시간

5. **강사 양성**
   - 인증 과정: 1,000,000원
   - 수익 배분: 강의료 70%
