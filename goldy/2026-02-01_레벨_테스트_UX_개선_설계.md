## 로직 개요

### 판정 방식: **점수제 (최대 5점)**

```
예 = 1점
아니오 = 0점

총점에 따른 레벨 배정:
• 0-1점 → Lv.1 초보자
• 2점 → Lv.2 중급자  
• 3점 → Lv.3 고급자
• 4-5점 → Lv.4 전문가
```

**이유**: 시니어는 복잡한 분기(갈래)보다 순차적인 질문이 이해하기 쉬움

---

## 5개 질문 구성

### 질문 1: 디지털 기초
```
스마트폰이나 컴퓨터로 
인터넷 검색이 가능하신가요?

[예] [아니오]
```
**의도**: 디지털 문해력 확인 (Lv.1의 가장 기본적인 기준)

---

### 질문 2: AI 경험
```
ChatGPT나 Claude 같은 
AI를 사용해보셨나요?

[예] [아니오]
```
**의도**: AI에 대한 경험 유무 (Lv.1과 Lv.2의 경계)

---

### 질문 3: 창작 활동
```
AI로 편지, 그림, 여행계획 등
무엇인가를 '만들어본' 적 있나요?

[예] [아니오]
```
**의도**: AI를 활용한 결과물 제작 경험 (Lv.2와 Lv.3의 경계)

---

### 질문 4: 공유/가르침
```
가족이나 친구에게 
AI 사용법을 알려준 적 있나요?

[예] [아니오]
```
**의도**: 타인에게 AI를 가르치는 경험 (Lv.3과 Lv.4의 경계)

---

### 질문 5: 수익/전문성
```
AI로 수익을 창출하거나
강사 활동을 하고 싶으신가요?

[예] [아니오]
```
**의도**: 전문가로서의 진로 의향 (Lv.3과 Lv.4의 최종 구분)

## 판정 로직 상세

```javascript
// 의사코드
function calculateLevel(score) {
  if (score <= 1) {
    return {
      level: 1,
      name: "초보자",
      color: "#E85D04",
      message: "기초부터 차근차근 시작해보세요",
      recommend: "Lv.1 초보자 과정 (15강/3주)"
    };
  } else if (score === 2) {
    return {
      level: 2,
      name: "중급자",
      color: "#FBBF24", 
      message: "AI와 즐겁게 놀 준비가 되셨네요",
      recommend: "Lv.2 중급자 과정 (12강/3주)"
    };
  } else if (score === 3) {
    return {
      level: 3,
      name: "고급자",
      color: "#2A9D8F",
      message: "AI로 능력자가 되실 준비가 되셨어요", 
      recommend: "Lv.3 고급자 과정 (12강/3주)"
    };
  } else {
    return {
      level: 4,
      name: "전문가",
      color: "#0077B6",
      message: "AI로 수익을 만들 준비가 되셨네요",
      recommend: "Lv.4 전문가 과정 (11강/3주)"
    };
  }
}
```

## 시니어 배려 기능

### 1. **음성 지원 (TTS)**
```javascript
// 질문 읽어주기
function speakQuestion(text) {
  const utterance = new SpeechSynthesisUtterance(text);
  utterance.lang = 'ko-KR';
  utterance.rate = 0.8; // 느린 속도
  speechSynthesis.speak(utterance);
}

// 🔊 버튼 클릭 시 읽어줌
```

### 2. **뒤로 가기 허용**
- 이전 질문으로 돌아갈 수 있음
- "이전 질문" 버튼 하단에 배치

### 3. **진행률 시각화**
- 5개 동그라미로 표시
- 완료된 질문은 ●, 현재는 ◉, 남은 것은 ○

### 4. **시간 제한 없음**
- 급하게 만들지 않아도 됨
- "천천히 생각해보세요" 문구 표시
