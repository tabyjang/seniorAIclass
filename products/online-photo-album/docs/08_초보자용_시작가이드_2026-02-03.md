## 이 앱이 뭔가요?

**온라인 포토앨범**은 사진을 인터넷에 저장하는 앱이에요.

**기존 방식의 문제:**
- 휴대폰이 고장나면? → 사진 날아감 😢
- 컴퓨터 바꾸면? → 사진 옮겨야 함 😢
- 다른 기기에서 보려면? → USB로 복사해야 함 😢

**이 앱을 쓰면:**
- 인터넷에 저장 → 어디서든 볼 수 있어요 ✅
- 기기가 고장나도 → 사진은 안전해요 ✅
- 가족과 공유 → 링크만 보내면 돼요 ✅

## 앱 사용법

### 1단계: 앱 열기

1. `albums.html` 파일을 더블클릭해서 열어요
2. 또는 VS Code에서 "Live Server"로 실행해요

### 2단계: 회원가입

1. **"로그인"** 버튼을 클릭해요
2. **"회원가입"** 탭을 클릭해요
3. 이메일 주소를 입력해요 (예: myemail@gmail.com)
4. 비밀번호를 입력해요 (6자 이상)
5. 비밀번호를 한 번 더 입력해요
6. **"회원가입"** 버튼을 클릭해요

**주의:** 이메일 인증이 필요할 수 있어요!
- 입력한 이메일로 확인 메일이 갈 수 있어요
- 메일함을 확인하고 링크를 클릭하세요

### 3단계: 로그인

1. 회원가입한 이메일과 비밀번호로 로그인해요
2. 성공하면 "내 앨범" 화면이 나와요

### 4단계: 앨범 만들기

1. **"+ 새 앨범"** 버튼을 클릭해요
2. 앨범 이름을 입력해요 (예: "손주 사진")
3. 설명을 입력해요 (선택사항)
4. 날짜를 선택해요 (선택사항)
5. **"앨범 만들기"** 버튼을 클릭해요

### 5단계: 사진 올리기

1. 만든 앨범을 클릭해요
2. 사진을 올리는 방법 2가지:
   - **방법 A:** 파란 영역에 사진 파일을 끌어다 놓기
   - **방법 B:** 파란 영역을 클릭하고 사진 선택하기
3. 여러 장을 한 번에 선택할 수 있어요
4. 업로드 진행률이 표시돼요
5. 완료되면 갤러리에 사진이 나타나요!

### 6단계: 사진 보기

1. 사진을 클릭하면 크게 볼 수 있어요
2. 좌우 화살표로 넘길 수 있어요
3. 키보드 ← → 키로도 넘길 수 있어요
4. ESC 키를 누르면 닫혀요

## 문제가 생겼을 때

### "로그인이 필요합니다" 라고 나와요
→ 로그인 버튼을 눌러서 로그인하세요

### "이메일 또는 비밀번호가 틀렸습니다" 라고 나와요
→ 비밀번호를 다시 확인하세요
→ 회원가입을 안 했다면 먼저 회원가입하세요

### "파일 크기는 10MB 이하여야 합니다" 라고 나와요
→ 너무 큰 사진이에요. 작은 사진을 올려주세요

### 사진이 안 올라가요
→ 인터넷 연결을 확인하세요
→ JPG, PNG, GIF, WebP 파일만 가능해요

### 앨범이 안 만들어져요
→ 앨범 이름을 입력했는지 확인하세요
→ 로그인 상태인지 확인하세요

**photos 테이블 (사진 정보)**
| 필드 | 설명 | 예시 |
|------|------|------|
| id | 고유 번호 | photo-123 |
| album_id | 앨범 번호 | abc-123-def |

| file_url | 사진 주소 | https://...photos/... |
| file_size | 파일 크기 | 2048000 (바이트) |
| width | 가로 크기 | 1920 |
| height | 세로 크기 | 1080 |
| description | 설명 | 생일파티 |
| location | 장소 | 제주도 |
| taken_at | 촬영일 | 2024-01-15 |
| is_favorite | 즐겨찾기 | true/false |
| created_at | 업로드 시간 | 2024-02-03 10:35:00 |

### Storage 버킷 (사진 파일 저장소)

```
버킷 이름: photos
공개 여부: 공개 (누구나 URL로 접근 가능)
파일 크기 제한: 10MB
허용 파일: JPEG, PNG, GIF, WebP
```

### 보안 설정 (RLS)

- 사용자는 자기 앨범/사진만 볼 수 있어요
- 다른 사람 앨범은 볼 수 없어요
- 이건 서버에서 자동으로 처리돼요
