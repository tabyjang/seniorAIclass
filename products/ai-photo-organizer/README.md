# 📸 AI 사진 정리사

> "수천 장 사진, AI가 알아서 정리해드려요"

## 📌 한 줄 소개
사진 자동 분류, 중복 삭제, 인물별/장소별 앨범 생성

## 🎯 타겟 사용자
- 사진이 너무 많아 정리 못하는 시니어
- "손주 사진만 모아보고 싶어요"
- 핸드폰 용량 부족한 분

## ✨ 핵심 기능
- **자동 분류**: 인물/장소/날짜별 정리
- **중복 삭제**: 비슷한 사진 찾아서 정리
- **인물 인식**: "손주 사진만 모아줘"
- **앨범 생성**: 자동으로 예쁜 앨범

## 🛠️ 기술 스택
- Claude Vision (이미지 분석)
- 얼굴 인식 AI
- 중복 감지 알고리즘

## 💰 수익 모델
- 무료: 100장까지
- 프리미엄: 월 9,900원 (무제한)
