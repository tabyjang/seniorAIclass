## 3. 핵심 알고리즘

### 중복 감지
`python
from imagehash import average_hash
from PIL import Image

def find_duplicates(images):
    hashes = {}
    for img_path in images:
        h = average_hash(Image.open(img_path))
        if h in hashes:
            # 중복 발견!
            yield (img_path, hashes[h])
        hashes[h] = img_path
`
