# AI 사진 정리사 PRD

**작성일**: 2026-02-03 | **버전**: v1.0

---

## 1. 제품 개요

### 제품명
AI 사진 정리사

### 한 줄 소개
수천 장 사진을 AI가 자동으로 분류하고 정리해주는 서비스

---

## 2. 핵심 기능

### 자동 분류
```
📁 내 사진 (5,432장)
    ↓ AI 분석
📂 정리된 앨범:
├── 👨‍👩‍👧 가족 (1,234장)
│   ├── 손주 민수 (456장)
│   ├── 손녀 지민 (389장)
│   └── 자녀들 (389장)
├── 🏠 집 (892장)
├── ✈️ 여행 (567장)
│   ├── 제주도 2025 (123장)
│   └── 부산 2024 (98장)
├── 🍽️ 음식 (445장)
└── 🌸 풍경 (1,294장)
```

### 중복 사진 삭제
```
🔍 중복 사진 발견!

비슷한 사진 23세트 (총 89장)

[세트 1] 손주 사진 4장
- 거의 똑같은 사진이에요
- 가장 잘 나온 1장만 남길까요?

✅ 정리하면 68장 공간 확보! (약 340MB)

[정리하기] [직접 선택]
```

### 인물 인식
```
👤 인물 등록

AI가 찾은 얼굴: 12명

1. 👦 이 사람은 누구예요? [이름 입력: 민수]
2. 👧 이 사람은 누구예요? [이름 입력: 지민]
...

등록 후:
"민수 사진만 보여줘" → 456장 표시!
```

### 자동 앨범 생성
```
🎁 AI 추천 앨범

"민수의 성장 앨범 (2020-2026)"
- 연도별 베스트 사진 자동 선정
- 예쁜 레이아웃 적용
- PDF/인화 주문 가능

[미리보기] [앨범 만들기]
```

## 3. 타겟 사용자

| 대상 | 문제 | 해결 |
|------|------|------|
| 시니어 | 사진 너무 많아 정리 못함 | AI 자동 정리 |
| 부모님 | "손주 사진만 보고 싶어" | 인물별 분류 |
| 용량 부족 | 핸드폰 꽉 참 | 중복 삭제 |

## 4. 시니어 특화 기능

- 🔊 **음성 명령**: "손주 사진 보여줘"
- 📱 **큰 썸네일**: 시니어 친화 UI
- 🖨️ **인화 주문**: 앨범 책으로 배송
- 👨‍👩‍👦 **가족 공유**: 선택한 앨범 자녀에게 전송

## 5. 기술 스택

| 분류 | 기술 |
|------|------|
| 이미지 분석 | Claude Vision |
| 얼굴 인식 | face_recognition 라이브러리 |
| 중복 감지 | 이미지 해시 비교 |
| 저장 | 로컬 / 클라우드 |

---

## 6. 워크플로우

```
[사진 폴더 선택]
      ↓
[1. 전체 스캔] ──→ 메타데이터 추출
      ↓
[2. AI 분석]
  ├─ 장면 분류 (풍경/음식/인물)
  ├─ 얼굴 인식 + 그룹화
  └─ 중복 감지
      ↓
[3. 정리 제안]
  ├─ 앨범 구조 제안
  └─ 삭제 후보 표시
      ↓
[4. 사용자 확인]
      ↓
[5. 실행]
```
