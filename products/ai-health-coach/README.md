# 🏋️ AI 건강 코치

> "매일 건강 체크, AI 코치가 챙겨드려요"

## 📌 한 줄 소개
식단 기록, 건강 체크, 약 복용 알림을 AI가 관리해주는 건강 도우미

## 🎯 타겟 사용자
- 당뇨/고혈압 관리가 필요한 시니어
- 약 복용 시간을 자주 잊는 분
- 건강 기록을 남기고 싶은 분

## ✨ 핵심 기능
- **식단 기록**: 사진 → 칼로리/영양소 분석
- **건강 체크**: "오늘 컨디션 어때요?" 매일 체크
- **운동 추천**: 시니어 맞춤 스트레칭/걷기
- **약 복용 알림**: "혈압약 드실 시간이에요"

## 🛠️ 기술 스택
- 음식 이미지 인식 (Claude Vision)
- 건강 데이터 기록 (Google Sheets)
- 크론 알림 시스템

## 💰 수익 모델
- 프리미엄 구독 (월 9,900원)
- 건강 리포트 PDF
