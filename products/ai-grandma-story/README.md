# 🧓 AI 할머니의 옛날이야기

> "따뜻한 AI 할머니가 들려주는 옛날이야기"

## 📌 한 줄 소개
AI TTS로 전래동화와 추억 이야기를 들려주는 오디오/영상 콘텐츠 서비스

## 🎯 타겟 사용자
- 시니어 (추억 회상)
- 손주에게 들려줄 이야기가 필요한 조부모
- 잠자리 동화가 필요한 가정

## ✨ 핵심 기능
- AI TTS 기반 오디오북 생성
- 전래동화/추억이야기 라이브러리
- 유튜브 숏폼/롱폼 콘텐츠

## 🛠️ 기술 스택
- **TTS**: Qwen3-TTS (Sohee 목소리)
- **이미지**: ComfyUI / AI 이미지 생성
- **영상**: FFmpeg
- **배포**: YouTube
