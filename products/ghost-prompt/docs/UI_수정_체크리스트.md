## 🔧 수정이 필요한 항목

### 1. 캡처 방지 버튼 색상
**문제:** 현재 회색 (OFF 상태처럼 보임)
**목업:** 초록색 (ON 상태)
**수정:**
```python
# 현재:
bg=COLORS['btn_gray']  # 회색

# 수정:
bg=COLORS['green']  # 초록색
```

### 2. 사이드바 보이지 않음
**문제:** 사이드바 프레임이 화면에 표시되지 않음
**원인:** pack() 순서 또는 geometry 문제
**수정:** 사이드바 pack 순서 확인 및 조정

### 3. 하단 힌트 배경색
**문제:** 현재 #252838, 목업 #0f0f1a
**수정:** COLORS['bg_bottom'] 값 변경
