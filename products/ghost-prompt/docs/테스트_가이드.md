## 🚀 빠른 시작 (3단계)

### Step 1: 실행
```bash
# 방법 1: 직접 실행
cd C:\NewProject\2026-02\Senior_AI_class\products\ghost-prompt
python ghost_prompt.py

# 방법 2: 전역 Python으로 실행  
"C:\Python314\python.exe" ghost_prompt.py

# 방법 3: 더블클릭 (파이썬이 설치되어 있다면)
# ghost_prompt.py 파일을 더블클릭
```

### Step 2: 스크립트 로드
1. **파일 선택** 버튼 클릭
2. `.txt` 파일 선택 (프롬프트 대본)
3. **로드** 버튼 클릭

### Step 3: 테스트
1. **캡처 방지 ON** 버튼 클릭
2. 화면 녹화 프로그램 실행 (OBS, Zoom 등)
3. 캡처되는지 확인!

## 🧪 테스트 체크리스트

### ✅ 기본 기능 테스트

| 테스트 | 방법 | 통과 기준 |
|--------|------|----------|
| **파일 로드** | `.txt` 파일 선택 | 텍스트가 프롬프터에 표시 |
| **스크롤** | ▶️ 버튼 클릭 | 텍스트가 위로 올라감 |
| **속도 조절** | 슬라이더 이동 | 스크롤 속도 변경 |
| **투명도** | 슬라이더 이동 | 창이 투명해짐 |
| **글자 크기** | +/- 버튼 | 글자 크기 변경 |
| **캡처 방지** | 캡처 방지 ON + OBS 실행 | 녹화에 안 보임 |

### ✅ 캡처 방지 테스트 (핵심!)

```
테스트 방법:

1. 고스트프롬프트 실행
   └─ 캡처 방지 ON 클릭

2. 녹화 프로그램 실행
   ├─ OBS Studio (무료)
   ├─ Zoom (화면 공유)
   ├─ Windows 캡처 (Win+Alt+R)
   └─ Snipping Tool (캡처 도구)

3. 녹화/캡처 시작

4. 결과 확인
   ├─ ✅ 성공: 녹화에서 프롬프터 안 보임
   └─ ❌ 실패: 녹화에서 프롬프터 보임
```

## 🎯 샘플 테스트 데이터

### test_script.txt 생성

```
=== 테스트 대본 ===

안녕하세요.
이것은 고스트프롬프트 테스트입니다.

이 창은 녹화에 잡히지 않아야 합니다.
OBS로 녹화하면 검은 화면으로 보입니다.

테스트가 성공하면,
정말로 마법 같은 경험입니다!

===================
```

### Python으로 샘플 생성
```python
# create_test_script.py
sample_text = """=== 고스트프롬프트 테스트 ===

1단계: 파일 로드
이 텍스트가 화면에 표시됩니다.

2단계: 스크롤 테스트
텍스트가 위로 올라갑니다.
속도를 조절해보세요.

3단계: 캡처 방지
캡처 방지 버튼을 ON으로 설정하세요.
그리고 OBS를 실행하세요.

4단계: 확인
녹화 화면에서 이 창이 보이지 않아야 합니다.
검은 화면으로 나타납니다.

테스트 성공!
"""

with open('test_script.txt', 'w', encoding='utf-8') as f:
    f.write(sample_text)

print("✅ test_script.txt 생성 완료")
```

## 🎬 녹화 프로그램별 테스트

### 1. OBS Studio (무료)

```
설치: https://obsproject.com/download

테스트 단계:
1. OBS 실행
2. [+] 버튼 클릭 → [디스플레이 캡처] 추가
3. 녹화 시작 (Ctrl+F9)
4. 고스트프롬프트 확인
5. 녹화 중지 (Ctrl+F10)
6. 녹화본 재생 → 프롬프터 안 보이면 성공!
```

### 2. Windows Xbox Game Bar

```
단축키: Win + Alt + R (녹화 시작/중지)

테스트 단계:
1. 고스트프롬프트 실행
2. Win + Alt + R (녹화 시작)
3. 5초 대기
4. Win + Alt + R (녹화 중지)
5. 알림 클릭 → 녹화본 확인
```

### 3. Zoom 화면 공유

```
테스트 단계:
1. Zoom 회의 시작
2. [화면 공유] 클릭
3. 전체 화면 선택
4. 고스트프롬프트 확인
5. 참가자 화면에서 안 보이면 성공!
```

### 4. Snipping Tool (캡처 도구)

```
단축키: Win + Shift + S

테스트 단계:
1. 고스트프롬프트 실행
2. Win + Shift + S
3. 전체 화면 영역 선택
4. 클립보드 확인
5. 프롬프터가 검은색이면 성공!
```

## 🔍 문제 해결

### ❌ "캡처 방지가 작동하지 않아요"

**원인 1: Windows 버전**
```
확인 방법:
Win + R → winver 실행

필요 버전: Windows 10 버전 2004 (빌드 19041) 이상

해결:
Windows 업데이트 실행
설정 → Windows 업데이트 → 업데이트 확인
```

**원인 2: 관리자 권한**
```
해결 방법:
1. PowerShell/터미널을 관리자 권한으로 실행
2. python ghost_prompt.py 실행

또는:
1. ghost_prompt.py 우클릭
2. "관리자 권한으로 실행" (python.exe에 연결된 경우)
```

**원인 3: DWM 문제**
```
해결 방법:
1. 창을 최소화했다가 다시 최대화
2. Alt+Tab으로 다른 창 갔다가 돌아오기
3. 캡처 방지 OFF → ON 다시 클릭
```

### ❌ "Python이 설치되어 있지 않아요"

```bash
# 확인
python --version

# 미설치 시 설치
# https://python.org/downloads (3.11 이상 권장)

# 설치 확인
python --version
# Python 3.11.x 출력 확인
```

### ❌ "tkinter 오류 발생"

```bash
# Python 재설치 (tkinter 포함)
# 설치 시 "tcl/tk and IDLE" 체크 확인

# 또는
python -m tkinter
# 창이 뜨면 정상
```
