# 🔮 고스트프롬프트 (Ghost Prompt) v1.0

> 화면에는 보이지만, 녹화에는 안 잡히는 마법의 프롬프터

## ✨ 한 줄 소개

유튜브/강의 촬영 시 대본을 보면서 녹화할 수 있는 "투명 프롬프터" 앱

## 핵심 가치

- **내 화면**: 대본이 보임
- **녹화 영상**: 대본이 안 보임
- **결과**: 자연스러운 눈 맞춤 + 대본 참고 가능

## 🎯 핵심 기능

### 1. 캡처 방지 (Ghost Mode)
- Windows 10/11 전용 기술
- OBS, Zoom, Discord 등 모든 녹화에서 숨김
- 화면에는 정상 표시, 녹화에는 검은 화면

### 2. 자동 스크롤
- 재생/일시정지 버튼
- 속도 조절 (느리게 ↔ 빠르게)
- 단축키 지원 (F8)

### 3. 접기/펴기 사이드바
- ◀ 버튼으로 사이드바 접기
- ▶ 버튼으로 사이드바 펴기
- 깔끔한 UI 전환

### 4. 키보드 단축키
| 단축키 | 기능 |
|--------|------|
| F8 | 재생/정지 |
| F9 | 창 표시/숨김 |
| Ctrl+↑↓ | 스크롤 속도 |
| PgUp/Dn | 수동 스크롤 |

## 🔬 기술 원리

Windows API `SetWindowDisplayAffinity(WDA_EXCLUDEFROMCAPTURE)` 사용
- Windows 10 버전 2004 (빌드 19041) 이상 필요
- DWM(Desktop Window Manager) 레벨에서 캡처 제외
- 하드웨어 가속 지원

## 🚀 빠른 시작

### 방법 1: 더블클릭 실행 (추천)
```
run.bat 더블클릭
```

### 방법 2: 명령어 실행
```bash
cd products/ghost-prompt
python ghost_prompt.py
```

### 방법 3: Python 직접 지정
```bash
"C:\Python314\python.exe" ghost_prompt.py
```

## 📋 사용 방법

1. **파일 열기** (📁 버튼 또는 Ctrl+O)
   - `.txt` 파일 선택
   - UTF-8 인코딩 권장

2. **캡처 방지 ON** (🔒 버튼 클릭)
   - 버튼이 초록색이면 활성화됨

3. **스크롤 시작** (▶ 버튼 또는 F8)
   - 텍스트가 위로 올라감
   - ⚡ 버튼으로 속도 조절

4. **사이드바 접기** (◀ 버튼)
   - 깔끔한 화면을 위해 접기
   - ▶ 버튼으로 다시 펼치기

## ⚠️ 주의사항

1. **Windows 10/11 전용**: Windows 10 버전 2004 (빌드 19041) 이상 필요
2. **Python 3.11+ 필요**: tkinter 포함된 Python
3. **관리자 권한**: 일부 환경에서 관리자 권한으로 실행 필요

## 🐛 문제 해결

| 문제 | 해결 방법 |
|------|----------|
| "캡처 방지 ON"이 안 됨 | Windows 업데이트 확인 |
| Python 오류 | Python 3.11+ 설치 확인 |
| tkinter 오류 | Python 재설치 (tkinter 포함) |
| 녹화에 보임 | Windows 10 2004 이상인지 확인 |
