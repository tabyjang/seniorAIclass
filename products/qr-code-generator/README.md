# QR 코드 생성기

**버전**: 1.0  
**제작일**: 2026-02-03  
**용도**: 시니어AI클래스 음성 아카이브 프로젝트용 QR 코드 생성

---

## 🎯 이 생성기로 뭘 할 수 있나요?

**가족 음성 아카이브 프로젝트**에서 사용하는 QR 코드를 쉽게 만들 수 있습니다.

### 사용 예시
1. 할머니가 동화를 읽어서 녹음 (스마트폰)
2. 구글 드라이브에 업로드 후 공유 링크 복사
3. 이 생성기에 링크 입력 → QR 코드 생성
4. QR 코드를 카드나 편지에 넣어 손주에게 선물
5. 손주가 QR 코드를 찍으면 할머니 목소리로 동화 재생!

---

## 📖 사용 방법

### 1. HTML 파일 열기
- 파일 탐색기에서 `index.html` 더블클릭
- 또는 브라우저로 드래그하여 열기

### 2. QR 코드 생성
1. 입력창에 웹사이트 주소(URL) 입력
   - 예: `https://drive.google.com/file/d/...`
2. "QR 코드 만들기" 버튼 클릭
3. 생성된 QR 코드 확인
4. "QR 코드 저장하기" 버튼으로 PNG 파일 저장

### 3. 활용하기
- 저장한 QR 코드 이미지를 프린트
- 카드, 편지, 동화책에 붙이기
- 손주에게 선물!

## ⚙️ 기술 정보

### 사용 라이브러리
- **QRCode.js**: MIT 라이선스, 무료 사용 가능
- CDN: `https://cdnjs.cloudflare.com/ajax/libs/qrcodejs/1.0.0/qrcode.min.js`

### 디자인 특징
- **시니어 친화적**: 글자 크기 18px 이상
- **고대비**: Midnight(#020617) + Gold(#D4A52C) 컬러
- **직관적 UI**: 큰 버튼, 명확한 안내 문구

### QR 코드 규격
- 크기: 256 x 256 픽셀
- 오류정정레벨: H (최고 수준, 30%까지 복원 가능)
- 출력 형식: PNG 이미지

## 💡 활용 팁

### 음성 파일 공유하기
1. **구글 드라이브**
   - 파일 업로드 → 우클릭 → "링크 복사"
   - 접근 권한: "링크가 있는 모든 사용자"

2. **네이버 MYBOX**
   - 파일 업로드 → 공유 → 링크 복사

3. **카카오톡**
   - "나와의 채팅"에 파일 전송
   - PC버전에서 링크 복사

### QR 코드 테스트
- 생성 후 반드시 스마트폰으로 테스트!
- 카카오톡, 카메라 앱 등으로 스캔해보기

## 🎓 강사용 가이드

### 수업 중 활용법
1. **이론 설명** (10분): QR 코드가 뭔지, 어디에 쓰는지
2. **실습** (20분): 학생들이 직접 URL 입력하고 생성
3. **적용** (10분): 생성한 QR 코드를 동화책/편지에 넣기

### 주의사항
- URL 입력 시 `https://` 자동 추가됨
- 너무 긴 URL은 QR 코드가 복잡해짐 (단축URL 서비스 권장)
- QR 코드 저장은 PNG 형식
