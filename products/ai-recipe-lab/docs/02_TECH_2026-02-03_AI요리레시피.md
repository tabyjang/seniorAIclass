# AI 요리 레시피 실험실 기술 설계서

**작성일**: 2026-02-03 | **버전**: v1.0

---

## 1. 아키텍처

```
[재료 입력] → [AI 레시피 생성] → [요리 촬영] → [영상 편집] → [업로드]
```

## 2. 기술 스택

| 분류 | 기술 | 용도 |
|------|------|------|
| AI | Claude API | 레시피 추천 |
| ASR | Whisper | 자막 생성 |
| 영상 | FFmpeg | 편집/자막 |
| TTS | Qwen3-TTS | 나레이션 (선택) |

## 3. AI 레시피 프롬프트

```
당신은 시니어 친화적인 요리 전문가입니다.

재료: {재료_목록}
조건: 조리시간 30분 이내, 간단한 조리법

다음 형식으로 레시피를 제안해주세요:
1. 요리명
2. 예상 조리시간
3. 재료 (분량 포함)
4. 조리 순서 (5단계 이내)
5. 건강 팁
```

## 4. 영상 제작 파이프라인

```bash
# 자막 생성
python asr_subtitle.py "cooking_raw.mp4"

# 자막 하드코딩
ffmpeg -i cooking_raw.mp4 -vf "subtitles=cooking.srt" output.mp4
```
