# 🍳 AI 요리 레시피 실험실

> "AI가 추천하고, 인간이 만들고, 함께 평가하는 요리 채널"

## 📌 한 줄 소개
AI가 재료 조합과 레시피를 추천하고, 실제로 만들어서 평가하는 실험형 요리 콘텐츠

## 🎯 타겟 사용자
- 요리 아이디어가 필요한 시니어
- 건강식 레시피를 찾는 분들
- 추억의 맛을 재현하고 싶은 분들

## ✨ 핵심 기능
- AI 재료 조합 추천
- 영양소 분석 레시피
- 추억의 맛 재현 시리즈

## 🛠️ 기술 스택
- **AI**: Claude/GPT (레시피 추천)
- **TTS**: Qwen3-TTS (나레이션)
- **영상**: 실제 촬영 + AI 자막
