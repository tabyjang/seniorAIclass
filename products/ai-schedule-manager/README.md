# 🗓️ AI 일정 관리자

> "말로 하면 다 알아서 등록해드려요"

## 📌 한 줄 소개
자연어로 일정 등록, 아침 브리핑, 알림을 AI가 관리

## 🎯 타겟 사용자
- 일정 관리가 어려운 시니어
- 약속을 자주 잊는 분
- 캘린더 앱이 복잡한 분

## ✨ 핵심 기능
- **자연어 등록**: "내일 3시 치과" → 자동 등록
- **아침 브리핑**: "오늘 일정 알려드릴게요"
- **알림**: 약속 30분 전 알림
- **반복 일정**: "매주 월요일 병원"

## 🛠️ 기술 스택
- Google Calendar API (이미 연동!)
- 자연어 처리 (날짜/시간 파싱)
- Moltbot 크론 알림

## 💰 수익 모델
- 프리미엄 기능 (가족 공유)
- 앱 구독
