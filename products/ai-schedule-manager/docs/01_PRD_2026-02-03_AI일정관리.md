## 2. 핵심 기능

### 자연어 일정 등록
```
사용자: "내일 오후 3시에 치과 예약"

AI: ✅ 일정 등록 완료!

📅 2026년 2월 4일 (화)
⏰ 오후 3:00
📍 치과 예약

30분 전에 알려드릴까요? [예/아니오]
```

### 아침 브리핑
```
🌅 좋은 아침이에요!

📅 오늘 일정 (2026.02.03)

09:00 - 혈압약 복용 💊
11:00 - 김철수님 전화 📞
15:00 - 치과 예약 🦷
18:00 - 손주 생일파티 🎂

오늘 하루도 화이팅! 💪
```

### 알림
```
⏰ 30분 후 일정 알림!

🦷 치과 예약
📍 서울치과의원
⏰ 오후 3:00

준비하세요! 🚗
```

### 반복 일정
```
사용자: "매주 월요일 아침 9시에 병원"

AI: ✅ 반복 일정 등록!

📅 매주 월요일
⏰ 오전 9:00
📍 병원

다음 일정: 2026년 2월 10일 (월)
```

## 9. 바로 시작 가능!

```python
# 이미 연동된 Google Calendar API!
from googleapiclient.discovery import build

service = build('calendar', 'v3', credentials=creds)
calendar_id = 'tabyjang0101@gmail.com'

# 일정 추가
event = {
    'summary': '치과 예약',
    'start': {'dateTime': '2026-02-04T15:00:00+09:00'},
    'end': {'dateTime': '2026-02-04T16:00:00+09:00'},
}
service.events().insert(calendarId=calendar_id, body=event).execute()
```
