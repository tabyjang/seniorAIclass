# 🎬 AI 영상 편집 도우미

> "긴 영상도 AI가 알아서 편집해드려요"

## 📌 한 줄 소개
긴 영상 하이라이트 추출, 자막 생성, 숏폼 자동 변환

## 🎯 타겟 사용자
- 영상 편집이 어려운 시니어
- 유튜브 크리에이터
- 가족 영상 정리하고 싶은 분

## ✨ 핵심 기능
- **하이라이트 추출**: 1시간 → 3분 핵심
- **자막 자동 생성**: Whisper + KOTRA 폰트
- **숏폼 자동 생성**: 세로 영상 변환
- **BGM 추천**: 분위기에 맞는 음악

## 🛠️ 기술 스택
- Whisper (자막 생성)
- FFmpeg (영상 편집)
- AI 장면 분석

## 💰 수익 모델
- 영상당 과금
- 월 구독 (무제한)
