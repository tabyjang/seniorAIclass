## 1. 제품 개요

### 제품명
AI 영상 편집 도우미

### 한 줄 소개
긴 영상 → AI가 알아서 하이라이트 + 자막 + 숏폼

---

## 2. 핵심 기능

### 하이라이트 자동 추출
```
입력: 1시간 강의 영상

AI 분석:
- 중요 포인트 감지
- 감정 피크 구간 감지
- 핵심 키워드 구간 추출

출력: 3분 하이라이트 영상
```

### 자막 자동 생성
```
1. 영상 업로드
2. Whisper ASR → 텍스트 변환
3. KOTRA Bold 폰트 적용
4. 영상에 하드코딩

설정:
- 폰트 크기: 48pt (시니어용 크게!)
- 색상: 흰색 + 검정 테두리
- 위치: 하단 중앙
```

### 숏폼 자동 생성
```
가로 영상 (16:9)
    ↓
AI 주요 인물/객체 감지
    ↓
자동 크롭 (9:16)
    ↓
세로 숏폼 완성!
```

## 4. 워크플로우

```
[영상 업로드]
     ↓
[1. 오디오 추출] ──→ [Whisper 자막]
     ↓
[2. 장면 분석] ──→ [하이라이트 구간]
     ↓
[3. 영상 편집]
  ├─ 자막 하드코딩
  ├─ 하이라이트 추출
  └─ 숏폼 변환
     ↓
[4. 결과물 저장]
```
