# AI 여행 가이드 PRD

**작성일**: 2026-02-03 | **버전**: v1.0

---

## 1. 제품 개요

### 제품명
AI 여행 가이드

### 한 줄 소개
집에서 떠나는 AI와 함께하는 가상 여행 콘텐츠

---

## 2. 콘텐츠 시리즈

### 국내 여행
```
🇰🇷 국내 여행 시리즈
├── 경주 1일 코스 (역사 탐방)
├── 제주 숨은 명소
├── 부산 해운대 완벽 가이드
├── 전주 한옥마을
└── 강릉 커피거리
```

### 해외 여행
```
🌍 해외 여행 시리즈
├── 파리 미술관 투어
├── 도쿄 맛집 탐방
├── 방콕 사원 순례
└── 하와이 힐링 여행
```

### 추억 여행 (차별화!)
```
📷 추억 여행 시리즈
├── 1970년대 부산 여행
├── 1980년대 서울 명동
├── 1990년대 해외여행 붐
└── 옛날 사진 vs 지금 비교
```

---

## 3. 에피소드 포맷

```
[인트로] 15초
"오늘은 AI와 함께 경주로 떠나볼까요?"

[역사/배경] 1분
해당 지역의 역사와 특징

[명소 소개] 3-5분
주요 명소 5곳 소개 (AI 이미지 + 실제 사진)

[맛집/꿀팁] 1분
현지인 추천 맛집, 여행 팁

[클로징] 15초
"다음엔 어디로 떠날까요? 댓글로 알려주세요!"
```

---

## 4. 기술 스택

| 분류 | 기술 |
|------|------|
| 이미지 | AI 생성 + 무료 사진 (Unsplash) |
| 정보 | 웹 검색 + AI 정리 |
| TTS | Qwen3-TTS |
| 영상 | FFmpeg 슬라이드쇼 |
