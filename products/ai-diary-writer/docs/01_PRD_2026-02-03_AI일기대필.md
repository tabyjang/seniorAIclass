# AI 일기 대필 PRD

**작성일**: 2026-02-03 | **버전**: v1.0

---

## 1. 제품 개요

### 제품명
AI 일기 대필

### 한 줄 소개
하루를 말해주면, AI가 예쁘게 기록해드립니다

---

## 2. 핵심 기능

### 입력 방식
1. **음성 입력**: "오늘 손주가 왔어..."
2. **텍스트 입력**: 카카오톡처럼 간단히
3. **사진 첨부**: 오늘의 사진과 함께

### 출력 형식
```
┌─────────────────────────────┐
│ 2026년 2월 3일 (월) ☀️      │
│                             │
│ 오늘은 특별한 날이었다.      │
│ 사랑하는 손주가 찾아왔다.    │
│                             │
│ 함께 공원을 거닐며,          │
│ 손주의 첫 자전거 도전을      │
│ 지켜보았다.                  │
│                             │
│ 넘어져도 울지 않는 모습에    │
│ 대견함이 밀려왔다.           │
│                             │
│ 🌟 오늘의 한마디:            │
│ "넘어져도 다시 일어나는 것,  │
│  그것이 성장이다"            │
└─────────────────────────────┘
```

---

## 3. 서비스 형태

### Phase 1: 유튜브 콘텐츠
- AI 일기 대필 소개 영상
- 시연 영상 (사용자 사연)
- "오늘 하루를 일기로" 챌린지

### Phase 2: 웹/앱 서비스
- 카카오톡 봇 또는 웹앱
- 매일 알림 → 일기 작성
- 월간 일기장 PDF 생성

### Phase 3: 프리미엄
- 가족 공유 기능
- 일기 인쇄본 배송
- 1년 일기 책자 제작

## 4. 기술 스택

| 분류 | 기술 |
|------|------|
| ASR | Whisper (음성→텍스트) |
| AI | Claude (일기 변환) |
| 출력 | PDF/이미지 생성 |

### AI 프롬프트
```
사용자의 하루 이야기를 아름다운 일기로 변환해주세요.

규칙:
1. 존댓말 사용
2. 감성적이고 따뜻한 톤
3. 핵심 감정 포착
4. 오늘의 명언 추가
5. 150자 내외
```
