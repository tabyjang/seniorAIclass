## 1. 개요

### 1.1 프로젝트명
**베네치아 게임** (Venezia Typing Game)

### 1.2 한 줄 설명
> "하늘에서 떨어지는 단어를 타자로 파괴하는 추억의 게임"

### 1.3 만드는 이유 (Why?)

**문제점:**
- 기존 문장 연습은 반복적이고 지루함
- 시니어들도 게임을 통해 더 즐겁게 연습하고 싶음
- 한메타자교사의 베네치아 게임에 대한 향수

**해결책:**
- 게임 형식으로 타자 연습
- 성취감과 재미 동시에 제공
- 친숙한 게임 규칙 (한메타자 베네치아)
- 쉬운 난이도부터 시작

### 1.4 영감 (Reference)
- **한메타자교사 베네치아** (1989~2003)
- **ztype** (phoboslab.org/ztype)
- **한메타자 베네치아 영어** (모바일 앱)

2. "게임하면서 연습하고 싶어요"
3. "손주랑 같이 할 수 있는 게임 없을까요"
4. "예전에 하던 베네치아 게임 기억나요"

## 6. 게임 규칙 (Game Rules)

### 6.1 단계별 진행 규칙 (핵심!)

**🎯 특징:** 점점 어려워지지만, 이전 단계 복습은 동일 비율로 유지

1. **레벨 1 (0~499점): 한글자 익히기**
   - 1글자 단어만 출현 (100%)
   - 속도: 매우 느림
   - 목표: 기본 타자 익히기

2. **레벨 2 (500~1499점): 두글자 도전**
   - 1글자 50% + 2글자 50%로 섞여서 출현
   - 속도: 느림 (레벨 1보다 약간 빠름)
   - 목표: 단어 단위로 타자 연습

3. **레벨 3+ (1500점 이상): 세글자 마스터**
   - 1글자 33% + 2글자 33% + 3글자 33% (동일 비율)
   - 속도: 보통 → 빠름 (점진적 가속)
   - 목표: 실전 타자 능력 향상

### 6.2 기본 게임 규칙

1. **단어 생성:** 랜덤한 X위치에서 단어가 생성됨
2. **단어 하강:** 단어는 위에서 아래로 떨어짐
3. **타자 입력:** 화면의 단어와 똑같이 입력하면 파괴 (Enter 키)
4. **점수 획득:** 단어 길이 × 100점 (1글자=100점, 2글자=200점, 3글자=300점)
5. **바닥 충돌:** 단어가 바닥에 닿으면 생명 -1
6. **게임오버:** 생명(♥) 0이면 게임 종료

### 6.3 점수 시스템

| 단어 길이 | 점수 | 예시 단어 |
|----------|------|----------|
| 1글자 | 100점 | 가, 각, 사 |
| 2글자 | 200점 | 사과, 한국, 바다 |
| 3글자 | 300점 | 바나나, 전화기, 학생들 |

### 6.4 속도 진행 (점진적 가속)

| 레벨 | 시작 속도 | 최대 속도 | 가속 방식 |
|------|----------|----------|----------|
| Level 1 | 1.0 px/frame | 1.5 px/frame | 점수 증가에 따라 서서히 |
| Level 2 | 1.5 px/frame | 2.5 px/frame | 500점마다 +0.1 |
| Level 3+ | 2.0 px/frame | 설정 최대값 | 1000점마다 +0.2 |

**사용자 조절:** 설정 패널에서 0.5~8 px/frame 범위로 자유롭게 조절 가능

### 8.2 핵심 알고리즘

```javascript
// 게임 루프
function gameLoop() {
    // 1. 모든 단어 이동
    fallingWords.forEach(word => {
        word.y += word.speed;
        word.element.style.top = word.y + 'px';
    });
    
    // 2. 바닥 체크
    checkBottomCollision();
    
    // 3. 새 단어 생성
    if (shouldSpawnWord()) spawnWord();
    
    requestAnimationFrame(gameLoop);
}

// 입력 처리
function handleInput(input) {
    const index = fallingWords.findIndex(w => w.text === input);
    if (index !== -1) {
        destroyWord(index);
        addScore(input.length * 100);
    }
}
```
