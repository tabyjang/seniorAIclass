## 1. 베네치아 게임이란?

### 1.1 게임 개요

베네치아(Venezia)는 **한메타자교사(1989~2003)**의 대표적인 미니게임으로, 타자 연습을 게임 형태로 만든 것입니다.

**스토리:** 외계 바이러스가 베네치아를 공격! 타자로 바이러스를 파괴하여 도시를 지켜라!

### 1.2 게임 화면 구성

```
┌─────────────────────────────────────┐
│  점수: 1500    생명: ♥♥♥           │  ← 상단 정보
│                                     │
│     안녕                            │  ← 하늘에서 떨어지는
│        감사                         │     단어들 (바이러스)
│   사랑       행복                   │
│                                     │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━    │  ← 바닥 (닿으면 게임오버)
│           🏛️ 베네치아 성             │
└─────────────────────────────────────┘
```

## 2. 게임 규칙 및 메커니즘

### 2.1 기본 규칙

| 항목 | 설명 |
|------|------|
| **목표** | 떨어지는 단어를 타자로 입력하여 파괴 |
| **패배 조건** | 단어가 바닥(바다)에 닿으면 생명 감소, 생명 0이면 게임오버 |
| **입력 방식** | 단어를 보고 타이핑 후 Enter 또는 자동 인식 |
| **점수** | 단어 파괴 시 점수 획득 (단어 길이에 비례) |

### 2.2 단어 생성 및 이동

```javascript
// 게임 메커니즘 요약
1. 일정 시간마다 랜덤 위치에서 단어 생성
2. 단어는 위에서 아래로 떨어짐 (Y좌표 증가)
3. 플레이어가 단어를 입력하면 해당 단어 파괴
4. 단어가 바닥에 닿으면 생명 감소
```

**세부 설정:**
- 생성 간격: 1~3초 (난이도에 따라 변동)
- 떨어지는 속도: 초당 30~100픽셀
- 단어 길이: 2~8글자 (난이도별)

## 3. 아이템 시스템

### 3.1 버프 아이템 (도움)

| 아이템명 | 효과 |
|----------|------|
| **깨끗이** | 화면의 모든 바이러스 제거 |
| **느리게** | 바이러스 하강 속도 감소 (일시적) |
| **조금** | 새로 생성되는 바이러스 수 감소 |
| **두곱** | 한 번 입력 시 2개 파괴 |
| **세곱** | 한 번 입력 시 3개 파괴 |
| **잠깐만** | 모든 바이러스 일시 정지 |
| **다막아** | 방어막 생성 (일정 시간 무적) |
| **퍼지게** | 파괴 시 주변 바이러스도 함께 파괴 |
| **짧게** | 모든 단어 길이 감소 |
| **새삶** | 생명 +1 |

### 3.2 디버프 아이템 (방해)

| 아이템명 | 효과 |
|----------|------|
| **무더기** | 소용돌이 치는 바이러스 무더기 등장 |
| **빠르게** | 하강 속도 증가 |
| **많이** | 생성 빈도 증가 |
| **어둡게** | 단어가 어두워져 잘 안 보임 |
| **돌려** | 단어 글자 순서 뒤섞임 |
| **셈** | 산수 문제 바이러스 등장 |
| **헛소리** | 랜덤 문자 바이러스 등장 |
| **덧말** | 단어 맨 뒤 글자 반복 (길어짐) |

## 4. 기술적 구현 분석

### 4.1 구현 가능성: ✅ 매우 높음

HTML5 + JavaScript로 완벽하게 구현 가능합니다.

### 4.2 필요한 기술 스택

| 구분 | 기술 | 용도 |
|------|------|------|
| **마크업** | HTML5 | 게임 화면 구조 |
| **스타일** | CSS3 | 애니메이션, 위치 조정 |
| **로직** | JavaScript (ES6+) | 게임 엔진 |
| **그래픽** | Canvas API 또는 DOM | 단어 렌더링, 애니메이션 |

### 4.3 핵심 알고리즘

```javascript
// 1. 단어 클래스
class Word {
    constructor(text, x, y, speed) {
        this.text = text;
        this.x = x;        // 가로 위치
        this.y = y;        // 세로 위치 (떨어지는)
        this.speed = speed; // 하강 속도
        this.element = null; // DOM 요소
    }
    
    update() {
        this.y += this.speed; // 아래로 이동
        this.element.style.top = this.y + 'px';
    }
}

// 2. 게임 루프
function gameLoop() {
    // 모든 단어 업데이트 (떨어트리기)
    fallingWords.forEach(word => word.update());
    
    // 바닥 체크
    checkBottomCollision();
    
    // 새 단어 생성 (일정 확률로)
    if (shouldSpawnWord()) {
        spawnWord();
    }
    
    requestAnimationFrame(gameLoop);
}

// 3. 입력 처리
function handleInput(input) {
    // 화면에 있는 단어 중 일치하는 것 찾기
    const matchedIndex = fallingWords.findIndex(
        word => word.text === input
    );
    
    if (matchedIndex !== -1) {
        // 파괴 처리
        destroyWord(matchedIndex);
        addScore(input.length * 100);
    }
}
```

### 7.2 참고 코드 예시

```javascript
// fallingWords 배열 관리
let fallingWords = [];

function spawnWord() {
    const word = new Word(
        getRandomWord(),           // 랜덤 단어
        randomX(),                 // 랜덤 X 위치
        0,                         // 시작 Y (상단)
        getSpeedByLevel()          // 난이도별 속도
    );
    
    // DOM 생성
    word.element = document.createElement('div');
    word.element.textContent = word.text;
    word.element.style.left = word.x + 'px';
    gameArea.appendChild(word.element);
    
    fallingWords.push(word);
}

function gameLoop() {
    // 1. 모든 단어 이동
    fallingWords.forEach(word => {
        word.y += word.speed;
        word.element.style.top = word.y + 'px';
    });
    
    // 2. 바닥 체크
    fallingWords = fallingWords.filter(word => {
        if (word.y > GAME_HEIGHT) {
            loseLife();
            word.element.remove();
            return false;
        }
        return true;
    });
    
    requestAnimationFrame(gameLoop);
}
```
