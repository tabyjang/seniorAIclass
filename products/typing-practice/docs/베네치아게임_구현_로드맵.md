# 베네치아 게임 구현 로드맵

> **프로젝트:** 시니어 타자 연습 - 베네치아 게임 모드
> **버전:** 1.0
> **작성일:** 2026-02-02
> **예상 기간:** 2일 (8~10시간)

---

## 🎯 전체 개요

### 게임 설명
한메타자교사의 전설적인 베네치아 게임을 웹으로 구현합니다.
하늘에서 떨어지는 단어(바이러스)를 타자로 입력하여 파괴하는 게임입니다.

### 목표
- 시니어들도 쉽게 즐길 수 있는 타자 게임
- 큰 글씨, 단순 조작
- 기존 문장 연습과 자연스럽게 연결

## 📋 상세 단계별 계획

### 🟢 Phase 1: 준비 (1시간) ⏳ 예정

| # | 태스크 | 예상 시간 | 산출물 |
|---|--------|----------|--------|
| 1 | 게임 화면 와이어프레임 그리기 | 15분 | 와이어프레임 스케치 |
| 2 | venezia.html 파일 생성 | 10분 | venezia.html (기본 구조) |
| 3 | CSS 기본 스타일 설정 | 20분 | 게임 화면 CSS |
| 4 | 단어 데이터 준비 | 15분 | 단어 배열 (40개+) |

**단어 데이터 예시:**
```javascript
const words = {
    easy: ['안녕', '감사', '사랑', '행복', '좋은'],
    medium: ['좋은하루', '건강하세요', '맛있게드세요'],
    hard: ['산토끼', '동해물과', '천리길도']
};
```

### 🟡 Phase 2: 핵심 기능 구현 (6시간) ⏳ 예정

#### 2-1. 게임 엔진 기본 (2시간)

| # | 태스크 | 예상 시간 | 설명 |
|---|--------|----------|------|
| 1 | 게임 영역 설정 | 30분 | 화면 크기, 바닥선 설정 |
| 2 | 단어 클래스 생성 | 30분 | Word 객체 (텍스트, 위치, 속도) |
| 3 | 단어 생성 함수 | 30분 | 랜덤 위치에서 단어 생성 |
| 4 | 게임 루프 구현 | 30분 | requestAnimationFrame 사용 |

**코드 구조:**
```javascript
class Word {
    constructor(text, x, speed) {
        this.text = text;
        this.x = x;
        this.y = 0;
        this.speed = speed;
    }
    update() {
        this.y += this.speed;
    }
}
```

#### 2-2. 이동 및 충돌 (1.5시간)

| # | 태스크 | 예상 시간 | 설명 |
|---|--------|----------|------|
| 5 | 단어 하강 애니메이션 | 30분 | Y좌표 증가, 화면 업데이트 |
| 6 | 바닥 충돌 체크 | 30분 | 바닥 닿으면 생명 감소 |
| 7 | 생명(하트) 시스템 | 30분 | ♥ 3개, 닿으면 -1 |

#### 2-3. 입력 및 파괴 (1.5시간)

| # | 태스크 | 예상 시간 | 설명 |
|---|--------|----------|------|
| 8 | 입력창 생성 | 20분 | 하단 입력 영역 |
| 9 | 입력 체크 로직 | 40분 | 화면 단어와 비교 |
| 10 | 단어 파괴 효과 | 30분 | 맞으면 사라지는 애니메이션 |

#### 2-4. 점수 및 UI (1시간)

| 13 | 게임오버 화면 | 20분 | 최종 점수, 다시 시작 버튼 |
