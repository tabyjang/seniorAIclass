## ✨ 소개

기존 타자 연습 프로그램은 너무 빠르고, 글씨가 작고, 어려웠습니다.
**시니어 타자 연습**은 어르신을 위해 만들어졌습니다.

- 🔤 **큰 글씨** - 눈이 편해요
- 🐢 **천천히** - 내 속도에 맞춰요
- 👏 **격려** - "잘하셨어요!"
- 🇰🇷 **한글** - 익숙한 문장으로

---

## 🎮 기능

| 기능 | 설명 |
|------|------|
| 난이도 선택 | 쉬움 / 보통 / 어려움 / 속담 |
| 실시간 확인 | 맞으면 초록, 틀리면 빨강 |
| 진행률 표시 | 10문장 목표 |
| 통계 측정 | 시간, 정확도, 타수 |
| 격려 메시지 | 완료 시 칭찬! |

---

## 🚀 사용 방법

1. `index.html` 파일을 브라우저에서 열기
2. 난이도 선택
3. **[시작]** 버튼 클릭
4. 화면의 문장을 따라 입력
5. 10문장 완료하면 결과 확인!

---

## 📁 폴더 구조

```
시니어_타자연습/
├── index.html          # 메인 프로그램
├── README.md           # 이 문서
└── docs/               # 개발 문서
    ├── PRD.md          # 요구사항
    ├── TECH_DESIGN.md  # 기술 설계
    ├── ROADMAP.md      # 로드맵
    └── CHECKLIST.md    # 체크리스트
```
