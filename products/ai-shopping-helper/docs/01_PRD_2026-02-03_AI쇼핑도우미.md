---

## 2. 핵심 기능

### 제품 추천
```
사용자: "혈압계 추천해줘"

AI: 🛒 혈압계 추천 Top 3

1. 오므론 HEM-7156
   - ⭐ 4.8 (리뷰 5,432개)
   - 💰 89,000원
   - ✅ 자동 측정, 부정맥 감지

2. 휴비딕 HBP-1800
   - ⭐ 4.6 (리뷰 3,211개)
   - 💰 45,000원
   - ✅ 가성비 최고

3. ...
```

### 가격 비교
```
🔍 "삼성 갤럭시탭 S9" 가격 비교

| 쇼핑몰 | 가격 | 배송 |
|--------|------|------|
| 네이버 | 899,000원 | 무료 |
| 쿠팡 | 879,000원 | 로켓 |
| 11번가 | 889,000원 | 무료 |

✅ 최저가: 쿠팡 879,000원
```

### 리뷰 요약
```
📝 리뷰 요약 (총 1,234개 분석)

👍 장점:
- 측정이 정확하다 (언급 89%)
- 사용이 간편하다 (언급 76%)
- 화면이 크다 (언급 45%)

👎 단점:
- 커프가 작다 (언급 12%)
- 배터리 소모 빠름 (언급 8%)

⭐ 종합: 4.7/5 (구매 추천!)
```

## 3. 타겟 사용자

| 대상 | 니즈 | 해결책 |
|------|------|--------|
| 시니어 | 온라인 쇼핑 어려움 | 음성/대화로 쇼핑 |
| 바쁜 직장인 | 비교할 시간 없음 | AI 자동 비교 |
| 꼼꼼한 소비자 | 리뷰 다 읽기 힘듦 | AI 요약 |

## 4. 시니어 특화 기능
