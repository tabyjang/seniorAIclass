# 🛒 AI 쇼핑 도우미

> "어르신도 쉽게! AI가 추천하고, 비교하고, 리뷰 요약해드려요"

## 📌 한 줄 소개
제품 추천, 가격 비교, 리뷰 요약을 AI가 한번에 해주는 쇼핑 도우미

## 🎯 타겟 사용자
- 온라인 쇼핑이 어려운 시니어
- 가격 비교할 시간 없는 분
- 리뷰 읽기 귀찮은 분

## ✨ 핵심 기능
- **제품 추천**: "혈압계 추천해줘" → 인기 제품 3개
- **가격 비교**: 네이버/쿠팡/11번가 한눈에
- **리뷰 요약**: 100개 리뷰 → 장점/단점 3줄
- **사기 방지**: 가짜 리뷰/의심 판매자 경고

## 🛠️ 기술 스택
- 네이버 쇼핑 API / 쿠팡 파트너스
- AI 리뷰 요약 (Claude)
- 가격 추적

## 💰 수익 모델
- 제휴 마케팅 (쿠팡 파트너스)
- 프리미엄 구독
