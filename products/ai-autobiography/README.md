# 📝 AI 자서전 작가

> "당신의 인생 이야기, AI가 책으로 만들어드려요"

## 📌 한 줄 소개
인터뷰 형식으로 인생 이야기를 듣고, AI가 자서전으로 정리해주는 서비스

## 🎯 타겟 사용자
- 인생을 기록하고 싶은 시니어
- 부모님께 특별한 선물을 드리고 싶은 자녀
- 가족 역사를 남기고 싶은 분

## ✨ 핵심 기능
- **AI 인터뷰**: 질문에 답하면 이야기 수집
- **자동 정리**: AI가 시간순/주제별 정리
- **책 제작**: PDF/전자책/종이책 출력
- **가족 공유**: 자녀/손주에게 선물

## 🛠️ 기술 스택
- AI 인터뷰 (Claude)
- 음성 인식 (Whisper)
- TTS 나레이션 (Qwen3-TTS)
- PDF/책 생성

## 💰 수익 모델
- 기본 인터뷰: 무료
- 전자책 생성: 19,900원
- 종이책 인쇄: 49,000원~
