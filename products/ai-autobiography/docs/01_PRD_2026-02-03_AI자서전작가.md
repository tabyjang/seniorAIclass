# AI 자서전 작가 PRD

**작성일**: 2026-02-03 | **버전**: v1.0

---

## 1. 제품 개요

### 제품명
AI 자서전 작가

### 한 줄 소개
당신의 인생 이야기를 AI가 책으로 만들어드립니다

### 왜 이 서비스가 필요한가?
- 시니어들의 소중한 인생 이야기가 기록되지 않고 사라짐
- 글쓰기는 어렵지만, 말하기는 쉬움
- 자녀/손주에게 남길 유산
- 치매 예방 효과 (회상 요법)

---

## 2. 핵심 기능

### AI 인터뷰
```
🎙️ 오늘의 인터뷰 주제: 어린 시절

AI: 어린 시절 가장 기억에 남는 일이 있으세요?

사용자: "6살 때 아버지랑 처음 서울 구경 갔던 게 
       아직도 기억나요. 기차를 처음 탔는데..."

AI: 기차를 처음 탔을 때 기분이 어땠어요?
    창밖 풍경은 어땠나요?

사용자: "너무 신기했어요. 창밖으로 논밭이 
       쭉 지나가고..."

[다음 질문] [오늘은 여기까지]
```

### 인터뷰 주제 가이드
```
📚 자서전 챕터 구성

1장. 어린 시절 (0-12세)
   - 태어난 곳, 가족
   - 어린 시절 추억
   - 학교 생활

2장. 청소년기 (13-19세)
   - 학창 시절
   - 첫사랑
   - 꿈과 희망

3장. 청년기 (20-30대)
   - 취업/직장
   - 결혼
   - 자녀 출산

4장. 중년기 (40-50대)
   - 가장으로서의 삶
   - 자녀 교육
   - 인생의 전환점

5장. 현재 (60대 이후)
   - 은퇴 후 삶
   - 손주 이야기
   - 후손에게 전하는 말
```

### 자동 정리
```
📝 AI가 정리한 내용

[어린 시절 - 첫 서울 구경]

1952년, 6살의 나는 아버지 손을 잡고 
처음으로 서울에 갔다. 

증기기관차를 타고 5시간을 달렸다. 
창밖으로 끝없이 펼쳐진 논밭이 
빠르게 지나갔다. 

서울역에 도착했을 때, 
그 크고 웅장한 건물에 
입이 떡 벌어졌다...

[수정하기] [다음 챕터]
```

### 책 생성
```
📖 자서전 완성!

"장삼순의 이야기 - 80년의 발자취"

📋 구성:
- 표지 디자인 (AI 생성)
- 목차
- 본문 5장 (총 120페이지)
- 사진 페이지 (업로드한 사진)
- 가계도
- 후손에게 전하는 말

📦 출력 옵션:
- PDF 다운로드: 무료
- 전자책 (ePub): 9,900원
- 종이책 1권: 49,000원
- 종이책 5권: 179,000원 (가족 선물용)

[미리보기] [주문하기]
```

## 3. 차별화 포인트

| 기존 자서전 서비스 | AI 자서전 작가 |
|-------------------|---------------|
| 전문 작가 필요 (수백만원) | AI가 정리 (저렴) |
| 오래 걸림 (수개월) | 빠름 (수주) |
| 글쓰기 능력 필요 | 말만 하면 됨 |
| 젊은 층 대상 | **시니어 특화** |

---

## 4. 시니어 특화 기능

- 🎙️ **음성 입력**: 말로 이야기하기
- 📱 **큰 글씨**: 시니어 친화 UI
- 🔄 **천천히 진행**: 하루 15분씩
- 👨‍👩‍👧 **가족 참여**: 자녀가 질문 추가
- 📸 **사진 추가**: 옛날 사진 스캔
- 🎧 **오디오북**: TTS로 읽어주기

---

## 5. 기술 스택

| 분류 | 기술 |
|------|------|
| AI 인터뷰 | Claude API |
| 음성 인식 | Whisper |
| 음성 합성 | Qwen3-TTS (오디오북) |
| 문서 생성 | PDF/ePub 라이브러리 |
| 사진 복원 | AI 이미지 복원 |

## 6. 워크플로우

```
[1. 시작하기]
     ↓
[2. 기본 정보 입력]
   - 이름, 생년월일
   - 출생지
     ↓
[3. AI 인터뷰] (2-4주)
   - 주제별 질문
   - 음성/텍스트 응답
   - 하루 15분씩
     ↓
[4. AI 정리]
   - 시간순 배열
   - 문체 통일
   - 챕터 구성
     ↓
[5. 검토 & 수정]
   - 본인/가족 확인
   - 내용 추가/수정
     ↓
[6. 사진 추가]
   - 옛날 사진 업로드
   - AI 복원 (선택)
     ↓
[7. 책 완성]
   - 표지 디자인
   - 최종 편집
     ↓
[8. 출력/공유]
   - PDF/전자책/종이책
   - 가족 공유
```
