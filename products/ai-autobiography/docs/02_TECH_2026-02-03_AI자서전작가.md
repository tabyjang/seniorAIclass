# AI자서전작가 기술 설계서

**작성일**: 2026-02-03 | **버전**: v1.0

---

## 1. 아키텍처

`
[음성 입력] → [Whisper ASR] → [텍스트]
                                  ↓
[AI 인터뷰] ← [Claude API] ← [대화 관리]
     ↓
[응답 저장] → [챕터별 정리] → [문체 통일]
                                  ↓
[사진 추가] → [레이아웃] → [PDF/책 생성]
                                  ↓
                            [출력/공유]
`

## 2. 기술 스택

| 분류 | 기술 |
|------|------|
| AI 대화 | Claude API |
| 음성 인식 | Whisper |
| 음성 합성 | Qwen3-TTS |
| 문서 생성 | reportlab (PDF), ebooklib (ePub) |
| 사진 복원 | AI 이미지 복원 모델 |

## 3. AI 인터뷰 프롬프트

`
당신은 따뜻하고 공감 능력이 뛰어난 자서전 인터뷰어입니다.

역할:
- 어르신의 인생 이야기를 경청합니다
- 추가 질문으로 이야기를 풍성하게 합니다
- 감정을 공감하고 격려합니다

현재 챕터: {chapter}
이전 대화: {history}

규칙:
- 한 번에 하나의 질문만
- 쉬운 말로 질문
- 충분히 시간을 드림
`
