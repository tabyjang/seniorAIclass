# 유튜브 강의 기획: 핸드폰 화면을 PC로 옮기기 (Scrcpy 활용)

**강의명**: 핸드폰 화면을 PC로 옮기는 법 (묣선으로도 가능!)  
**시리즈**: AI, 처음 시작하시나요? (Lv.1 기초)  
**재생목록**: 누구나 처음일 때가 있다

**이 영상을 보시면:**
1. ✅ 핸드폰 화면을 PC/노트북으로 볼 수 있습니다
2. ✅ 케이블 없이 묣선(WiFi)으로도 연결할 수 있습니다
3. ✅ OBS로 녹화/방송할 때 활용할 수 있습니다

### [본론 3] 프로그램 설치 3:30-6:00 (2.5분)

**단계별 설명:**

#### Step 1: Scrcpy 다운로드
```
1. 인터넷 주소창에 입력: github.com/Genymobile/scrcpy
2. "Releases" 클릭
3. "scrcpy-win64-v2.7.zip" 다운로드
```

**화면:**
- 브라우저 화면 녹화 (Scrcpy GitHub 페이지)
- 다운로드 버튼 강조 표시

#### Step 2: 압축 풀기
```
1. 다운로드 폴터에서 ZIP 파일 찾기
2. 마우스 우클릭 → "압축 풀기"
3. 원하는 위치에 폴터 생성 (예: C:\Users\사용자명\scrcpy)
```

**화면:**
- 파일 탐색기 화면
- 압축 풀기 과정 시연

#### Step 3: 핸드폰 설정 (처음 1회만)
```
1. 설정 → 휴제전화 정보 → 소프트웨어 정보
2. "빌드번호" 7번 연속 탭
3. "개발자 모드 활성화" 확인
4. 설정 → 개발자 옵션 → "USB 디버깅" 켜기
```

**화면:**
- 핸드폰 화면 녹화 (설정 메뉴 이동)
- 각 단계 화살표/강조 표시

**팁:**
> "처음 설정이 조금 번거롭지만, 한 번만 합니다! 그 뒤로는 편하게 쓸 수 있어요."

### [본론 4] USB로 연결하기 6:00-8:00 (2분)

**실습 시연:**

#### Step 1: 케이블 연결
```
1. USB 케이블로 핸드폰-PC 연결
2. 핸드폰에서 "파일 전송" 선택
3. "USB 디버깅 허용" 팝업 → "허용" 체크 후 확인
```

**화면:**
- 케이블 연결 클로즈업
- 핸드폰 팝업 화면 (USB 디버깅 허용)

#### Step 2: 프로그램 실행
```
1. C:\Users\사용자명\scrcpy 폴터 열기
2. scrcpy.exe 더블클릭
3. 핸드폰 화면이 PC에 뜹니다!
```

**화면:**
- 파일 탐색기 → 실행 과정
- 성공: 핸드폰 화면이 PC 창에 표시되는 순간!

**성공 화면:**
- 핸드폰 화면이 PC 창에 실시간으로 표시됨
- 마우스로 터치, 키보드로 타이핑 가능

**팁:**
> "화면이 뜨면 마우스로 터치하고, 키보드로 글자도 칠 수 있어요!"

---

### [본론 5] 묣선(WiFi)으로 연결하기 8:00-11:00 (3분)

**핵심:** 처음 1회는 USB로, 그 다음부터 묣선!

#### Step 1: TCP/IP 모드 활성화 (USB 연결 상태에서)
```
1. USB로 연결된 상태 유지
2. 명령 프롬프트(CMD) 또는 PowerShell 열기
3. 명령어 입력: scrcpy --tcpip
4. 완료 메시지 확인
```

**화면:**
- CMD/PowerShell 화면
- 명령어 입력 과정
- 성공 메시지: "restarting in TCP mode port: 5555"

#### Step 2: USB 케이블 제거
```
USB 케이블을 뽑아도 됩니다!
```

**화면:**
- 케이블을 는 장면
- 화면 연결이 유지되는 모습 (끊기지 않음!)

#### Step 3: WiFi로 연결
```
1. 핸드폰 설정 → WiFi → 연결된 네트워크 확인
2. IP 주소 확인 (예: 192.168.1.100)
3. 명령어 입력: scrcpy --tcpip=192.168.1.100
```

**화면:**
- 핸드폰 WiFi 설정 화면 (IP 주소 확인)
- CMD에 IP 주소 입력
- 연결 성공!

#### Step 4: 이후 재연결
```
한 번 설정하면 다음부터는:
scrcpy --tcpip=192.168.1.100
(또는 IP 입력 없이 scrcpy만 입력해도 됨)
```

**화면:**
- 묣선으로 연결된 상태
- 자유롭게 핸드폰을 들고 이동하는 모습

**강조:**
> "이제 케이블 없이도 됩니다! 핸드폰을 자유롭게 들고 다니면서 컴퓨터 화면으로 볼 수 있어요!"

### [본론 6] OBS에서 활용하기 11:00-13:00 (2분)

**목적:** 강의 영상 녹화/방송에 활용

#### Step 1: OBS 실행
```
OBS Studio 실행
```

#### Step 2: 소스 추가
```
1. 소스 박스에서 [+] 버튼 클릭
2. "창 캡처" (Window Capture) 선택
3. 이름: "핸드폰 화면" 입력
```

**화면:**
- OBS 인터페이스
- 소스 추가 과정 시연

#### Step 3: 창 선택
```
1. "창" 드롭다운에서 [scrcpy.exe] 또는 핸드폰 모델명 선택
2. 캡처 방법: Windows 10/11 (또는 BitBlt)
3. 확인
```

**화면:**
- OBS 소스 속성 창
- 창 선택 드롭다운

#### Step 4: 녹화 시작
```
[녹화 시작] 버튼 클릭
→ 핸드폰에서 시연 진행
→ [녹화 중지] 버튼 클릭
→ 영상 파일 저장 완료!
```

**화면:**
- OBS 녹화 인터페이스
- 녹화 시작/중지 과정
- 저장된 영상 재생
