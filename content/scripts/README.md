## 💡 촬영 팁

### 시니어 친화적 촬영법
1. **조명**: 밝고 따뜻한 조명 (그림자 주의)
2. **카메라 각도**: 눈높이에서 촬영 (내려다보지 않기)
3. **글자 크기**: 화면 속 텍스트는 20pt 이상
4. **속도**: 말하기 속도는 일반보다 20% 느리게
5. **반복**: 중요한 내용은 2-3번 반복 설명

### 효과적인 B-roll
- 김영희가 일상생활하는 모습 (차 마시기, 책상 앞)
- 손주와 함께 있는 모습
- 완성된 결과물을 자랑하는 모습
- AI가 생성하는 과정 (화면 녹화)
