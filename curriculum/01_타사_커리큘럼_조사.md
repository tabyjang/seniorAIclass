## 7. 고급/전문가 과정

### 7.1 서울대 AI 기업교육센터 - 생성형 AI 과정 (50시간/5주)

#### 기초 이론 (16시간)
| 주제 | 시간 | 상세 내용 |
|------|------|----------|
| 생성형 AI 개요 | 3h | 정의, 역사, 활용 사례, GAN/VAE/Diffusion/GPT 개요, 윤리 이슈 |
| ML/DL 기초 | 3h | 딥러닝 구조, 손실함수, 최적화, Transformer, Attention |
| 모델 유형 | 7h | GAN(DCGAN/CycleGAN/StyleGAN), VAE, Diffusion, VLM |
| Python 환경 | 3h | TensorFlow/PyTorch, Hugging Face, OpenAI API |

#### 실습 및 모델 구현 (17시간)
| 주제 | 시간 | 상세 내용 |
|------|------|----------|
| GAN 실습 | 4h | DCGAN, StyleGAN 이미지 생성 |
| VAE 실습 | 3h | 데이터 복원, 잠재 공간 탐험 |
| Transformer 실습 | 3h | HuggingFace, GPT Fine-Tuning |
| Diffusion 실습 | 4h | 노이즈 추가/제거, 조건부 생성 |
| VLIP-2 실습 | 3h | 이미지 설명 생성, 텍스트 기반 이미지 검색 |

#### 응용 사례 연구 (7시간)
| 주제 | 시간 | 상세 내용 |
|------|------|----------|
| 이미지 생성 응용 | 1.5h | 아트/디자인, 얼굴 생성, 패션, 의료 |
| 텍스트 생성 응용 | 1.5h | 기사 작성, 창작, 챗봇, 번역, 요약 |
| 멀티모달 응용 | 2h | DALL-E, Stable Diffusion, TTS, 음악 생성 |
| 팀 프로젝트 | 2h | 프로젝트 설명 |

#### 고급 주제 (10시간)
| 주제 | 시간 | 상세 내용 |
|------|------|----------|
| 최신 기술 동향 | 3h | 연구 트렌드, Zero-shot, Few-shot |
| 한계와 윤리 | 3h | Deepfake, 데이터 편향, AI 책임성, 규제 |
