# 프로덕트 소개 페이지 구성 아이디어 3가지

> **유형**: 홈페이지 마케팅용  
> **목적**: "이런 걸 만들 수 있어요" 보여주기  
> **작성자**: 그러쥬

---

## 아이디어 1: **"프로덕트 쇼케이스" (박물관 전시 스타일)**

### 컨셉
> "40개의 AI 걸작, 지금 만나보세요"

### 구성
```
┌─────────────────────────────────────────┐
│     📷 40개 프로덕트 쇼케이스 📷        │
│                                         │
│   "Lv.1 수료생 김영희(62세)님의         │
│    실제 완성물을 확인해보세요"           │
└─────────────────────────────────────────┘

[필터] 전체 | 초보자 | 중급자 | 고급자 | 전문가

┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐
│ 📧 Gmail │ │ 📱 폰    │ │ 🖱️ 클릭  │ │ 💬 대화  │
│          │ │  세팅    │ │ 마스터   │ │  기록    │
├──────────┤ ├──────────┤ ├──────────┤ ├──────────┤
│완성 예시 │ │완성 예시 │ │완성 예시 │ │완성 예시 │
│ 이미지   │ │  이미지  │ │  이미지  │ │  이미지  │
│  보여주기│ │  보여주기│ │  보여주기│ │  보여주기│
├──────────┤ ├──────────┤ ├──────────┤ ├──────────┤
│ ✅ 실제  │ │ ✅ 실제  │ │ ✅ 실제  │ │ ✅ 실제  │
│  수료생  │ │  수료생  │ │  수료생  │ │  수료생  │
│  작품    │ │  작품    │ │  작품    │ │  작품    │
└──────────┘ └──────────┘ └──────────┘ └──────────┘

[더보기 →]  [이 레벨 시작하기 →]
```

### 특징
- **4×4 그리드**: 한 화면에 16개 정도 보여줌
- **완성 예시**: 실제 수료생의 완성물 사진/스크린샷
- **필터**: 레벨별로 볼 수 있음
- **호버**: 클릭하면 완성물 상세 보기

### 장점
- ✅ 실제 완성물이라 신뢰감 ↑
- ✅ 다양한 결과물이 한눈에 들어옴
- ✅ "나도 할 수 있겠다" 자신감 유발

## 아이디어 2: **"AI 여정 지도" (모험/탐험 스타일)**

### 컨셉
> "40개의 보물을 찾아 떠나는 AI 여정"

### 구성
```
┌─────────────────────────────────────────┐
│   🗺️ AI 마스터 여정 지도 🗺️            │
│                                         │
│   40개 프로덕트 = 40개의 보물           │
│   12주 완성 = 4개의 대륙 탐험          │
└─────────────────────────────────────────┘

                [시작]
                  │
    ┌─────────────┼─────────────┐
    │             │             │
   🏝️          🏔️           🌊
 초보자의     중급자의      고급자의
 섬 (10개)   산 (10개)    바다 (10개)
    │             │             │
    └─────────────┼─────────────┘
                  │
               🏰 전문가의
                성 (10개)
                  │
              [마스터]

[각 섬/산/바다/성 클릭하면 해당 프로덕트 10개 보기]
```

### 특징
- **모험 지도 UI**: 섬→산→바다→성으로 성장 여정 표현
- **단계별 클릭**: 각 지역 클릭하면 10개 프로덕트 리스트
- **진행률**: 방문한 곳은 색깔 채워짐 (게임화)

### 장점
- ✅ "12주 여정" 메시지가 시각적으로 강렬
- ✅ 초보자→전문가 성장 스토리텔링
- ✅ 시니어에게 "여행"은 친근한 메타포

## 아이디어 3: **"실전 사례집" (잡지/카탈로그 스타일)**

### 컨셉
> "실제 수료생들이 만든 40가지 AI 활용 사례"

### 구성
```
┌─────────────────────────────────────────┐
│  📖 AI 마스터 성공 사례집 📖            │
│                                         │
│   시니어들이 AI로 만든 놀라운 결과물    │
│   총 40개 실제 사례를 소개합니다        │
└─────────────────────────────────────────┘

[Featured Story] ⭐
┌────────────────────────────────────────┐
│  👤 김영희 (62세, 주부)                │
│                                        │
│  "손주에게 AI로 동화책 만들어줬어요"    │
│                                        │
│  ┌──────────┐  만든 것:               │
│  │          │  • 동화책 1권           │
│  │ 동화책   │  • AI 그림 20장         │
│  │ 표지     │  • 손주 최애 작품       │
│  │ 사진     │                        │
│  └──────────┘  사용한 AI: ChatGPT     │
│                 3개월 소요            │
│                                        │
│  [전체 사례 보기 →]                    │
└────────────────────────────────────────┘

[카테고리별 보기]
┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐
│ 📧     │ │ 💌     │ │ 📚     │ │ 🎬     │
│ 메일   │ │ 편지   │ │ 동화   │ │ 영상   │
│ 10개   │ │ 12개   │ │ 5개    │ │ 8개    │
└────────┘ └────────┘ └────────┘ └────────┘

[전체 40개 프로덕트 보기 →]
```

### 특칙
- **사람 중심**: 누가(수료생) 무엇을(프로덕트) 만들었는지 스토리텔링
- **피처드 스토리**: 대표 사례 1개 상세히 보여줌
- **카테고리**: 프로덕트 유형별로 모아볼 수 있음

### 장점
- ✅ "누군가 성공했다"는 사회적 증명
- ✅ 감성적 연결 (할머니/손주 관계 등)
- ✅ "나도 저렇게 되고 싶다" 욕구 자극

## 그러쥬 추천

### 🥇 **아이디어 2 (AI 여정 지도)** 

**이유**:
1. **시니어 친화적**: "여행"은 누구나 좋아하는 메타포
2. **성장 스토리**: 초보자→전문가 4단계가 시각적으로 명확
3. **게임화**: 방문한 곳 채워지는 느낌으로 동기부여

### 🥈 **아이디어 1 (프로덕트 쇼케이스)**

**대안으로 좋은 이유**:
- 완성물이 많아서 압도적임
- 실제 결과물이라 신뢰도 높음
- 필터로 원하는 레벨만 볼 수 있음
