## 팔레트 1: "미드나잇 럭셔리" ⭐ **그러쥬 추천**

### 컨셉
> 깊은 밤하늘 + 골드 문양

### 색상 구성
```css
color: {
  // 메인 - 미드나잇 네이비
  primary: '#0F172A',        // 딥 네이비 (메인)
  primaryLight: '#1E293B',   // 라이트 네이비
  
  // 포인트 - 앤티크 골드
  accent: '#D4A853',         // 앤티크 골드 (CTA 버튼)
  accentLight: '#E8D4A8',    // 라이트 골드
  
  // 배경 - 크림 아이보리
  background: '#FAF9F6',     // 웜 화이트
  backgroundAlt: '#F5F5F0',  // 오프 화이트
  
  // 텍스트
  text: '#1C1917',           // 차콜 블랙
  textMuted: '#78716C',      // 그레이
  
  // 레벨별 (톤온톤)
  level: {
    1: '#0F172A',  // 네이비 (기초)
    2: '#334155',  // 슬레이트 (성장)
    3: '#D4A853',  // 골드 (전문)
    4: '#1E293B'   // 딥 네이비 (마스터)
  }
}
```

### 적용 예시
```
[로고] 시니어 AI 클래스 (골드 #D4A853)

[배경] 미드나잇 네이비 (#0F172A)
[텍스트] 크림 아이보리 (#FAF9F6)

[CTA 버튼] 앤티크 골드 (#D4A853) + 화이트 텍스트
[보조 버튼] 투명 테두리 + 골드 텍스트

[카드] 크림 배경 + 네이비 텍스트
[구분선] 골드 라인 (1px)
```

### 분위기
- ✅ 고급스러운 호텔/라운지 느낌
- ✅ 금융/컨설팅 신뢰감
- ✅ 시니어에게 "품격" 있는 느낌

## 팔레트 2: "세이지 가든" (내추럴 럭셔리)

### 컨셉
> 고급 스파 + 유기농 카페

### 색상 구성
```css
color: {
  // 메인 - 세이지 그린
  primary: '#6B7B5E',        // 세이지 그린
  primaryLight: '#8A9A7D',   // 라이트 세이지
  
  // 포인트 - 코퍼/테라코타
  accent: '#B98B73',         // 코퍼 브라운
  accentLight: '#D4B5A8',    // 로즈 코퍼
  
  // 배경 - 에크루/토프
  background: '#F7F5F0',     // 에크루
  backgroundAlt: '#EDE8E0',  // 토프
  
  // 텍스트
  text: '#2C2420',           // 다크 브라운
  textMuted: '#8B7E74',      // 웜 그레이
  
  // 레벨별 (톤온톤)
  level: {
    1: '#8A9A7D',  // 라이트 세이지 (초보)
    2: '#6B7B5E',  // 세이지 (중급)
    3: '#4A5A45',  // 딥 세이지 (고급)
    4: '#B98B73'   // 코퍼 (전문가)
  }
}
```

### 분위기
- ✅ 자연스러운 고급스러움
- ✅ 건강/웰빙 이미지
- ✅ 여성 시니어 친화적

## 팔레트 3: "샴페인 클래식" (타임리스)

### 컨셉
> 샴페인 바 + 클래식 카

### 색상 구성
```css
color: {
  // 메인 - 차콜 그레이
  primary: '#2D3436',        // 차콜
  primaryLight: '#636E72',   // 그레이
  
  // 포인트 - 샴페인 골드
  accent: '#C9B037',         // 샴페인 골드
  accentLight: '#E5D78A',    // 라이트 샴페인
  
  // 배경 - 아이보리/펄
  background: '#FDFCF8',     // 펄 화이트
  backgroundAlt: '#F0EFE9',  // 아이보리
  
  // 텍스트
  text: '#1A1A1A',           // 블랙
  textMuted: '#6C6C6C',      // 중간 그레이
  
  // 레벨별 (그레이 스케일)
  level: {
    1: '#95A5A6',  // 실버 (초보)
    2: '#636E72',  // 그레이 (중급)
    3: '#C9B037',  // 샴페인 골드 (고급)
    4: '#2D3436'   // 차콜 (전문가)
  }
}
```

### 분위기
- ✅ 클래식한 명품 브랜드 느낌
- ✅ 남성 시니어에게 강한 신뢰감
- ✅ 시간이 지나도 안 올드

## 팔레트 4: "모던 브론즈" (컨템포러리)

### 컨셉
> 현대 미술관 + 디자인 호텔

### 색상 구성
```css
color: {
  // 메인 - 딥 인디고
  primary: '#1A1F3C',        // 딥 인디고
  primaryLight: '#2D3561',   // 인디고
  
  // 포인트 - 브론즈/구리
  accent: '#B87333',         // 브론즈
  accentLight: '#CD853F',    // 퍼스널 브론즈
  
  // 배경 - 스톤/콘크리트
  background: '#F5F5F3',     // 스톤 화이트
  backgroundAlt: '#E8E8E6',  // 라이트 그레이
  
  // 텍스트
  text: '#0F0F0F',           // 딥 블랙
  textMuted: '#6B6B6B',      // 그레이
  
  // 레벨별
  level: {
    1: '#4A5568',  // 슬레이트 (초보)
    2: '#2D3561',  // 인디고 (중급)
    3: '#B87333',  // 브론즈 (고급)
    4: '#1A1F3C'   // 딥 인디고 (전문가)
  }
}
```

### 분위기
- ✅ 현대적인 고급스러움
- ✅ IT/테크 느낌 + 온도감
- ✅ 젊은 시니어(50대)에게 어필

## 그러쥬 최종 추천: "미드나잇 럭셔리"

### 이유
1. **고급스러움 극대화**: 네이비+골드는 영원한 럭셔리 조합
2. **시니어 신뢰**: 금융권/컨설팅에서 자주 쓰는 신뢰색
3. **가독성**: 딥 네이비 배경에 크림색 텍스트 = 선명한 대비
4. **차별화**: 대부분의 교육 사이트가 밝은색일 때 반전 매력

### 실제 적용 예시 (Hero 섹션)

```
┌─────────────────────────────────────────────┐
│                                             │
│    [로고: 골드 #D4A853]                     │
│                                             │
│                                             │
│         AI, 어렵지 않아요                   │
│         ───────────────                     │
│         (크림색 #FAF9F6)                    │
│                                             │
│    50-70대를 위한 맞춤형 AI 교육            │
│                                             │
│    [무료 상담 받기]                         │
│    (골드 배경 #D4A853, 네이비 텍스트)       │
│                                             │
│    [카톡 상담하기]                          │

│                                             │

│  배경: 미드나잇 네이비 그라데이션           │
│       (#0F172A → #1E293B)                  │
└─────────────────────────────────────────────┘
```
