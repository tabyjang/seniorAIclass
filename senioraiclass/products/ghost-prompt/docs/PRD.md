# 고스트프롬프트 PRD (Product Requirements Document)

## 1. 제품 개요

### 1.1 제품명
**고스트프롬프트 (Ghost Prompt)**

### 1.2 한 줄 소개
> "내 화면에는 보이지만, 영상에는 담기지 않는 마법의 프롬프터"

### 1.3 타겟 사용자
- 유튜브 크리에이터 (특히 시니어)
- 온라인 강사
- 웨비나/화상회의 발표자
- 튜토리얼 영상 제작자

### 1.4 핵심 문제
| 기존 방식 | 문제점 |
|-----------|--------|
| 듀얼 모니터 | 시선이 카메라에서 벗어남 |
| 물리 프롬프터 | 비용, 설치 복잡 |
| 암기 | 실수, 자연스럽지 않음 |
| 편집으로 제거 | 시간 소모, 기술 필요 |

### 1.5 솔루션
Windows API `SetWindowDisplayAffinity(WDA_EXCLUDEFROMCAPTURE)`를 활용해 **화면에는 보이지만 캡처에는 안 잡히는** 투명 프롬프터

## 3. 기능 요구사항

### 3.1 핵심 기능 (6가지)

| # | 기능 | 설명 |
|---|------|------|
| 1 | **캡처 방지** | 화면 녹화/공유 시 미표시 |
| 2 | **반투명 프롬프터** | 모든 프로그램 위에 항상 표시 |
| 3 | **자동/수동 스크롤** | 설정 속도 자동 스크롤 + 단축키 제어 |
| 4 | **전역 단축키** | 다른 앱 사용 중에도 제어 가능 |
| 5 | **커스터마이징** | 글자 크기, 색상, 배경 투명도 조절 |
| 6 | **접이식 UI** | 필요할 때만 설정 패널 펼침 |

### 3.2 설정 항목

| 설정 | 범위 | 기본값 |
|------|------|--------|
| 글자 크기 | 12px ~ 72px | 24px |
| 스크롤 속도 | 0.25x ~ 4.0x | 1.0x |
| 배경 투명도 | 0% ~ 100% | 85% |
| 글자 색상 | 흰색/노랑/초록 | 흰색 |
| 배경 색상 | 검정/네이비 | 검정 |

### 3.3 파일 기능
- `.txt` 파일 불러오기 (UTF-8, EUC-KR)
- 스크립트 저장
- 최근 파일 목록

## 4. 단축키

### 4.1 기본 단축키

| 단축키 | 기능 |
|--------|------|
| `F8` | 재생/정지 |
| `F9` | 프롬프터 표시/숨김 |
| `Ctrl + ↑` | 스크롤 속도 증가 |
| `Ctrl + ↓` | 스크롤 속도 감소 |
| `PgUp` | 위로 스크롤 |
| `PgDn` | 아래로 스크롤 |
| `Home` | 처음으로 |
| `End` | 끝으로 |

### 4.2 전역 단축키
- 다른 앱(PPT, 브라우저 등) 사용 중에도 동작
- 모든 단축키 사용자 변경 가능
