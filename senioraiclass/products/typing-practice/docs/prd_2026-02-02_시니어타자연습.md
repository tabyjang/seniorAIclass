## 1. 개요

### 1.1 프로젝트명
**시니어 타자 연습** (Senior Typing Practice)

### 1.2 한 줄 설명
> "어르신을 위한 쉽고 큰 글씨 타자 연습 프로그램"

### 1.3 만드는 이유 (Why?)

**문제점:**
- 기존 타자 연습 프로그램은 글씨가 너무 작음
- 설명이 복잡하고 어려움
- 영어 위주 (한글 연습 부족)
- 시니어를 배려한 UI가 없음

**해결책:**
- 큰 글씨 (최소 24pt 이상)
- 단순하고 직관적인 UI
- 한글 문장 위주
- 따뜻하고 친근한 디자인
- 격려 메시지로 동기부여

### 1.4 영감 (Reference)
- **한메타자연습**: 도스 시절 국민 타자 연습 프로그램
- "산토끼 토끼야 어디를 가느냐" 같은 친숙한 문장

## 2. 타겟 사용자

### 2.1 주요 타겟
| 항목 | 내용 |
|------|------|
| **연령** | 55세 ~ 75세 |
| **특징** | 컴퓨터/스마트폰 초보자 |
| **니즈** | 키보드 타이핑 연습 |
| **환경** | PC, 태블릿, 스마트폰 |

### 2.2 사용자 페르소나

**👵 김순자 어르신 (68세)**
- 손주와 카톡하고 싶은데 타자가 느림
- 두 손가락으로 하나씩 누름
- 글씨가 작으면 잘 안 보임
- 틀리면 당황하고 포기하고 싶어짐
- 칭찬받으면 더 열심히 하게 됨

### 2.3 사용자가 해결하고 싶은 문제
1. "타자가 너무 느려요"
2. "키보드 어디 누르는지 모르겠어요"
3. "연습하고 싶은데 어려워요"
4. "재미없어서 금방 포기해요"

## 3. 핵심 기능 (Features)

### 3.1 필수 기능 (Must Have) ⭐

| # | 기능 | 설명 |
|---|------|------|
| 1 | **큰 글씨 문장 표시** | 24pt 이상, 눈에 잘 보이게 |
| 2 | **실시간 입력 확인** | 맞으면 초록, 틀리면 빨강 |
| 3 | **난이도 선택** | 쉬움/보통/어려움/속담 |
| 4 | **진행률 표시** | "3/10 완료" 형태 |
| 5 | **정확도 측정** | 몇 % 맞았는지 |
| 6 | **타수 측정** | 분당 타수 (속도) |
| 7 | **시간 측정** | 얼마나 걸렸는지 |
| 8 | **격려 메시지** | "잘하셨어요! 👏" |

### 3.2 추가 기능 (Nice to Have) 💡

| # | 기능 | 설명 | 우선순위 |
|---|------|------|---------|
| 1 | 효과음 | 타이핑 소리, 완료 소리 | 중 |
| 2 | 키보드 가이드 | 어떤 키 누르는지 표시 | 중 |
| 3 | 연습 기록 저장 | 이전 기록 보기 | 하 |
| 4 | 나만의 문장 추가 | 사용자 커스텀 | 하 |
| 5 | 다크모드 | 어두운 테마 | 하 |

### 3.3 제외 범위 (Out of Scope) ❌

| 기능 | 제외 이유 |
|------|----------|
| 회원가입/로그인 | 복잡함, 나중에 |
| 랭킹 시스템 | 서버 필요, 나중에 |
| 영어 타자 | 한글 집중 |
| 게임 모드 | 1.0에서는 제외 |
| 광고 | 사용자 경험 저해 |

## 5. 콘텐츠 (연습 문장)

### 5.1 난이도별 문장

**🌱 쉬움 (2~5글자)**
```
안녕하세요
감사합니다
사랑합니다
좋은 하루
행복하세요
```

**🌿 보통 (6~12글자)**
```
오늘 날씨가 좋습니다
맛있는 점심 드세요
건강이 최고입니다
가족이 보고 싶어요
```

**🌳 어려움 (13글자 이상)**
```
산토끼 토끼야 어디를 가느냐
깊은 산속 옹달샘 누가 와서 먹나요
동해물과 백두산이 마르고 닳도록
```

**📚 속담/명언**
```
낮말은 새가 듣고 밤말은 쥐가 듣는다
천 리 길도 한 걸음부터
가는 말이 고와야 오는 말이 곱다
```

### 5.2 문장 개수
- 난이도당 10개 이상
- 랜덤 셔플로 매번 다른 순서
- 총 40개+ 문장 준비

## 6. 성공 기준 (Definition of Done)

### 6.1 기능 완료 조건

| # | 조건 | 확인 |
|---|------|------|
| 1 | 시작 버튼 누르면 연습 시작 | ☐ |
| 2 | 문장이 큰 글씨로 표시됨 | ☐ |
| 3 | 입력하면 실시간으로 맞/틀 표시 | ☐ |
| 4 | 문장 완료하면 자동으로 다음 문장 | ☐ |
| 5 | 10문장 완료하면 결과 표시 | ☐ |
| 6 | 난이도 변경 가능 | ☐ |
| 7 | 정확도/타수/시간 측정됨 | ☐ |
| 8 | 모바일에서도 동작 | ☐ |

### 6.2 품질 기준

| 항목 | 기준 |
|------|------|
| **반응 속도** | 입력 후 0.1초 이내 반응 |
| **가독성** | 1m 거리에서 문장 읽힘 |
| **접근성** | 색맹도 구분 가능 (밑줄 추가) |
| **호환성** | Chrome, Edge, Safari, 모바일 |
