# 📋 베네치아 게임 PRD (Product Requirements Document)

> **프로젝트명:** 시니어 타자 연습 - 베네치아 게임 모드
> **버전:** 1.0
> **작성일:** 2026-02-02
> **작성자:** 범식님 & 골디

## 1. 개요

### 1.1 프로젝트명
**베네치아 게임** (Venezia Typing Game)

### 1.2 한 줄 설명
> "하늘에서 떨어지는 단어를 타자로 파괴하는 추억의 게임"

### 1.3 만드는 이유 (Why?)

**문제점:**
- 기존 문장 연습은 반복적이고 지루함
- 시니어들도 게임을 통해 더 즐겁게 연습하고 싶음
- 한메타자교사의 베네치아 게임에 대한 향수

**해결책:**
- 게임 형식으로 타자 연습
- 성취감과 재미 동시에 제공
- 친숙한 게임 규칙 (한메타자 베네치아)
- 쉬운 난이도부터 시작

### 1.4 영감 (Reference)
- **한메타자교사 베네치아** (1989~2003)
- **ztype** (phoboslab.org/ztype)
- **한메타자 베네치아 영어** (모바일 앱)

## 2. 타겟 사용자

### 2.1 주요 타겟

| 항목 | 내용 |
|------|------|
| **연령** | 55세 ~ 75세 |
| **특징** | 컴퓨터/스마트폰 초보자 |
| **니즈** | 지루하지 않은 타자 연습 |
| **환경** | PC, 태블릿 |

### 2.2 사용자 페르소나

**👴 박영수 어르신 (62세)**
- 손주와 게임하고 싶은데 타자가 느려서 못함
- 복잡한 게임은 어려움
- 예전에 베네치아 게임 해봤음
- 점수 내는 재미에 동기부여됨

### 2.3 사용자가 해결하고 싶은 문제
1. "타자 연습이 지루해요"
2. "게임하면서 연습하고 싶어요"
3. "손주랑 같이 할 수 있는 게임 없을까요"
4. "예전에 하던 베네치아 게임 기억나요"

## 6. 게임 규칙 (Game Rules)

### 6.1 기본 규칙

1. **단어 생성:** 랜덤한 X위치에서 단어가 생성됨
2. **단어 하강:** 단어는 위에서 아래로 떨어짐
3. **타자 입력:** 화면에 있는 단어와 똑같이 입력하면 파괴
4. **점수 획득:** 단어 길이 × 100점
5. **바닥 충돌:** 단어가 바닥에 닿으면 생명 -1
6. **게임오버:** 생명(♥) 0이면 게임 종료

### 6.2 점수 시스템

| 단어 길이 | 점수 |
|----------|------|
| 2글자 | 200점 |
| 3글자 | 300점 |
| 4글자 | 400점 |
| 5글자 | 500점 |
| 6글자+ | 600점 |

### 6.3 레벨 업 (추가 기능)

| 레벨 | 조건 | 효과 |
|------|------|------|
| Level 1 | 시작 | 기본 속도 |
| Level 2 | 1,000점 | 속도 +20% |
| Level 3 | 2,500점 | 속도 +40% |
| Level 4 | 5,000점 | 속도 +60% |

## 8. 기술 요구사항

### 8.1 기술 스택

| 구분 | 기술 | 이유 |
|------|------|------|
| 마크업 | HTML5 | 표준 |
| 스타일 | CSS3 | 애니메이션 |
| 로직 | JavaScript (ES6+) | 실시간 처리 |
| 렌더링 | DOM + CSS | Canvas보다 간단, 글씨 크게 표시 용이 |

### 8.2 핵심 알고리즘

```javascript
// 게임 루프
function gameLoop() {
    // 1. 모든 단어 이동
    fallingWords.forEach(word => {
        word.y += word.speed;
        word.element.style.top = word.y + 'px';
    });
    
    // 2. 바닥 체크
    checkBottomCollision();
    
    // 3. 새 단어 생성
    if (shouldSpawnWord()) spawnWord();
    
    requestAnimationFrame(gameLoop);
}

// 입력 처리
function handleInput(input) {
    const index = fallingWords.findIndex(w => w.text === input);
    if (index !== -1) {
        destroyWord(index);
        addScore(input.length * 100);
    }
}
```
