## 1. 콘텐츠 제작 방식

### 영상 제작 도구
| 용도 | 도구 | 비용 |
|------|------|------|
| 화면 녹화 | OBS Studio | 무료 |
| 스마트폰 미러링 | scrcpy / LetsView | 무료 |
| 영상 편집 | CapCut / Vrew | 무료 |
| 자막 생성 | Vrew (AI 자동) | 무료 |
| 썸네일 | Canva / Gemini | 무료 |
| 음성 | 직접 녹음 or TTS (qwen3-tts) | 무료 |

### 영상 스펙
| 항목 | 값 |
|------|-----|
| 해상도 | 1920x1080 (FHD) |
| 프레임 | 30fps |
| 길이 | 5-10분 |
| 자막 | 항상 표시, 큰 글씨 |
| 형식 | MP4 |

## 2. 시니어용 앱/서비스 가이드

### 모듈 1: 사진 디지털화

#### 구글 포토스캔 (PhotoScan)
- **플랫폼**: iOS / Android
- **비용**: 무료
- **기능**: 반사 제거, 자동 크롭, 회전 보정
- **다운로드**: 앱스토어/플레이스토어에서 "포토스캔" 검색

#### 사용법 핵심
```
1. 앱 실행
2. 사진 위에 폰 위치
3. 4개 점에 맞춰 이동 (자동 촬영)
4. 완성된 스캔 확인
5. 구글 포토에 자동 저장
```

---

### 모듈 2: 클라우드 저장

#### 구글 포토 (Google Photos)
- **플랫폼**: iOS / Android / Web
- **비용**: 무료 (15GB)
- **기능**: 자동 백업, 검색, 공유, AI 분류

#### 설정 핵심
```
1. 구글 계정 로그인
2. 설정 → 백업 켜기
3. "저장공간 절약" 선택 (용량 절약)
4. 와이파이에서만 백업 (데이터 절약)
```

#### 용량 계산
| 화질 | 15GB로 저장 가능 |
|------|-----------------|
| 저장공간 절약 | 약 10만 장 |
| 원본 화질 | 약 5천 장 |

---

### 모듈 3: 가족 공유

#### 공유 방법 3가지
| 방법 | 난이도 | 설명 |
|------|--------|------|
| 링크 공유 | ⭐ | 앨범 선택 → 공유 → 링크 복사 → 카톡 전송 |
| 파트너 공유 | ⭐⭐ | 특정 사람과 모든 사진 자동 공유 |
| 공유 앨범 | ⭐⭐ | 가족 모두 사진 추가 가능 |

---

### 모듈 4: AI 사진 보정

#### Remini (AI 화질 개선)
- **플랫폼**: iOS / Android
- **비용**: 무료 (하루 5장) / 유료
- **기능**: 흐린 사진 선명하게, 얼굴 보정

#### Palette.fm (흑백→컬러)
- **플랫폼**: Web
- **비용**: 무료
- **URL**: https://palette.fm
- **기능**: AI가 흑백 사진에 자연스러운 색 입힘

#### Gemini 활용
- 사진 업로드 → "이 사진 설명해줘"
- 자동으로 사진 내용, 시대, 장소 추측
- 앨범 정리할 때 유용

## 3. 영상 제작 워크플로우

```
[기획]
  ↓
주제 선정 (모듈/영상 번호)
  ↓
대본 작성 (Claude 활용)
  ↓
[촬영]
  ↓
스마트폰 미러링 + OBS 녹화
  ↓
음성 녹음 (또는 TTS)
  ↓
[편집]
  ↓
Vrew로 자막 자동 생성
  ↓
CapCut으로 편집 (화살표, 강조)
  ↓
썸네일 제작 (Canva/Gemini)
  ↓
[업로드]
  ↓
유튜브 업로드
  ↓
제목/설명/태그 최적화
```
