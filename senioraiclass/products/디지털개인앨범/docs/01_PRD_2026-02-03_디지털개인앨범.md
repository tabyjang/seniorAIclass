## 1. 제품 개요

### 제품명
**디지털 개인앨범 소장 가이드**

### 한 줄 소개
시니어가 소중한 추억 사진을 디지털로 안전하게 보관하고, 가족과 쉽게 공유하는 방법을 알려주는 교육 콘텐츠

### 타겟 사용자
- 50-70대 시니어
- 옛날 사진(인화 사진)을 많이 가지고 있는 분
- 스마트폰은 쓰지만 클라우드/앨범 앱은 모르는 분
- 자녀/손주에게 사진 공유하고 싶은 분

### 핵심 문제
| 문제 | 상세 |
|------|------|
| 사진 유실 위험 | 인화 사진 바래짐, 화재/수해 시 소실 |
| 공유 어려움 | 실물 사진은 한 명만 볼 수 있음 |
| 정리 안 됨 | 상자에 쌓여있고 언제 찍은 건지 모름 |
| 디지털 전환 모름 | 스캔? 클라우드? 어떻게 하는지 모름 |

### 솔루션
| 해결책 | 방법 |
|---------|------|
| 디지털화 | 스마트폰/스캐너로 사진 디지털 변환 |
| 안전 보관 | 클라우드(구글 포토)에 자동 백업 |
| 쉬운 공유 | 링크/앨범으로 가족에게 전송 |
| AI 활용 | 사진 보정, 흐린 사진 복원, 자동 분류 |

## 2. 기능 요구사항

### 핵심 콘텐츠 (MVP)

#### 모듈 1: 사진 디지털화
| 영상 | 내용 |
|------|------|
| 1-1 | 스마트폰으로 옛날 사진 찍기 (제대로 찍는 법) |
| 1-2 | 포토스캔 앱 사용법 (구글) |
| 1-3 | 스캐너 없이 깔끔하게 스캔하는 팁 |

#### 모듈 2: 클라우드 저장
| 영상 | 내용 |
|------|------|
| 2-1 | 구글 포토란? (무료 저장소 개념) |
| 2-2 | 구글 포토 가입하기 |
| 2-3 | 사진 업로드하기 (자동 백업 설정) |
| 2-4 | 폴더/앨범으로 정리하기 |

#### 모듈 3: 가족과 공유
| 영상 | 내용 |
|------|------|
| 3-1 | 앨범 링크 만들어서 카톡으로 보내기 |
| 3-2 | 가족 공유 앨범 만들기 |
| 3-3 | 사진에 설명 달기 (언제, 누구, 어디서) |

### 추가 콘텐츠 (v2)

#### 모듈 4: AI로 사진 살리기
| 영상 | 내용 |
|------|------|
| 4-1 | 흐릿한 사진 선명하게 (AI 보정) |
| 4-2 | 흑백 사진 컬러로 변환 |
| 4-3 | 찢어진/손상된 사진 복원 |
| 4-4 | Gemini로 사진 설명 자동 생성 |

#### 모듈 5: 디지털 앨범북 만들기
| 영상 | 내용 |
|------|------|
| 5-1 | 온라인 포토북 서비스 소개 |
| 5-2 | AI로 앨범 레이아웃 자동 구성 |
| 5-3 | 주문해서 실물 앨범으로 받기 |

## 3. 필요한 것

### 시니어가 준비할 것
| 항목 | 필수 여부 | 비용 |
|------|----------|------|
| 스마트폰 | 필수 | 이미 있음 |
| 구글 계정 | 필수 | 무료 |
| 인터넷/와이파이 | 필수 | 이미 있음 |
| 옛날 사진 | 필수 | 이미 있음 |
| 스캐너 | 선택 | 없어도 됨 |

### 추천 앱/서비스
| 용도 | 앱/서비스 | 비용 |
|------|----------|------|
| 사진 스캔 | 구글 포토스캔 | 무료 |
| 저장/공유 | 구글 포토 | 무료 (15GB) |
| AI 보정 | Remini | 무료/유료 |
| 컬러 복원 | Palette.fm | 무료 |
| 포토북 | 스냅스, 포토몬 | 유료 |

## 4. UI/UX 설계

### 영상 형식
- 길이: 5-10분 (짧게!)
- 화면: 스마트폰 화면 녹화 + 음성 설명
- 자막: 큰 글씨, 항상 표시
- 속도: 천천히, 반복 설명

### 핵심 원칙
- ❌ 전문 용어 사용 금지
- ✅ "여기 누르세요" 화살표 표시
- ✅ 실수해도 괜찮다는 메시지
- ✅ 각 단계 끝에 "여기까지 따라오셨나요?" 확인
