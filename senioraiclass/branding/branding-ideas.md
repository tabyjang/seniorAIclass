## 🎯 채널 핵심 컨셉

### 탄생 스토리
> "유튜브로 AI 배우고 싶었는데,
> 너무 빠르고, 어려운 부분은 대충 넘어가고...
> 답답해서 **제가 직접 만들기로 했습니다.**"

### 기존 채널의 문제점 (우리가 해결할 것)
| 문제 | 시청자 심정 |
|------|------------|
| 너무 빠름 | "잠깐, 뭐라고?" |
| 어려운 부분 대충 넘어감 | "거기가 중요한데!" |
| 전문 용어 막 씀 | "그게 뭔 말이야..." |
| 클릭 위치 안 보여줌 | "어디를 누르라는 거야?" |
| 설명만 하고 끝 | "그래서 뭘 해보라는 거지?" |

### 우리 채널의 약속 (차별화)
```
✅ 말 속도 50% 느리게
✅ 어려운 부분은 2번, 3번 반복
✅ 클릭할 곳 빨간 동그라미로 표시
✅ 전문 용어 쓰면 바로 설명
✅ "여기서 막히시는 분 많아요" 미리 말해주기
✅ 영상 하나마다 실제 결과물 만들기
```

### 핵심 메시지
> **"무엇보다 실제 결과물을 만들어내실 수 있게 해드리겠습니다."**

## 🎵 인트로 영상 구성 (5초)

```
[0-2초] 로고 등장 + 효과음 "띵~"
        - 따뜻한 색감 배경 (주황/하늘색 그라데이션)
        
[2-4초] 슬로건 텍스트 등장
        - "영상 하나, 결과물 하나"
        - 또는 "천천히, 자세히, 끝까지"
        
[4-5초] 페이드 아웃 → 본 영상 시작
```

### 인트로 스펙
- **길이:** 5초 (길면 이탈)
- **배경색:** 따뜻한 그라데이션 (차가운 파랑 ❌)
- **효과음:** 밝고 짧은 "띵~" (무겁지 않게)
- **음악:** 희망적, 너무 빠르지 않게
- **폰트:** 둥글둥글, 친근한 느낌
