# 📚 Coursera Machine Learning (Stanford)

> **제공:** Stanford University (Coursera)
> **강사:** Andrew Ng
> **형태:** 온라인 강의 (Coursera)
> **난이도:** 초급~중급
> **URL:** https://www.coursera.org/learn/machine-learning-course

## 📌 개요

전세계에서 가장 많이 수강한 ML 강의. Andrew Ng 교수의 원조 머신러닝 강의. 2022년 리뉴얼되어 Python 기반으로 업데이트.

### Course 1: Supervised Machine Learning
| 주차 | 주제 | 내용 |
|------|------|------|
| 1 | Introduction to ML | ML이란, 지도학습 |
| 2 | Linear Regression | 선형 회귀, 경사하강법 |

| 4 | ML Development | 모델 평가, 과적합 |

- 집값 예측 모델
- 종양 분류기

### Course 2: Advanced Learning Algorithms
| 주차 | 주제 | 내용 |

### Course 3: Unsupervised Learning

## 📝 Andrew Ng의 교수법 (참고!)

1. **"Intuition First"** - 직관 먼저, 수학 나중
2. **"Show, Don't Tell"** - 설명보다 보여주기
3. **"Repeat Key Concepts"** - 핵심 개념 반복
4. **"Celebrate Progress"** - 작은 성취도 축하
